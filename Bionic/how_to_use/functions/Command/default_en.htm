Commands

Summary

 This list contains all predefined Bionic Command
Command Shortname Description
SYSTEM_INFO:=1 System Info List all the relevant elevator data
READ_SIM_CARD:=1 SIM card content List the data stored into the SIMcard
GC_SHOW_EE:=1 EEPROM content List the data stored into the EE-PROM
5 Active system status System status (Shows activeservices running)
4 Error history Shows the error log
CTRL_AVAILABILITY:=1 Availability subsystems Shows the availability of the elevator subsystems
READ_SHAFT_EE:=1 Hoistway information Shows hoistway image (in clo-sed loop, also the floor height is shown)
CTRL_IOSTATUS:=1 Control I/Os Shows the current status of the SCIC I/O ports
CAR_IOSTATUS_RQST:=1 Car I/Os Shows the status of the SDIC I/O ports. List mode must be active
CAR_LOAD_RQST:=1 CLC frequency Shows the current cell measurement (Hz) in Hex
LIST_MODE:=1 Watch all data Scrolls the filtered messages (mode one in this case)
LIST_MODE:=0 HSTOP Watch all data Stops the messages scrolling
SIM_FLOOR_CALL:=1-n Call to floor 1 ..n. Lowest floor is (1)
CLR_PERSISTENT:=1 Reset persistent fatal error Clear persistent fatal error
CF_DUMP:=1 Actual configuration Read out the complete configuration)
di Read out the complete configuration)
SET_AAT_MODE:=0 Disable automatic acceptance test mode in case of SW hangup during acceptance tests(HMI display S001r3)
CTRL_SHOW_SERVICES:=1 (Instead of 5) Shows the service currently active
SHOW_LAST_ERR:=1x (Instead of 4) List the error history
