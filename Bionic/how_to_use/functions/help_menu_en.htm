Help Menu

Summary

 The CADI GC help systems consist of four main parts (operation, parameters, telegrams, errors). Whereas the Help command in the menu system calls the overview, which contains all parts of the help system; the Help buttons in individual windows lead to individual help pages.

Help Menu

General Help

 The CADI GC help system uses Internet technology and HTML pages. It consists of the four web documents, which can be displayed by a Java HTML viewer:
  • How to use CADI GC
  • Parameter Description
  • Telegram Definition
  • Error Definition.

The CADI GC General Help, as it is called from the Help menu, is the 'home page' for these four parts. Individual help buttons in individual windows of CADI GC are context-sensitive and evoke the respective individual pages from one of these documents.

General help can also be started with an icon from the tool bar.
