Parameter

 CF53_PA21.

DOOR_VRMGHA_IS_SUPPLY

only visible if Door_Type = 0 or 1
In some cases door systems are used which have only one
control signal. To achieve a proper Emergency Stop
behaviour, it is usual to switch off the power supply of the
door system in case of emergency stop. For HW reasons, the
door system power supply is controlled with the
output “VRMGH_A” in such cases.
If the feature is set, the next door movement is considered
as “first trip”, see door
feature “DOOR_IS_SLOW_ON_FIRST_TRIP”.
