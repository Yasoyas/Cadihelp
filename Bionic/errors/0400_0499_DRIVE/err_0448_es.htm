0448 E_DRIVE_INVERTER_CONTROLLER_FAILURE

Summary

 This section describes the 0400_0499_DRIVE error E_DRIVE_INVERTER_CONTROLLER_FAILURE.

Description

The drive inverter has detected a internal controller electronics failure

Extra Info

0: Unknown (not supported)
1: Controller board
2: Controller board memory checksum
3: Option board
4: Option board unknown

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Internal option board failure Check type and insertion of option board (if present)
2 Other serious internal failure cause Replace drive inverter
