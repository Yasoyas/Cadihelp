0446 E_DRIVE_TRIP_START_FAILURE

Summary

 This section describes the 0400_0499_DRIVE error E_DRIVE_TRIP_START_FAILURE.

Description

There was no real car movement detected by the elevator control although the drive confirmed the start of a trip.

Note, this failure only occurs if no other failure monitor has detected this error condition before (last instance).
