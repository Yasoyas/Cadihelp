0450 E_DRIVE_CAR_RELEVELING_FAILURE

Summary

 This section describes the 0400_0499_DRIVE error E_DRIVE_CAR_RELEVELING_FAILURE.

Description

The elevator (car drive) wasn't able to relevel the car

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Maximum car releveling distance exceeded Check traction means for slip.
Check for correct car releveling zone detection (hoistway information).
2 The car has overshot the releveling zone (too fast car position correction) Check for correct drive parameter setting (e.g. releveling speed).
Check for fast car load changes.
Check for correct car releveling zone detection (hoistway information).
