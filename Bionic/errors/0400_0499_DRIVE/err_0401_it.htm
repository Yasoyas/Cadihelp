0401 E_DRIVE_SAFETY_CHAIN_INTERRUPTED

Summary

 This section describes the 0400_0499_DRIVE error E_DRIVE_SAFETY_CHAIN_INTERRUPTED.

Description

Apertura imprevista del circuito di sicurezza (ad es. durante una corsa)

Extra Info

Nessuna.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Uno dei dispositivi di sicurezza è scattato (circuito di sicurezza aperto) Controllare la causa dello scatto del dispositivo di sicurezza e risolvere il problema
2 Cablaggio del circuito di sicurezza difettoso o contatto/i del circuito di sicurezza difettoso/i o regolato/i male Controllare il cablaggio generale del circuito di sicurezza
e i contatti del circuito di sicurezza
