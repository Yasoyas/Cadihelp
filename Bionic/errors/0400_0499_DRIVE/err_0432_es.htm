0432 E_DRIVE_BRAKE_DISCONNECTOR_FAILURE

Summary

 This section describes the 0400_0499_DRIVE error E_DRIVE_BRAKE_DISCONNECTOR_FAILURE.

Description

One or more brake power disconnectors (e.g. contactor) does not operate as expected

Extra Info

0: Unknown (not supported)
1: Activation failure
2: Deactivation failure

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 The disconnector is defective (e.g. contacts stuck together) Replace defective disconnector (e.g. SB)
2 The disconnector feedback signal is faulty Check general wiring of disconnector feedback signal (e.g. between SB and drive inverter)
