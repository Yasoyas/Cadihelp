0447 E_DRIVE_CAR_POSITION_CONTROL_DEVIATION

Summary

 This section describes the 0400_0499_DRIVE error E_DRIVE_CAR_POSITION_CONTROL_DEVIATION.

Description

Too high difference between reference and actual car position. The actual car position is provided by the hoistway information.

Extra Info

0: Unknown (not supported)
1: Over Position
2: Under Position

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Too high traction slip
 Check traction means for enough friction
2 Wear of traction means Check traction means for wear
3 Malfunction of car position detection Check hoistway information system (e.g. vanes, PHS)
4 Wrong parameter setting Check parameter (e.g traction sheave diameter) for correct setting
5 Bad balancing of car and counterweight Check balancing according instruction manual (TK)
