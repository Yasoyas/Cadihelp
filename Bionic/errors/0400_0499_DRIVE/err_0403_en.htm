0403 E_DRIVE_CONTACTORS_FEEDBACK

Summary

 This section describes the 0400_0499_DRIVE error E_DRIVE_CONTACTORS_FEEDBACK.

Description

E_DRIVE_CONTACTORS_FEEDBACK

Extra Info

For HY: The car travels to the lowest floor and blocks the elevator. Re- leveling is enabled. In some cases re- leveling upwards is disabled.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 One of the feedback inputs(1) did switch during travelling, or(2) did not switch after starting, or(3) did not switch after stopping. Check: wiring
Check: for contactor fault or input fault in the circuit associated with the input?
2 Emergency stop?
