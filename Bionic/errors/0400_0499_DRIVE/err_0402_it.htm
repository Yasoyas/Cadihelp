0402 E_DRIVE_TRIP_TOO_LONG

Summary

 This section describes the 0400_0499_DRIVE error E_DRIVE_TRIP_TOO_LONG.

Description

E_DRIVE_TRIP_TOO_LONG

Extra Info

Per HY: La cabina arriva al piano più basso e blocca l'ascensore. Il rilivellamento è attivato.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Il tempo di attivazione della corsa supera i limiti ammessi. Controllare: perché cabina è bloccata, corsa bloccata
Controllare: ostruzione della cabina Controllare: se corsa bloccata Controllare: problemi di cablaggio Controllare: perché il freno non si è aperto
Controllare: limite di tempo non corretto, velocità insufficiente Controllare: se informazione vano disponibile

Field Feedback
ID Location Actions to be taken
1 Car / Shaft Controllare : perché la cabina é bloccata. Possibilità di avere ostacoli che ne impediscono il suo movimento.
2 Drive Controllare : perché il drive é bloccato, oppure se il freno non si é aperto. Controllare : cablaggio del freno.
