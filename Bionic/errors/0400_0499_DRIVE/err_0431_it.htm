0431 E_DRIVE_BRAKE_OPERATION_FATAL_FAILURE

Summary

 This section describes the 0400_0499_DRIVE error E_DRIVE_BRAKE_OPERATION_FATAL_FAILURE.

Description

Fatal failure detected at the mechanical drive brake

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Any brake position feedback signal (e.g. KB/KB1) is faulty Check brake position sensors (e.g. KB/KB1) for dirt. Check general wiring of brake position sensors.
2 The brake control circuit is faulty Check contactors controlling the brake (e.g. SB, RB, SF).
Check general wiring to brake actuator.
Check brake modul if present (PCB).
3 Brake operates only partially Check brake supply voltage
4 The brake is mechanically bad adjusted Check for available field information. Replace brake if necessary.
