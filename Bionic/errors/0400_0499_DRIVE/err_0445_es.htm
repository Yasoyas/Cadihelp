0445 E_DRIVE_TRIP_PREPARATION_FAILURE

Summary

 This section describes the 0400_0499_DRIVE error E_DRIVE_TRIP_PREPARATION_FAILURE.

Description

The drive trip preparation wasn't successful due to logical data mismatch

Extra Info

0: Unknown (not supported)
1: Trip profile invalid
2: Pretorque out of range
3: Expected drive inverter preparation state not reached (software hangs)
4: NTSD monitor not set when required

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 The car drive got invalid trip preparation data from the elevator control (e.g. car laod necessary to calculate pretorque out of range, or invalid trip profile) Check drive parameter
Check drive capability
Check load measurement
2 Communication problem between elevator control and car drive Check data connection between elevator control and car drive
3 Software problem (deadlock) at elevator control or at car drive Wait for successful recovery
