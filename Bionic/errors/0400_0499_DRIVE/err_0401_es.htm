0401 E_DRIVE_SAFETY_CHAIN_INTERRUPTED

Summary

 This section describes the 0400_0499_DRIVE error E_DRIVE_SAFETY_CHAIN_INTERRUPTED.

Description

Circuito de seguridad abierto de forma inesperada (p.ej., durante un viaje)

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Algún dispositivo de seguridad se ha disparado (circuito de seguridad abierto) Comprobar el motivo del disparo y resolver el problema
2 Cableado del circuito de seguridad defectuoso o contacto(s) del circuito de seguridad defectuoso(s) o mal ajustados Comprobar el cableado general del circuito de seguridad
y sus contactos
