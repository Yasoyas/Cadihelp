1559 E_FC_POSITION_CORRECTION

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_POSITION_CORRECTION.

Description

E_FC_POSITION_CORRECTION

Extra Info

Nessuna.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Errore di correzione posizione troppo alta richiesta da EC. FC avvia un arresto. Controllare: Par. 11.2 PosCorrectLim e parametri curva s (max. sobbalzo, max. accelerazione, velocità prescritta).
2 Vengono utilizzati valori di sobbalzo e/o di accelerazione troppo elevati. (Par. 1.10, 1.11, 4.2, 4.3, 4.10...4.13). Controllare: per l'applicazione con circuito di regolazione i valori massimi per Par. 1.10, 1.11, 4.2, 4.3 4.10...4.13), che il valore limite è imposto dal sistema meccanico
3 Le funi della cabina slittano sulla puleggia di trazione Controllare: meccanica dell'ascensore (freno, funi, pulegge, ecc.)
4 Immagine vano del EC (comando ascensore)
