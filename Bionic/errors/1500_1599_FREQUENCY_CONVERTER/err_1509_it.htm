1509 E_FC_UNDERVOLTAGE

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_UNDERVOLTAGE.

Description

E_FC_UNDERVOLTAGE

Extra Info

Nessuna.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Voltaggio del bus DC è sceso del 65% sotto il voltaggio nominale
- la causa più comune è l'avaria dell'alimentazione di rete
- anche avaria interna del convertitore di frequenza Vacon può causare un'escursione di voltaggio		 - In caso di interruzione temporanea dell'alimentazione, eliminare il guasto e riavviare.
- Controllare ingresso alimentazione, se l'alimentazione di rete è corretta, si è verificato un difetto interno. Sostituire il convertitore di frequenza.
