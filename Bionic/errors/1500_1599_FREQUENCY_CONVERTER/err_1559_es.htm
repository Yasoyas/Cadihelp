1559 E_FC_POSITION_CORRECTION

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_POSITION_CORRECTION.

Description

E_FC_POSITION_CORRECTION

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Too high position correction error requested by the EC.
FC initiates an stop.		 Check: Par. 11.2 PosCorrectLim and s-curve paramenters (max. jerks, max. acceleration, rated speed).
2 Too high jerk and/ or acceleration values are used. (Par. 1.10, 1.11, 4.2, 4.3, 4.10...4.13). Check: for the closed loop application the max. values for Par. 1.10, 1.11, 4.2, 4.3 4.10...4.13), that limit is imposed by the mechanics
3 Car ropes slips over the traction pulley Check: elevator mechanics (brake, ropes, pulley, etc.)
4 EC shaft immage
