1556 E_FC_OUTPUT_CONTACTORS_SUPERVISOR_PERS

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_OUTPUT_CONTACTORS_SUPERVISOR_PERS.

Description

E_FC_OUTPUT_CONTACTORS_SUPERVISOR_PERS

Extra Info

Nota: Questo è un PERSISTENT FATAL ERROR per il FC.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Uno o più contatti principali SH/SH1 sono fusi. Controllare: se uno o più contatti principali SH/SH1 sono effettivamente fusi.
2 Stato previsto errato mentre contattori sono attivi (in movimento) o durante il tentativo di avviare una corsa (durante tempo di fermo, fase di preparazione partenza) Controllare: ingressi di feedback SH1_STATE e SH1_STATE (DIA1, DIA2) delle uscite di protezione e/o dei corrispondenti contatti ausiliari SH/SH1.
3 Eseguire: risettaggio manuale
