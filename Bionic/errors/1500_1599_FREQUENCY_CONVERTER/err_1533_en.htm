1533 E_FC_SPEED_SUPERVISION

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_SPEED_SUPERVISION.

Description

Too high difference between reference and actual car drive motor speed. The actual speed is derived from the incremental encoder information.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Driving mechanics inhibited or even blocked Check brake for proper opening.
Check driving mechanics for blocking (e.g. gear at geared drive, oiling).
Check for bad weight balancing of car and counterweight.
2 Motor or motor encoder parameter set wrong or speed limit parameter set too narrow Change corresponding parameter accordingly.
Note, at some drives there are several limits dedicated for the different speeds like inspection speed, evacuation speed, etc. (e.g. shaft speed limit/time, shaft service speed limit/time)
3 Malfunction of drive motor rotation encoder Replace the encoder.
