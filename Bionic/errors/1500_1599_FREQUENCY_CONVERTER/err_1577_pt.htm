1577 E_FC_RPM_IDENT_STOPPED

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_RPM_IDENT_STOPPED.

Description

The learning procedure of the drive motor revolution parameter (RPM) was aborted.

Extra Info

ndef

Source ID

ndef

Causes and Actions
ID Possible causes of message Actions to be taken
1 The fitter has cancelled the learning procedure by interaction on the user interface None
2 Mal manipulation by the fitter. Two consecutive trips in the same direction have been initiated. Follow instruction J42101241 (V3 or later) 'Biodyn XX C/P BR Comissioning' or TK.
3 The safety circuit has opened unexpectedly Check where safety circuit was opened. Get rid of the Problem. Restart the drive motor revolution parameter learning procedure.
