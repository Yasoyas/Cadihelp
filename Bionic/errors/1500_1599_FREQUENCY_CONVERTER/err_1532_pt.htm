1532 E_FC_ENCODER_DIRECTION

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_ENCODER_DIRECTION.

Description

Wrong rotation direction signal from the drive motor encoder

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 The rotation direction signaled by the drive motor encoder (differential inputs) is contrary to the drive motor mains phase sequence Change encoder direction parameter (CF 16, PA 14) respectively the drive motor mains phase sequence (CF 16, PA 15)
