1567 E_FC_STANDSTILL

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_STANDSTILL.

Description

L'invertitore dell'azionamento ha rilevato un movimento del motore di azionamento mentre questo doveva essere fermo (freno in posizione chiusa)

Extra Info

Nessuna.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Il freno dell'azionamento non si chiude correttamente Controllare il freno dell'azionamento (ad es. temperatura troppo elevata)
2 Segnale encoder difettoso Controllare il cablaggio generale all'encoder
3 Ricezione del segnale encoder difettosa Sostituire l'elettronica (ad es. scheda opzionale o intero invertitore dell'azionamento) del dispositivo corrispondente
4 Encoder difettoso Sostituire l'encoder

Field Feedback
ID Location Actions to be taken
1 Drive Controllare : Encoder (meccanica e elettrico). Se necessario, rimpiazzarlo.
