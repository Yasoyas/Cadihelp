1567 E_FC_STANDSTILL

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_STANDSTILL.

Description

De aandrijvingsondulator heeft een beweging van de aandrijfmotor waargenomen, terwijl deze in stilstand zou moeten zijn (rem in gesloten positie)

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Aandrijvingsrem sluit niet correct Aandrijvingsrem controleren (bijv. temperatuur te hoog)
2 Encodersignaal is fout Bedrading naar encoder controleren
3 Encodersignaal ontvangststadium defect Elektronica vervangen (bijv. option board of zelfs gehele aandrijvingsondulator) van corresponderend apparaat)
4 Encoder defect Encoder vervangen

Field Feedback
ID Location Actions to be taken
1 Drive Controleer: de encoder (mechanisch en electrisch). Deze eventueel vervangen.
