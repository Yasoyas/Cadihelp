1557 E_FC_OUTPUT_CONTACTORS_SUPERVISOR

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_OUTPUT_CONTACTORS_SUPERVISOR.

Description

One or more drive main contactor does not operate as expected.
Note, this error message might be logged at certain drive types after opening of the safety circuit. In this case no real contactor error exists unless it is confirmed by subsequent logging of error 1556 E_FC_OUTPUT_CONTACTORS_SUPERVISOR_PERS.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Safety circuit has opened When the safety circuit has opened unexpectedly check elevator message log for previous reported messages in order to identify the root cause of the problem
2 The contactor feedback signal is faulty Check on the service interface menu (e.g. 723 or 30623) for the state of the contactor feedback signal.
Check general wiring of contactor feedback signal (e.g. to DIN1 and DIN2 of drive)
3 A contactor is defective (e.g. contacts stuck together) Replace defective contactor (e.g. SFx, SHx)
