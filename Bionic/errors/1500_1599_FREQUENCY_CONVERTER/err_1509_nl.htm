1509 E_FC_UNDERVOLTAGE

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_UNDERVOLTAGE.

Description

Undervoltage at the drive inverter internal DC link detected

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Failure of the mains supply Check mains voltage for disturbances (short breaks) and tolerances
2 Automatic evacuation was running None, normal behavior
3 Drive inverter electronics failure Replace drive inverter
