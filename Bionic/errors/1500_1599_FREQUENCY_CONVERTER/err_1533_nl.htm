1533 E_FC_SPEED_SUPERVISION

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_SPEED_SUPERVISION.

Description

Verschil tussen referentie en werkelijke motorsnelheid kooi te hoog. Actuele snelheid wordt afgeleid van de incrementele encoder informatie

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Aandrijvingsmechanica vertraagd of zelfs geblokkeerd Controleren of rem afdoende opent.
Aandrijvingsmechanica op blokkades controleren (bijv. drijfwerk bij getande overbrenging, smering).
Kooi en tegengewicht controleren op slechte balancering van gewicht.
2 Motor of motor-encoder parameter verkeerd gezet of parameter snelheidsbeperking te nauw ingesteld Corresponderende parameter wijzigen.
Geen, bij sommige some aandrijvingen zijn er diverse limieten aangegeven voor diverse snelheden, zoals inspectiesnelheid, evacuatiesnelheid, etc. (bijv. schachtsnelheid limiet/tijd, schachtservicesnelheid limiet/tijd)
3 Storing rotatie-encoder aandrijfmotor Encoder vervangen.
