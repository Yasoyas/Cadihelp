1512 E_FC_BRAKE_CHOPPER_SUPERVISION

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_BRAKE_CHOPPER_SUPERVISION.

Description

The braking chopper at the drive inverter is not working correctly

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 The brake resistor is not installed correctly
 Check presence and wiring of brake resistor
2 The brake resistor is broken Replace brake resistor
3 The brake chopper is broken Replace drive inverter (FC)
