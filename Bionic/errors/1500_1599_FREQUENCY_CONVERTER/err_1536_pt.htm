1536 E_FC_ANALOG_INPUT_UNDER_CURRENT

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_ANALOG_INPUT_UNDER_CURRENT.

Description

E_FC_ANALOG_INPUT_UNDER_CURRENT

Extra Info

Fault not relevant/ should not appear in Schindler' s Elevator Application.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 The current in the analog input line is below 4 mA.
Signal source has failed.
Control cable is broken.		 Check: mechanical break
Check: make sure that the motor is not running on the current limit.
Check: Increase Parameter 11.8 SpeedErrorLim without surpassing 1/3 of rated output frequency.
