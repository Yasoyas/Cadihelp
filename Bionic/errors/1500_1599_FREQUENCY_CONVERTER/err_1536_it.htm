1536 E_FC_ANALOG_INPUT_UNDER_CURRENT

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_ANALOG_INPUT_UNDER_CURRENT.

Description

E_FC_ANALOG_INPUT_UNDER_CURRENT

Extra Info

Difetto non rilevante/non deve apparire in applicazioni di ascensori Schindler.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 La corrente nella linea di ingresso analogica è sotto 4 mA. Fonte di segnale mancante. Il cavo di comando è rotto. Controllare: rottura meccanica Controllare: accertare che il motore non stia girando al limite di corrente.
