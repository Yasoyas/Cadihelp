1505 E_FC_CHARGING_CONTACTOR

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_CHARGING_CONTACTOR.

Description

The drive inverter has detected a operation failure at the internal DC link

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 EMC disturbances Release elevator from blocking state (reset)
2 Drive inverter internal component (e.g. charging contactor) defective Replace drive inverter
