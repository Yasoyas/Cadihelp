1511 E_FC_OUTPUT_LINE_SUPERVISION

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_OUTPUT_LINE_SUPERVISION.

Description

No current detected at one or more drive inverter output phases to the motor

Extra Info

0: unknown (not supported)
1: Failure at phase 1
2: Failure at phase 2
3: Failure at phase 3

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Power connection between drive inverter and drive motor bad or missing Check power wires and connectors between drive inverter and drive motor
