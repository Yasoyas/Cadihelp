1533 E_FC_SPEED_SUPERVISION

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_SPEED_SUPERVISION.

Description

Differenza troppo grande fra la velocità di riferimento e la velocità reale del motore dell'azionamento cabina. La velocità effettiva viene calcolata in base alle informazioni dell'encoder incrementale.

Extra Info

Nessuna.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 I meccanismi di azionamento sono inibiti o bloccati Controllare la corretta apertura del freno.
Controllare l'eventuale blocco dei meccanismi di azionamento (ad es.ingranaggi del riduttore, lubrificazione).
Controllare l'eventuale errato bilanciamento del peso della cabina e del contrappeso.
2 Errata impostazione del parametro del motore o dell'encoder motore o parametro del limite di velocità impostato con margine troppo scarso Modificare il parametro corrispondente di conseguenza.
Da notare che in alcuni azionamenti sono presenti diversi limiti specifici per le varie velocità, quali la velocità di ispezione, di evacuazione ecc. (ad es. limite/tempo di velocità vano, limite/tempo di velocità manutenzione vano)
3 Malfunzionamento dell'encoder di rotazione del motore dell'azionamento Sostituire l'encoder.

Field Feedback
ID Location Actions to be taken
