1515 E_FC_MOTOR_STALLED

Summary

 This section describes the 1500_1599_FREQUENCY_CONVERTER error E_FC_MOTOR_STALLED.

Description

The load on the drive motor is too high (detected by current measurement on drive inverter output)

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Driving mechanics inhibited or blocked Check driving mechanics for blocking (e.g. gear at geared drive, oiling)
Check brake for proper opening.
2 Sudden heavy load increase Check for impermissible loading
