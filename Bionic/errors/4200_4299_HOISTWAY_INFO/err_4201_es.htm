4201 E_HI_HOISTWAY_END_DETECTION

Summary

 This section describes the 4200_4299_HOISTWAY_INFO error E_HI_HOISTWAY_END_DETECTION.

Description

Failure at the hoistway end detection

Extra Info

0: Unknown (not supported)
1: Failure at upper hoistway end
2: Failure at lower hoistway end

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 A hoistway end position indicator (e.g. KSE, KSE_U, KSE_D) is missing or not at the correct position. Check correct position of the corresponding indicator (e.g. magnet and its polarity, vane)
2 A hoistway end position sensor is faulty. Check functionality of corresponding sensor. Replace it if necessary.
3 Wiring to hoistway end position sensor is faulty. Check wiring at the corresponding sensor.
