4204 E_HI_CAR_POSITION_RECOVERY_INHIBITED

Summary

 This section describes the 4200_4299_HOISTWAY_INFO error E_HI_CAR_POSITION_RECOVERY_INHIBITED.

Description

A recovery from a lost car position is required but the current active function does not allow to perform it.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Sync trip requested while firefighter is active Analyze previous logged error in message log for reason of lost car position. Fix the problem accordingly and release afterwards the elevator from blockage by manual reset on the service interface (menu).
