4203 E_HI_LANDING_ZONE_DETECTION_POSITION_INVALID

Summary

 This section describes the 4200_4299_HOISTWAY_INFO error E_HI_LANDING_ZONE_DETECTION_POSITION_INVALID.

Description

Mismatch between the position of the current landing zone indicators compared to the position learned during the learning travel (hoistway image).

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Landing zone indicator mechanically displaced or even lost Check Landing zone indicator (e.g, vanes, magnets) for correct position in hoistway
