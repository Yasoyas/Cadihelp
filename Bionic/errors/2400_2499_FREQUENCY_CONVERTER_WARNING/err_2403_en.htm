2403 E_FC_TRUE_START_TIMEOUT

Summary

 This section describes the 2400_2499_FREQUENCY_CONVERTER_WARNING error E_FC_TRUE_START_TIMEOUT.

Description

The elevator main controller does not receive the expected accelaration confirmation data telegram from the drive within the expected time (e.g. 2.5s) after a start command.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Different causes Check elevator message log for previous reported messages in order to identify the root cause of the problem.
2 Data transmisssion faulty (e.g. CAN bus) Check general data line connection.
Check for correct data line termination (jumpers and switches, if present) of all devices (PCBs) connected to the the data bus.
Check shielding of data line (if present).
3 The elevator main controller and the drive are out of sync. Perform a reset of the entire system
