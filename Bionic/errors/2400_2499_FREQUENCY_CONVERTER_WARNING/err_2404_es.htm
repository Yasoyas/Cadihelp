2404 E_FC_MOVE_CMD_TIMEOUT_ERROR

Summary

 This section describes the 2400_2499_FREQUENCY_CONVERTER_WARNING error E_FC_MOVE_CMD_TIMEOUT_ERROR.

Description

The elevator main controller does not receive the expected data from the drive node within the expected time after a request.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Different causes Check elevator message log for previous reported messages in order to identify the root cause of the problem.
2 The elevator main controller and the drive are out of sync. Perform a reset of the entire system
