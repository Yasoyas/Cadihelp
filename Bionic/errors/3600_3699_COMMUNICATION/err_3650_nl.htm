3650 E_COM_ELEVATOR_DATA_BUS_TRANSMISSION_DISTURBED

Summary

 This section describes the 3600_3699_COMMUNICATION error E_COM_ELEVATOR_DATA_BUS_TRANSMISSION_DISTURBED.

Description

The data transmission on the elevator data bus is disturbed. Proper elevator operation is not guaranteed anymore.

Note, the elevator data bus connects the elevator control with the elevator essential subsystems (e.g. drive, door).
