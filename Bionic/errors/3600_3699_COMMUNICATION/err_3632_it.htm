3632 E_COM_NODE_CONFIGURATION_FAILURE

Summary

 This section describes the 3600_3699_COMMUNICATION error E_COM_NODE_CONFIGURATION_FAILURE.

Description

The corresponding node is not configured correctly

Extra Info

PCB type (if supported)

Detail reason of failure:
1. Double node identification error
2. Location out of bounds
3. Location not allowed for multiple use
4. Unknown functionality
5. Multiple instances of this I/O not allowed

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Unknown BMK-code, wrong location etc. programmed in corresponding node Check extra info for more details.
Perform teach-in.
