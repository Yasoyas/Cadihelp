3618 E_COM_CAR_NODE_DEAD

Summary

 This section describes the 3600_3699_COMMUNICATION error E_COM_CAR_NODE_DEAD.

Description

The communication to the car node (see extra info) has broken

Extra Info

PCB type (if supported)

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Node disconnected Reconnect node
2 Data transmisssion faulty Check general data line connection.
Check for correct data line termination (if present).
Check shielding of data line (if present).
Check for EMC disturbances.
3 No or bad node power supply Check node power supply
