3631 E_COM_STORED_NODES_IN_LIST_UNSUCCESSFUL

Summary

 This section describes the 3600_3699_COMMUNICATION error E_COM_STORED_NODES_IN_LIST_UNSUCCESSFUL.

Description

There was a problem storing the connected bus nodes in the bus nodes reference list at the elevator main control (e.g. not enough memory space or too many nodes connected on bus). The reference list gets cleared. The storing of the connected bus nodes has to be repeated.
