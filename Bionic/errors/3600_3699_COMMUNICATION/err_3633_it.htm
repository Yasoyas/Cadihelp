3633 E_COM_NODE_COMPATIBILITY_FAILURE

Summary

 This section describes the 3600_3699_COMMUNICATION error E_COM_NODE_COMPATIBILITY_FAILURE.

Description

The corresponding node is unknown respectively not supported by the elevator main controller

Extra Info

PCB type (if supported)

Detail reason of failure:
1. Node authentication failed
2. Incompatible firmware version
3. Incompatible hardware version
4. Incompatible protocol version

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Connected node not compatible with main elevator main controller Check extra info for details.
Replace node.
