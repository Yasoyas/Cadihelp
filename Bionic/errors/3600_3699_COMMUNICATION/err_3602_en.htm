3602 E_LOP_COUNT_FAILED

Summary

 This section describes the 3600_3699_COMMUNICATION error E_LOP_COUNT_FAILED.

Description

After configuration of the bus nodes (e.g. landing operating panels) a check of all possible node adresses is performed and the corresponding IO function mapping get read. This error occurs if there is a communication problem to any bus node during this phase.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Bad connection to any bus node (e.g. landing operating panel) Check generally bus wiring. Check bus connector on controller main board and at all bus nodes.
Redo LOP count (CF00, LE00).
2 Bad bus power supply Check supply voltage on bus for instability.
Redo LOP count (CF00, LE00).
3 EMC problems Check entire bus for interferences.
Redo LOP count (CF00, LE00).
