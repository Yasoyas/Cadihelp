4406 E_UI_MACHINE_ROOM_INSPECTION_CMD_UNEXPECTED

Summary

 This section describes the 4400_4499_USER_INTERFACE error E_UI_MACHINE_ROOM_INSPECTION_CMD_UNEXPECTED.

Description

The elevator has detected a car move command from the machine room inspection panel while the machine room inspection operation wasn't active

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Maloperation by the user Do not give travel up/down commands while corresponding inspection operation isn't active
2 A car move command signal (e.g. DRH-U, DRH-D) is faulty Check general wiring of inspection panel
