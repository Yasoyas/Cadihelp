0314 E_DOOR_PRE_OPENING

Summary

 This section describes the 0300_0399_CAR_DOOR error E_DOOR_PRE_OPENING.

Description

Si è verificato un difetto di attivazione o disattivazione del dispositivo di bypass del circuito di sicurezza della porta mentre l'ascensore stava per effettuare l'apertura anticipata della porta.

Extra Info

Per HY: La cabina arriva al piano più basso e blocca l'ascensore. Il rilivellamento è attivato.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Cablaggio del dispositivo di bypass del circuito di sicurezza porta difettoso Controllare il cablaggio generale del dispositivo di bypass del circuito di sicurezza porta
2 Rilevazione difettosa della zona porte Controllare i segnali e i sensori della zona porte (ad es. PHSx, PHUETx).
Controllare la posizione degli indicatori della zona di sblocco della porta nel vano di corsa (ad es. lame a livello)
Sostituire il PCB contenente il meccanismo di bypass (ad es.SUET)
3 Guasto nel dispositivo di bypass del circuito di sicurezza della porta (KTS, KTC) Sostituire il PCB contenente il meccanismo di bypass (ad es.SUET)

Field Feedback
ID Location Actions to be taken
1 Shaft Controllare la distanza del magnete KUET1/KUET2 (identico a PHUET) sullo stesso livello. Regolarli se necessario.
2 Car Roof Controllare le connessioni sul SUET.
