0315 E_DOOR_NOT_RECOVERABLE

Summary

 This section describes the 0300_0399_CAR_DOOR error E_DOOR_NOT_RECOVERABLE.

Description

La porta non ha potuto ripristinare l'errore di apertura o chiusura porte (generalmente, dopo 20 tentativi ripetuti di apertura e chiusura). L'ascensore è bloccato.

Extra Info

Per HY: La cabina arriva al piano più basso e blocca l'ascensore. Il rilivellamento è attivato.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 La porta presenta un problema che causa errori ripetuti di 'apertura porta' o 'chiusura porta'. Controllare il motivo del mancato ripristino della porta.
Controllare le cause e le azioni previste per gli errori 0301 e 0302.

Field Feedback
ID Location Actions to be taken
1 Car Door / Shaft Door Controllare : regolaggio meccanico della porta (tutti i modelli). Controllare : encoder, cinghia e pignone di trasmissione (V20, VVVF4, Fermator)
