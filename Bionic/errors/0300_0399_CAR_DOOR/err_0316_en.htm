0316 E_DOOR_HEARTBEAT_ERROR

Summary

 This section describes the 0300_0399_CAR_DOOR error E_DOOR_HEARTBEAT_ERROR.

Description

The communication to the door node respectively car node (controller) has broken

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Door node respectively car node disconnected Reconnect node
2 No or bad door node respectively car node power supply Check node power supply
3 Data transmisssion faulty Check general data line connection.
Check for correct data line termination (if present) of all devices connected to the the data bus.
Check shielding of data line (if present).
Check for EMC disturbances.
4 Door node respectively car node defective Replace corresponding node
