0318 E_DOOR_SDIC_HEARTBEAT

Summary

 This section describes the 0300_0399_CAR_DOOR error E_DOOR_SDIC_HEARTBEAT.

Description

E_DOOR_SDIC_HEARTBEAT

Extra Info

Per HY: Non è disponibile alcuna informazione vano. Parcheggio di emergenza a livello 1.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 SDIC rileva mancanza heartbeat SCIC e passa in stato di errore. Il SDIC invia questo messaggio. Controllare cavo bus CAN e terminale del cavo del bus CAN su tutti i sottosistemi (SCIC; SDIC; SEM; FC; COP).
2 Eventuale errata combinazione di versioni SW.
