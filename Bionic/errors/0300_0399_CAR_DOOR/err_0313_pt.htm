0313 E_DOOR_SHAFT_ERROR

Summary

 This section describes the 0300_0399_CAR_DOOR error E_DOOR_SHAFT_ERROR.

Description

O subsistema da porta recebeu um comando de porta aberta apesar de a cabina não ter sido detectada no piso. O comando é rejeitado.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Diferentes causas Verificar a existência de mensagens anteriores incluídas em relatório no registo de mensagens do ascensor para identificar a causa principal do problema.
2 Falha na detecção da zona de porta do sistema de informação de caixa Verificar a detecção da zona de porta do sistema de informação de caixa (por ex. PHSx, ajuste mecânico, sensor com defeito, etc.)
