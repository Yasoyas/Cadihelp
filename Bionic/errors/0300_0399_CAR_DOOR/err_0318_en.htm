0318 E_DOOR_SDIC_HEARTBEAT

Summary

 This section describes the 0300_0399_CAR_DOOR error E_DOOR_SDIC_HEARTBEAT.

Description

The car node has recognized a lost of communication to the elevator main control.
Note, this error mainly occurs together with other errors. Please check message log first for other reported errors.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Car node disconnected (e.g. CAN bus) Reconnect node
2 Data transmisssion faulty (e.g. CAN bus) Check general data line connection.
Check for correct data line termination (jumpers and switches, if present) of all devices (PCBs) connected to the the data bus.
Check shielding of data line (if present).
Check for EMC disturbances.
3 No or bad power supply of elevator main control Check power supply
4 Mismatch of car node software and elevator main controller software Update the software accordingly
5 Elevator main control defective Replace corresponding hardware
