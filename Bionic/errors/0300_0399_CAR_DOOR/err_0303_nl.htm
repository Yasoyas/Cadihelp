0303 E_DOOR_MAX_LOCK_TIME

Summary

 This section describes the 0300_0399_CAR_DOOR error E_DOOR_MAX_LOCK_TIME.

Description

A door was for a too long period of time in locked position. Note, monitoring the time the door is in locked position prevents the motor from burning.

Note, this error might occur while the car is moving slowly (e.g. during learning travel or emergency power recall travel)

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Parameter 'max door lock time' is set wrong Check corresponding parameter (CF03 PA07)
