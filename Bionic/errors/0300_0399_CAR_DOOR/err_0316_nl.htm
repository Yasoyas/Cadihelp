0316 E_DOOR_HEARTBEAT_ERROR

Summary

 This section describes the 0300_0399_CAR_DOOR error E_DOOR_HEARTBEAT_ERROR.

Description

De communicatie met het knooppunt van de deur, resp. van de kooi (controller) is verbroken

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Knooppunt deur, resp. knooppunt kooi verbroken Knooppunt opnieuw verbinden
2 Geen knooppunt of slecht knooppunt, resp. stroomtoevoer kooiknooppunt verbroken. Stroomtoevoer knooppunt controleren
3 Gegevensoverdracht storing Algemene gegevensverbinding controleren.
Correcte afsluiten gegevenskabel controleren (indien aanwezig) van alle apparaten die aangesloten zijn aan de gegevensbus.
Mantel gegevenskabel (indien aanwezig).
Controleren op EMC-storingen.
4 Knooppunt deur, resp. knooppunt kooi defect Betreffende knooppunt vervangen

Field Feedback
ID Location Actions to be taken
1 Controller / MR Controleer: CAN-bus kabel en connectoren van de subsystemen (SCIC,FC)
2 Car / Shaft Controleer: CAN-bus kabel en connectoren van de subsystemen (SDIC,COP, SEM)
3 Car / Controller Controleer of Software van SDIC en SCIC compatible is. Zo niet dan dient deze uitgewisseld te worden
