0317 E_DOOR_UNEXPECTED_STATE

Summary

 This section describes the 0300_0399_CAR_DOOR error E_DOOR_UNEXPECTED_STATE.

Description

E_DOOR_UNEXPECTED_STATE

Extra Info

Nessuna.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Se EC riceve una sequenza di stato della porta LOCKED -> CLOSED vi è qualcosa di errato nel sottosistema della porta. Controllare: contatto instabile nell'alimentazione di SDIC
2 Fa scattare il Door Heatbeat Service e imposta il SDIC stato di errore.
3 Eventuale errata combinazione di versioni SW.
