0301 E_DOOR_CLOSING

Summary

 This section describes the 0300_0399_CAR_DOOR error E_DOOR_CLOSING.

Description

The door wasn't able to close successfully within a certain time limit. As a typical response a door recovery gets started (repetitive opening and closing of the door).

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Door closed position feedback signal is faulty Check door closed position contact (e.g. KET-S, if present and not emulated, depending on door type) for dirt or bad mechanical adjustment.
Check general wiring door closed position contact.
2 Door doesn't move or moves to slow Check for obstacle or dirt at door shoes and tracks.
Check door mechanism (broken belt, mechanical coupling, dirt).
Check door motor power supply (fuses).
3 Door close parameter set wrong Check door close parameter for correct setting
