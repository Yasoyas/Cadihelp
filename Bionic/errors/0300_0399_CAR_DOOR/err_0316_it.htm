0316 E_DOOR_HEARTBEAT_ERROR

Summary

 This section describes the 0300_0399_CAR_DOOR error E_DOOR_HEARTBEAT_ERROR.

Description

La comunicazione al nodo porta o al nodo cabina (manovra) si è interrotta

Extra Info

Per HY: La cabina arriva al piano più basso e blocca l'ascensore.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Nodo porta o nodo cabina scollegato Ricollegare il nodo
2 Assenza o errata alimentazione del nodo porta o del nodo cabina Verificare l'alimentazione del nodo
3 Trasmissione dati difettosa Controllare il collegamento della linea dati generale.
Controllare la corretta terminazione della linea dati (se presente) di tutti i dispositivi collegati al bus dati.
Controllare la schermatura della linea dati (se presente).
Verificare la presenza di eventuali disturbi CEM.
4 Nodo porta o nodo cabina difettoso Sostituire il nodo corrispondente

Field Feedback
ID Location Actions to be taken
1 Controller / MR Controllare : Cavo CAN-Bus e terminazioni su tutti i subsistemi (SCIC, FC)
2 Car / Shaft Controllare : Cavi CAN-Bus e terminali CAN-Bus su tutti i subsistemi (SDIC, COP, SEM)
3 Car / Controller Controllare se il Software SDIC e SCIC sono compatibili. Se necessario, cambiarlo.
