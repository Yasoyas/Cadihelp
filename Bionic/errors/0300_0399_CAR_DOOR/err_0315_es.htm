0315 E_DOOR_NOT_RECOVERABLE

Summary

 This section describes the 0300_0399_CAR_DOOR error E_DOOR_NOT_RECOVERABLE.

Description

La puerta no pudo recuperarse de un error de apertura o cierre (normalmente, después de 20 intentos de apertura y cierre repetidos). El ascensor está bloqueado.

Extra Info

ndef

Source ID

ndef

Causes and Actions
ID Possible causes of message Actions to be taken
1 La puerta tiene un problema que provoca errores de "apertura" o "cierre" de puerta. Comprobar por qué la puerta no puede recuperarse.
Comprobar la causa de los errores 0301 y 0302 y las acciones a adoptar.

Field Feedback
ID Location Actions to be taken
1 Car Door / Shaft Door Comprobar el ajuste mecánico del sistema de puertas (todos los modelos). Comprobar encoder, correa principal y piñón de transmisión (V20, VVVF4 Fermator)
