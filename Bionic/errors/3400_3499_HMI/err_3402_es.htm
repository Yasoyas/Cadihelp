3402 E_HMI_VALUE_OUT_OF_UPPER_BOUND

Summary

 This section describes the 3400_3499_HMI error E_HMI_VALUE_OUT_OF_UPPER_BOUND.

Description

While parameterizing, the entered value on the elevator user interface is above the allowed maximum value
