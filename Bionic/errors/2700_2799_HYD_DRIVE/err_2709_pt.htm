2709 E_HYD_DRIVE_SPEED_LIMIT_EXCEEDED_IN_STANDBY

Summary

 This section describes the 2700_2799_HYD_DRIVE error E_HYD_DRIVE_SPEED_LIMIT_EXCEEDED_IN_STANDBY.

Description

E_HYD_DRIVE_SPEED_LIMIT_EXCEEDED_IN_STANDBY

Extra Info

For HY: The car travels to the lowest floor and blocks the elevator. Re- leveling is disabled.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 speed limit (0.3 m/s) exceeded (for pre- opening or releveling) none
