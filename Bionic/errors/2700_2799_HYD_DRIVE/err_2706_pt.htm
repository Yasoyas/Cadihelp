2706 E_HYD_DRIVE_TRIP_CONTACTOR_FEEDBACK_AFTER_DEACTIVA

Summary

 This section describes the 2700_2799_HYD_DRIVE error E_HYD_DRIVE_TRIP_CONTACTOR_FEEDBACK_AFTER_DEACTIVA.

Description

E_HYD_DRIVE_TRIP_CONTACTOR_FEEDBACK_AFTER_DEACTIVATION

Extra Info

For HY: The car travels to the lowest floor and blocks the elevator. Re- leveling is disabled.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Feedback of Trip contactor is ON after the contactor is deactivated. Check: Trip contactor
