1128 E_LMS_ENABLED

Summary

 This section describes the 1100_1199_CAR_LOAD_CELL error E_LMS_ENABLED.

Description

The car load measuring device has been enabled (manually by the user or automatically by the system after elapsed timeout). Its measured values are used for the elevator's operation.

Extra Info

0: Unknown
1: Manual enable (by user)
2: Auto enable (by system)
