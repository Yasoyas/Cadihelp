1105 E_CLC_NO_CALIBRATION

Summary

 This section describes the 1100_1199_CAR_LOAD_CELL error E_CLC_NO_CALIBRATION.

Description

The elevator system requires measuring values from the car load measuring system but it isn't calibrated

Extra Info

none

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 No calibration of the load measuring system (e.g. CLC) has been done Do respectively redo calibration of load measuring system
