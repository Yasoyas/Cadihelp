1108 E_CLC_OUT_OF_RANGE

Summary

 This section describes the 1100_1199_CAR_LOAD_CELL error E_CLC_OUT_OF_RANGE.

Description

Het signaal van het meetapparaat kooibelasting is aanwezig, maar ongeldig (bijv. signaal buiten bereik)

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Meetapparaat kooibelasting slecht ingesteld Mechanische bevestiging meetapparaat kooibelasting controleren
2 Meetsignaal kooibelasting ongeldig Bedrading naar meetapparaat kooibelasting controleren.
Afscherming controleren (indien aanwezig).
Stroomtoevoer meetapparaat kooibelasting.
3 Meetapparaat kooibelasting functioneert slecht Meetapparaat kooibelasting vervangen
