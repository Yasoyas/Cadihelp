1110 E_CLC_CALIBRATION_REFERENCE_LOAD_FREQUENCY_OUT_OF_

Summary

 This section describes the 1100_1199_CAR_LOAD_CELL error E_CLC_CALIBRATION_REFERENCE_LOAD_FREQUENCY_OUT_OF_.

Description

Kalibratie van referentiële kooibelasting is mislukt door een ongeldig signaal (frequentie) van het meetapparaat kooibelasting

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Meetapparaat kooibelasting slecht ingesteld Mechanische bevestiging meetapparaat kooibelasting controleren
2 Meetsignaal kooibelasting ongeldig Bedrading naar meetapparaat kooibelasting controleren.
Controleren op EMC-storingen.
3 Incorrecte werking tijdens kalibreren meetapparaat kooibelasting Op correcte procedure controleren (bijv. correct gewicht in kooi)
4 Meetapparaat kooibelasting functioneert slecht Meetapparaat kooibelasting vervangen
