4008 E_AAT_ABORTED_NORMAL_BRAKE_CLOSED_UNEXPECTED

Summary

 This section describes the 4000_4099_AAT_2 error E_AAT_ABORTED_NORMAL_BRAKE_CLOSED_UNEXPECTED.

Description

The running acceptance test has been aborted because the normal brake has closed unexpectedly during the test. (E.g. because the brake close inhibit plug (MBG) wasn't mounted by the fitter)
