4007 E_AAT_ABORTED_CAR_TEST_LOAD_NOT_PRESENT

Summary

 This section describes the 4000_4099_AAT_2 error E_AAT_ABORTED_CAR_TEST_LOAD_NOT_PRESENT.

Description

The running acceptance test has been aborted because the corresponding car load necessary for the test execution is not present. See user manual for correct handling.
