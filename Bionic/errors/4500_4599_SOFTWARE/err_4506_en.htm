4506 E_SW_NODE_FIRMWARE_DOWNLOAD_DATA_NOT_FOUND

Summary

 This section describes the 4500_4599_SOFTWARE error E_SW_NODE_FIRMWARE_DOWNLOAD_DATA_NOT_FOUND.

Description

At least one of the expected data files for the firmware update of the subsystems (e.g. drive, door) is missing at elevator main control.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Last downloaded elevator software contains invalid data. Get data carrier with correct data, insert it and re-perform a elevator software download.
