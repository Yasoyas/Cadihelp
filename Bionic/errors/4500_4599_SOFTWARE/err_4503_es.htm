4503 E_SW_DOWNLOAD_DATA_INCOMPATIBLE

Summary

 This section describes the 4500_4599_SOFTWARE error E_SW_DOWNLOAD_DATA_INCOMPATIBLE.

Description

The data on the data carrier is incompatible and therefore impossible to be downloaded on the elevator system. See extra info for more information.

Extra Info

0: Unknown (not supported)
1: Incompatible data structure or content (e.g. xml-file <-> xsd-file)
2: Incompatible data signature (encryption)

10: Incompatible software version
11: Incompatible target hardware version

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Wrong data carrier inserted Get right data carrier, insert it and re-perform download.
