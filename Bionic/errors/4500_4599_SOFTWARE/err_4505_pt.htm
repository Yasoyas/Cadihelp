4505 E_SW_DOWNLOAD_DATA_NOT_FOUND

Summary

 This section describes the 4500_4599_SOFTWARE error E_SW_DOWNLOAD_DATA_NOT_FOUND.

Description

At least one of the expected data files for the data download (e.g. software, configuration) is missing on the data carrier

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Wrong data carrier inserted Get right data carrier, insert it and re-perform a download.
