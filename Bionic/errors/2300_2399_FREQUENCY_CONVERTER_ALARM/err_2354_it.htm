2354 E_FC_ALARM_BRAKING_RESISTOR_OVERTEMPERATURE

Summary

 This section describes the 2300_2399_FREQUENCY_CONVERTER_ALARM error E_FC_ALARM_BRAKING_RESISTOR_OVERTEMPERATURE.

Description

E_FC_ALARM_BRAKING_RESISTOR_OVERTEMPERATURE

Extra Info

Nessuna.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Interruttore termico bimetallico resistore del freno. Il circuito di sorveglianza temperatura del resistore del freno ha aperto. Se l'ascensore sta effettuando una corsa, questa in genere viene portata a termine. Una nuova corsa non può essere avviata prima che l'interruttore termico bimetallico non si sia richiuso (5 min.) Controllare temperatura resistori del freno
Controllare cablaggio e/o interruttore termico bimetallico
