2316 E_FC_ALARM_MOTOR_OVERTEMPERATURE

Summary

 This section describes the 2300_2399_FREQUENCY_CONVERTER_ALARM error E_FC_ALARM_MOTOR_OVERTEMPERATURE.

Description

A máquina de tracção da transmissão da cabina excedeu a temperatura de funcionamento

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 A dissipação de calor não está a funcionar Verificar funcionamento do dispositivo de dissipação de calor (por ex. ventilador ou ventilação forçada), se existente
2 Temperatura do ar ambiente demasiado quente (por ex. luz solar directa na caixa de vidro) Aguardar o arrefecimento
3 Falhas no sinal de retorno de temperatura Verificar cablagem geral para o sensor de temperatura.
Verificar funcionamento do sensor de temperatura.
4 Funcionamento demasiado intensivo (por ex. demasiadas viagens por unidade de tempo) Aguardar o arrefecimento
