2316 E_FC_ALARM_MOTOR_OVERTEMPERATURE

Summary

 This section describes the 2300_2399_FREQUENCY_CONVERTER_ALARM error E_FC_ALARM_MOTOR_OVERTEMPERATURE.

Description

La macchina motrice dell'azionamento cabina ha superato la temperatura di esercizio

Extra Info

Nessuna.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Il dispositivo di dissipazione del calore non funziona Controllare il funzionamento del dispositivo di dissipazione del calore (ad es. ventilatore o ventilazione forzata), se presente
2 Temperatura dell'aria ambiente troppo elevata (ad es. luce solare diretta sul vano in vetro) Attendere il raffreddamento
3 Il segnale di ritorno della temperatura è difettoso Controllare il cablaggio generale al sensore di temperatura.
Controllare il funzionamento del sensore di temperatura.
4 Funzionamento troppo intensivo (ad es. numero di corse troppo elevato per unità di tempo) Attendere il raffreddamento
