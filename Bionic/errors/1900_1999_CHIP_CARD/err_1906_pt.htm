1906 E_CHIP_CARD_READING_ERROR

Summary

 This section describes the 1900_1999_CHIP_CARD error E_CHIP_CARD_READING_ERROR.

Description

There is a problem reading the SIM card. Note, this error typically relates to any other SIM card error.

ndef

ID Possible causes of message Actions to be taken
1 SIM card reading error Check error log for other SIM card errors. Please refer to corresponding causes and actions. If no other error is reported, replace SIM card.
