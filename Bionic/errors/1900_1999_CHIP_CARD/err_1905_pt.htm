1905 E_CHIP_CARD_NO_OR_NO_SCHINDLER_CARD

Summary

 This section describes the 1900_1999_CHIP_CARD error E_CHIP_CARD_NO_OR_NO_SCHINDLER_CARD.

Description

No or invalid SIM card inserted on the elevator main controller board.

Extra Info

ndef

Source ID

ndef

Causes and Actions
ID Possible causes of message Actions to be taken
1 No SIM card inserted
 Get right SIM card, insert it and perform a elevator main controller reset.
2 The SIM Card is inserted wrongly Remove SIM card, insert it correctly and perform a elevator main controller reset.
3 No Schindler SIM Card inserted Get right SIM card, insert it and perform a elevator main controller reset.
