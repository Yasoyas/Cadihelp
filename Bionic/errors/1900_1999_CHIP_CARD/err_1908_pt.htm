1908 E_CHIP_CARD_WRONG_FILE_SYSTEM_VERSION

Summary

 This section describes the 1900_1999_CHIP_CARD error E_CHIP_CARD_WRONG_FILE_SYSTEM_VERSION.

Description

Invalid Schindler SIM card inserted on the elevator main controller board.

Extra Info

ndef

Source ID

ndef

Causes and Actions
ID Possible causes of message Actions to be taken
1 SIM card has a wrong file system version respectively a wrong data format. This means the SIM card does not contain the data (e.g. FC parameters, P-Care data structure) as expected by the elevator main controller (SCIC). Get right SIM card, insert it and perform an elevator main controller reset.
