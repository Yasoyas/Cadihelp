0069 E_ELEVATOR_INSPECTION_MOVE_COMMAND_UNEXPECTED

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_INSPECTION_MOVE_COMMAND_UNEXPECTED.

Description

Probably OLD. To be deleted...... TB 24.10.07

The elevator has detected a car move command from any of the inspection panels (machine room, top-of-car, terminal landing) while the corresponding inspection operation wasn't activated.

Extra Info

0: Unknown (not supported)
1: Failure at top-of-car inspection panel
2: Failure at machine room inspection panel
3: Failure at terminal landing inspection panel

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Maloperation by the user Do not give travel up/down commands while corresponding inspection operation isn't active
2 A car move command signal (e.g. DREC-U, DREC-D) is faulty Check wiring of inspection panel
