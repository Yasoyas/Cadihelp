0031 E_ELEVATOR_LEARNING_INTERRUPTED

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_LEARNING_INTERRUPTED.

Description

The learning travel has failed.

Extra Info

ndef

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Wrong manipulation on elevator. E.g. Recall switched on or any stop switch pressed during learning travel. Release corresponding switches. Re-execute the learning travel.
2 Safety circuit opened Check reason for open safety circuit and get rid of the problem. Re-execute the learning travel.
