0063 E_ELEVATOR_BACKUP_BAT_INSUFFICIENT_CHARGE

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_BACKUP_BAT_INSUFFICIENT_CHARGE.

Description

The charge of the elevator backup battery has fallen below the required minimum level or wasn't able to get recharged to the required minimum level after elevator mains power up.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Battery connection missing or bad Check battery's connections
2 Battery faulty (e.g. old) Replace battery
3 Battery charging problem Check wiring to charging device.
Check function of charging device (charging voltage, fuses).
