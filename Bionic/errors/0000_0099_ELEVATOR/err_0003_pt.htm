0003 E_ELEVATOR_OVERLOAD_MODE

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_OVERLOAD_MODE.

Description

The elevator has detected overload in the car.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Too high load in the car Decrease the load in the car
2 The car load measuring signal is faulty Check general wiring to the car load measuring device.
Redo calibration if signal is present but invalid.
