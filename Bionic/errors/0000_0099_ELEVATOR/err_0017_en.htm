0017 E_ELEVATOR_REVISION_NUMBERS_DO_NOT_MATCH

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_REVISION_NUMBERS_DO_NOT_MATCH.

Description

The commissioning number stored on the SIM card doesn't match with the current elevator commissioning number.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Wrong SIM card inserted (e.g. wrong delivery or SIM card from another elevator). Get right SIM card, insert it and perform an elevator main controller reset.
