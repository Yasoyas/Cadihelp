0001 E_ELEVATOR_FATAL_ERROR

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_FATAL_ERROR.

Description

De lift is peramanent geblokkeerd en functioneert niet.
Opmerking: deze melding volgt meestal op een andere foutmelding.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Diverse oorzaken Controleer in het log-bestand van de lift of er voorafgaande meldingen zijn. Hieruit het probleem worden vastgesteld.
