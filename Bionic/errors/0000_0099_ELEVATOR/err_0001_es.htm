0001 E_ELEVATOR_FATAL_ERROR

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_FATAL_ERROR.

Description

El ascensor está permanentemente bloqueado e inoperativo.
Nota, normalmente este mensaje sigue a otro error fatal.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Diferentes causas Comprobar los mensajes comunicados anteriormente en el registro de mensajes del ascensor para identificar el origen del problema.

Field Feedback
ID Location Actions to be taken
