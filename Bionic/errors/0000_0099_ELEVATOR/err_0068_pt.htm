0068 E_ELEVATOR_INSPECTION_ACTIVATION_SIGNAL_INVALID

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_INSPECTION_ACTIVATION_SIGNAL_INVALID.

Description

Probably OLD. To be deleted...... TB 24.10.07

The input signals of any of the inspection panels (machine room, top-of-car, terminal landing) are invalid (e.g. implausible signal combination). See extra info for more information.

Extra Info

0: Unknown (not supported)
1: Failure at top-of-car inspection panel
2: Failure at machine room inspection panel
3: Failure at terminal landing inspection panel

ID Possible causes of message Actions to be taken
