0012 E_ELEVATOR_NOAUTHORIZATION_MODE

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_NOAUTHORIZATION_MODE.

Description

The elevator is blocked because no or an invalid SIM card is inserted at the elevator main control board (PCB).
