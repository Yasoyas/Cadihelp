0012 E_ELEVATOR_NOAUTHORIZATION_MODE

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_NOAUTHORIZATION_MODE.

Description

E_ELEVATOR_NOAUTHORIZATION_MODE

Extra Info

Per HY: La cabina è bloccata al piano più basso.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Inviato se si avvia il Noauthorization service
2 Scheda SIM e PCB non coincidono Prendere la scheda SIM corretta.

Field Feedback
ID Location Actions to be taken
1 Controller Controllare se la SIM é presente, oppure se é stata manipolata. Procurare la SIM corretta.
