0069 E_ELEVATOR_REDUCED_DRIVE_FAILURE_DETECT_FINISHED

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_REDUCED_DRIVE_FAILURE_DETECT_FINISHED.

Description

The reduced drive failure detection mode is finished. The drive can not be used in open loop mode any more, failure monitoring of the drive is restored.
