0017 E_ELEVATOR_REVISION_NUMBERS_DO_NOT_MATCH

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_REVISION_NUMBERS_DO_NOT_MATCH.

Description

E_ELEVATOR_REVISION_NUMBERS_DO_NOT_MATCH

Extra Info

Nessuna.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 SIMCARD disponibile, ma il numero COMM interno non corrisponde con quello SCIC. Fornitura di SIMCARD errata o SCIC già utilizzata in un altro ascensore. Sostituire con SIMCARD apposita (specificatamente configurata) per questa installazione. Installare la combinazione SCIC. SIMCARD prevista per questa installazione.
