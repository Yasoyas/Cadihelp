0055 E_ELEVATOR_STARTUP_FAILURE

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_STARTUP_FAILURE.

Description

The elevator was not able to start up within a predefined time (typ. 10s). The elevator control could not read the state of all configured system resources, safety and critical inputs. At least one subsystem relevant for the elevator's operation is not available.
