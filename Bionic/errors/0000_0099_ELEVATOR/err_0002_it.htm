0002 E_ELEVATOR_SAFETY_CHAIN

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_SAFETY_CHAIN.

Description

E_ELEVATOR_SAFETY_CHAIN

Extra Info

Per HY: se la catena di sicurezza viene interrotta durante il rilivellamento, l'errore fatale sarà duraturo.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Inviato dal SafetyChain service
2 ISK deve essere chiuso almeno 800 ms dopo KET_S. In caso contrario, si avvia il Safety Chain service. (KET_S + 500 ms = LOCKED + 300 ms -> ISK è previsto) Controllare il circuito Safety Chain. Controllare il timing KET_S -> ISK. Controllare il fusibile 110V sulla scheda SMIC.
