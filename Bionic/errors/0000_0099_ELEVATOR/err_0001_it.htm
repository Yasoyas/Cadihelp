0001 E_ELEVATOR_FATAL_ERROR

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_FATAL_ERROR.

Description

L'ascensore rimane bloccato e non è azionabile.
Da notare che questo messaggio è generalmente preceduto da un altro errore.

Extra Info

Per HY: La cabina arriva al piano più basso e blocca l'ascensore. Il rilivellamento è attivato.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Cause diverse Controllare se sono presenti precedenti messaggi nel registro messaggi dell'ascensore al fine di identificare la causa del problema.

Field Feedback
ID Location Actions to be taken
1 Controller Premere tasto RESET sul print SCIC.
