0003 E_ELEVATOR_OVERLOAD_MODE

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_OVERLOAD_MODE.

Description

E_ELEVATOR_OVERLOAD_MODE

Extra Info

Per HY: Il rilivellamento è attivato.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Sovraccarico individuato dalla cellula di carico cabina (CLC) Controllare il circuito CLC; se non vi è sovraccarico: eseguire una nuova calibratura del circuito CLC
