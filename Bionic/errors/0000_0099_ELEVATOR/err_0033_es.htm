0033 E_ELEVATOR_STARTED_UP

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_STARTED_UP.

Description

Indicates that the main controller has just started up.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 The power got switched off and on again None
2 The reset button got pressed None
3 A watchdog reset got performed Check first power supply (quality) and possible reset by other staff. If message appears unexpectedly update the main controller with new software release.
