0019 E_ELEVATOR_WATCHDOG_RESET

Summary

 This section describes the 0000_0099_ELEVATOR error E_ELEVATOR_WATCHDOG_RESET.

Description

Indicates that a watchdog reset (initiated by hardware or software watchdog) has been performed previously. The elevator main control has just started up.
