9042 S_NO_OPERATION_DUE_TO_INVALID_LOAD_CONFIGURATION

Summary

 This section describes the 9000_9099_ELEVATOR_STATUS_1 error S_NO_OPERATION_DUE_TO_INVALID_LOAD_CONFIGURATION.

Description

The elevator is blocked because the configuration of the load measurement is invalid or missing (e.g. car load measurement device not calibrated).
