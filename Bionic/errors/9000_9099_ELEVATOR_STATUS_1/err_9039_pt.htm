9039 S_NO_OPERATION_DUE_TO_CAR_OVERLOAD

Summary

 This section describes the 9000_9099_ELEVATOR_STATUS_1 error S_NO_OPERATION_DUE_TO_CAR_OVERLOAD.

Description

The elevator is not operating because the car is overloaded.

Note, the elevator only changes into this state in case the overload condition persists too long.
