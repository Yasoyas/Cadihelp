4391 E_SAFETY_DEVICE_BYPASS_OF_IN_CAR_STOP_SWITCH

Summary

 This section describes the 4300_4399_SAFETY_DEVICE error E_SAFETY_DEVICE_BYPASS_OF_IN_CAR_STOP_SWITCH.

Description

There was an activation or deactivation failure of the bypass of the in-car stop switch.

Note, this bypass is normally activated in fire recall operation. In any other operation modes this bypass is expected to be deactivated.

Extra Info

0: Unknown (not supported)
1: Bypass activation failure
2: Bypass deactivation failure

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Failure at device which bypasses the in-car stop switch (e.g. JHC1) Replace PCB containing this bypass mechanism (e.g. CANCNA)
