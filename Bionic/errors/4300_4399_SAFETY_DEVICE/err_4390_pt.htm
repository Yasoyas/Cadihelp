4390 E_SAFETY_DEVICE_BYPASS_OF_DOOR_CLOSED_DETECTION_HA

Summary

 This section describes the 4300_4399_SAFETY_DEVICE error E_SAFETY_DEVICE_BYPASS_OF_DOOR_CLOSED_DETECTION_HA.

Description

There was an activation or deactivation failure of the bypass of the car door closed detection device and the terminal landing door closed detection devices.

Note, this bypass is normally activated in hoistway end inspection operation (hoistway access operation mode). In any other operation modes this bypass is expected to be deactivated.

Extra Info

0: Unknown (not supported)
1: Bypass activation failure
2: Bypass deactivation failure

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Failure at device which bypasses the car door closed detection (e.g. KTC) and the terminal landing door locked detection (e.g. KTS) Replace PCB (e.g. WDPNA) containing this bypass mechanism (e.g. RACCESS)
