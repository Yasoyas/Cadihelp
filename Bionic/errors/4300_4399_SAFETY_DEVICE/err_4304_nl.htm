4304 E_SAFETY_DEVICE_CAR_LOW_SPEED_LIMIT_EXCEEDED

Summary

 This section describes the 4300_4399_SAFETY_DEVICE error E_SAFETY_DEVICE_CAR_LOW_SPEED_LIMIT_EXCEEDED.

Description

A elevator safety device has detected that the car has exceeded the low speed limit (typ 0.75m/s) while the elevator was in maintencance operation (e.g. inspection)

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Wrong drive parameter setting Check all speed relevant drive parameters
2 Wrong low speed detection Check low speed detection device for malfunction
