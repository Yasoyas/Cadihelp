4350 E_SAFETY_DEVICE_CAR_UNINTENDED_MOVEMENT_DETECTION

Summary

 This section describes the 4300_4399_SAFETY_DEVICE error E_SAFETY_DEVICE_CAR_UNINTENDED_MOVEMENT_DETECTION.

Description

The car unintended movement device is not working as expected

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Door zone signal faulty Check corresponding sensors (e.g. PHS, PHUET) and wiring
2 Failure at car unintended movement detection device Replace corresponding device (e.g. PCB)
