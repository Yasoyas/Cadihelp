3303 ELEVATOR_KSR_A_IN_NON_TSD

Summary

 This section describes the 3300_3399_TSD error ELEVATOR_KSR_A_IN_NON_TSD.

Description

The elevator has detected a change of the signal KSR_A (state of TSD lever) which is only used together with TSD (temporary safety device) but the TSD is not detected as present.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 The wiring at the signal KSR_A is faulty Check general wiring
2 The elevator main controller board was exchanged by a board from another elevator Perform the learning travel. At no success replace elevator main controller board (e.g. SCIC, SCPU) by a new one from the factory.
