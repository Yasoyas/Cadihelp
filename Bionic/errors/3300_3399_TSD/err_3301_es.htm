3301 ELEVATOR_N_KNET_IN_NON_TSD

Summary

 This section describes the 3300_3399_TSD error ELEVATOR_N_KNET_IN_NON_TSD.

Description

The elevator has detected a change of the signal KNET (state of triangle key at landing door) which is only used together with TSD (temporary safety device) but the TSD is not detected as present.

Extra Info

Source ID
