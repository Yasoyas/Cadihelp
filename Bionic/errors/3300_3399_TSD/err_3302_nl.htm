3302 ELEVATOR_JREC_IN_TSD

Summary

 This section describes the 3300_3399_TSD error ELEVATOR_JREC_IN_TSD.

Description

The elevator has detected a change of the signal JREC (state of inspection switch on top-of-car inspection panel) which is only used together with standard top-of-car inspection but the standard top-of-car inspection is not detected as present. TSD (temporary safety device) is detected as present instead.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 The wiring at the signal JREC is faulty Check general wiring
2 The elevator main controller board was exchanged by a board from another elevator Perform the learning travel. At no success replace elevator main controller board (e.g. SCIC, SCPU) by a new one from the factory.
