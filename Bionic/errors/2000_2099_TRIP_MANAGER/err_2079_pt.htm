2079 E_TRIP_PHSx_SIGNAL_PERSISTENTLY_BRIDGED

Summary

 This section describes the 2000_2099_TRIP_MANAGER error E_TRIP_PHSx_SIGNAL_PERSISTENTLY_BRIDGED.

Description

During a trip the shaft information signal PHS and/or PHS1 do not change their state as expected. They are permanently active (e.g. bridged for test).

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 PHS/PHS1 bridge for test still mounted Remove bridge.
Perform a manual reset respectively perform the persistent fatal error reset procedure to set the elevator back in operation.
2 Connection to PHS or PHS1 bad Check wiring to PHS/PHS1.
Perform a manual reset respectively perform the persistent fatal error reset procedure to set the elevator back in operation.
3 PHS or PHS1 sensor defect Replace corresponding sensor.
Perform a manual reset respectively perform the persistent fatal error reset procedure to set the elevator back in operation.
