2075 E_TRIP_POSITION_CORRECTION_TOO_BIG

Summary

 This section describes the 2000_2099_TRIP_MANAGER error E_TRIP_POSITION_CORRECTION_TOO_BIG.

Description

The first limit for the correction of the car position (typ. 30mm) during a trip has exceeded. Note, this warning is only applicable for closed loop drives. The current running trip does not get interrupted.
