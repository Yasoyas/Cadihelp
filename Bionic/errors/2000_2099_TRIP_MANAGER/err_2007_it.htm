2007 E_TRIP_POSITION_MOVE_NOT_IN_DOOR_ZONE

Summary

 This section describes the 2000_2099_TRIP_MANAGER error E_TRIP_POSITION_MOVE_NOT_IN_DOOR_ZONE.

Description

L'ascensore stava per iniziare una corsa mentre la cabina non era al piano
Da notare che questo errore si verifica solo quando non si è in modalità manuale (ad es. ispezione)

Extra Info

Nessuna.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Cause diverse Controllare i precedenti messaggi nel registro dei messaggi dell'ascensore
2 Difetto nella rilevazione della zona porte del sistema di informazioni di vano Controllare la rilevazione della zona porte del sistema di informazioni di vano (ad es. PHSx, regolazione meccanica, sensore difettoso ecc.)

Field Feedback
ID Location Actions to be taken
1 Controller Aggiornare la SW di base dello SCIC almeno con V8.6
