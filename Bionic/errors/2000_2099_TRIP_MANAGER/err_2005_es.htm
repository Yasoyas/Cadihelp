2005 E_TRIP_POSITION_TARGET_NOT_REACHED

Summary

 This section describes the 2000_2099_TRIP_MANAGER error E_TRIP_POSITION_TARGET_NOT_REACHED.

Description

El ascensor ha finalizado un viaje, pero el sistema de información del hueco señaliza que la cabina no se encuentra en la zona de puerta.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Diferentes causas Comprobar los mensajes comunicados anteriormente en el registro de mensajes del ascensor para identificar el origen del problema.
2 Señal de la detección de zona de puerta defectuosa Comprobar los sensores de la zona de puerta (p.ej., PHSx) y el cableado.

Field Feedback
ID Location Actions to be taken
1 Controller Actualizar el software de placa SCIC a una versión V8.6 o superior
