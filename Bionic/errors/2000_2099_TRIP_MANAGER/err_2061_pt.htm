2061 E_TRIP_LEARNING_INTOLERABLE_FLAG_LENGTH

Summary

 This section describes the 2000_2099_TRIP_MANAGER error E_TRIP_LEARNING_INTOLERABLE_FLAG_LENGTH.

Description

Detected a too long or too short door zone.

Note, this error is typically reported at the learning travel.

Extra Info

0: unknown
1: too short
2: too long

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Tacho factor or drive pulley diameter invalid Check corresponding parameter for right values
2 Signal of door zone detection faulty Check door zone sensors (e.g. PHS)
3 Flag length out of range Install correct flags
