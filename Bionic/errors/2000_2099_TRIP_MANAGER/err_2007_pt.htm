2007 E_TRIP_POSITION_MOVE_NOT_IN_DOOR_ZONE

Summary

 This section describes the 2000_2099_TRIP_MANAGER error E_TRIP_POSITION_MOVE_NOT_IN_DOOR_ZONE.

Description

O ascensor devia ter iniciado uma viagem apesar de a cabina não ter sido detectada no piso
Nota, este erro ocorre apenas quando não está em modo de funcionamento de viagem manual (por ex. inspecção)

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Diferentes causas Verificar a existência de mensagens anteriores incluídas em relatório no registo de mensagens do ascensor
2 Falha na detecção da zona de porta do sistema de informação de caixa Verificar a detecção da zona de porta do sistema de informação de caixa (por ex. PHSx, ajuste mecânico, sensor com defeito, etc.)

Field Feedback
ID Location Actions to be taken
1 Controller Actualize a versão de SW para 8.6 ou superior
