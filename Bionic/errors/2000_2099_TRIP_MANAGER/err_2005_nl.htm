2005 E_TRIP_POSITION_TARGET_NOT_REACHED

Summary

 This section describes the 2000_2099_TRIP_MANAGER error E_TRIP_POSITION_TARGET_NOT_REACHED.

Description

De lift heeft de rit niet afgesloten, maar het schachtinformatiesysteem heeft gesignaleerd dat de kooi zich niet in de deurzone bevindt.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Verschillende oorzaken Controleer in het log-bestand van de lift of er voorafgaande meldingen zijn. Hieruit kan de bron voor het probleem worden vastgesteld.
2 Signaal deurzonedetectie fout Zonesensoren en bedrading controleren (bijv. PHSx).

Field Feedback
ID Location Actions to be taken
1 Controller Upgrade besturings software naar minimaal v8.6
