2007 E_TRIP_POSITION_MOVE_NOT_IN_DOOR_ZONE

Summary

 This section describes the 2000_2099_TRIP_MANAGER error E_TRIP_POSITION_MOVE_NOT_IN_DOOR_ZONE.

Description

Verwacht werd dat de lift start met de rit, terwijl de kooi niet op de verdieping werd gedetecteerd
Let op: deze fout gebeurt alleen bij handmatige ritten (bijv. inspectie)

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken

2 Storing bij deurzone-detectie van schachtinformatiesysteem Controleer de zonedetectie van het schachtinformatiesysteem (bijv. PHSx, mechanische instelling, defecte sensor etc.)
