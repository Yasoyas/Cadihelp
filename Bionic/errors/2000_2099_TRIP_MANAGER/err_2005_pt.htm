2005 E_TRIP_POSITION_TARGET_NOT_REACHED

Summary

 This section describes the 2000_2099_TRIP_MANAGER error E_TRIP_POSITION_TARGET_NOT_REACHED.

Description

O ascensor concluiu uma viagem, mas o sinal enviado pelo sistema de informação de caixa indica que a cabina não se encontra na zona de porta.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Diferentes causas Verificar a existência de mensagens anteriores incluídas em relatório no registo de mensagens do ascensor para identificar a causa principal do problema.
2 Sinal de detecção da zona de porta com falhas Verificar sensores de zona de porta (por ex. PHSx) e cablagem.

Field Feedback
ID Location Actions to be taken
1 Controller Actualize a versão de SW para 8.6 ou superior
