2007 E_TRIP_POSITION_MOVE_NOT_IN_DOOR_ZONE

Summary

 This section describes the 2000_2099_TRIP_MANAGER error E_TRIP_POSITION_MOVE_NOT_IN_DOOR_ZONE.

Description

The elevator was intended to start a trip while the car was not detected to be on floor

Note, this error only occurs while not in manual trip operation (e.g. inspection)

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken

2 Failure at door zone detection of hoistway information system Check door zone detection of hoistway information system (e.g. PHSx, mechanical adjustment, defective sensor etc.)
