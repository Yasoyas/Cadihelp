2078 E_TRIP_FINAL_LEVEL_NOT_FOUND

Summary

 This section describes the 2000_2099_TRIP_MANAGER error E_TRIP_FINAL_LEVEL_NOT_FOUND.

Description

The terminal floor indication (top or bottom floor) is faulty

Extra Info

0: Unknown (not supported)
1: Failure at upper hoistway end
2: Failure at lower hoistway end

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 The distance between the signalization of the hoistway end and the corresponding terminal floor (top respectively bottom floor) is out of range Check position of terminal floor sensor (top or bottom floor, e.g. PHS)
Check position of hoistway end sensor (e.g. KSEx)
2 The signal indicating the hoistway end is faulty Check hoistway end sensors (e.g. KSEx).
Check general wiring to this sensor.
3 The signal indicating the terminal floor is faulty Check (terminal) floor sensor (e.g. PHS).
Check general wiring to this sensor.

Field Feedback
ID Location Actions to be taken
1 Shaft Verificar posição do KSE. O distância do sinal KSE aos pisos extremos deve estar de acordo com os esquemas
