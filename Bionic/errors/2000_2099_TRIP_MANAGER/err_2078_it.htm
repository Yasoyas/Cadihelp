2078 E_TRIP_FINAL_LEVEL_NOT_FOUND

Summary

 This section describes the 2000_2099_TRIP_MANAGER error E_TRIP_FINAL_LEVEL_NOT_FOUND.

Description

E_TRIP_FINAL_LEVEL_NOT_FOUND

Extra Info

Nessuna.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 La sincronizzazione della corsa di apprendimento non ha trovato un indicatore per il livello più alto o per il livello più basso dopo l'ultimo segnale KSE prima del buffer. Controllare la posizione KSE Il segnale Kse deve essere situato a SKA=1250mm dai livelli dei piani estremi. Altrimenti la cabina raggiunge sistematicamente il contatto di ultracorsa KNE.

Field Feedback
ID Location Actions to be taken
1 Shaft Controllare : posizione del KSE. Il KSE deve essere posizionato a SKA=1250mm dal livello dei piani estremi.
