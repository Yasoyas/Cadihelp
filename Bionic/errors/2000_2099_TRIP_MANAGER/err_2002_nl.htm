2002 E_TRIP_LEARNING_LEVEL_MISSING

Summary

 This section describes the 2000_2099_TRIP_MANAGER error E_TRIP_LEARNING_LEVEL_MISSING.

Description

The number of floor levels counted during the learning trip in upward direction does not correspond to the one counted with the check in downward direction.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Bad alignment of floor sensors with magnets (KS) or PHS flags in shaft Check alignment of magnets (KS) or PHS flags.
Check position of floor sensor.
Check magnet to sensor distance.
