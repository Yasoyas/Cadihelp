2004 E_TRIP_LEARNING_LEVEL_OUTSIDE_ARRAY_LIMITS

Summary

 This section describes the 2000_2099_TRIP_MANAGER error E_TRIP_LEARNING_LEVEL_OUTSIDE_ARRAY_LIMITS.

Description

Indicates that the number of learned floors is invalid

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 The number of learned floors exceeded the max floors supported by the elevator system Check the number of door zone indicators in the hoistway (e.g. magnets, vanes)
2 The number of learned floors does not correspond with the commisioning data Crosscheck the number of door zone indicators in the hoistway (e.g. magnets, vanes) with the allowed number of floors according the commissioning data
