5005 E_RM_MAX_DATA_CONNECTION_ATTEMPTS_TO_CC_EXCEEDED

Summary

 This section describes the 5000_5099_REMOTE_MONITORING error E_RM_MAX_DATA_CONNECTION_ATTEMPTS_TO_CC_EXCEEDED.

Description

There were several failed attempts (typically 5) from the elevator control to establish the data connection to the control center. No further attempts are performed within the next 24 hours. See previous logged errors to get the reason of the problem.

Note, this mechanism prevents to get inadmissable connection costs at data transmission by phoneline.
