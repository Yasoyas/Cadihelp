5007 E_RM_DATA_COMMUNICATION_DEVICE_DEAD

Summary

 This section describes the 5000_5099_REMOTE_MONITORING error E_RM_DATA_COMMUNICATION_DEVICE_DEAD.

Description

The communication between the elevator control and the remote monitoring data communication device (e.g. modem) has failed

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 The connection between the elevator control board and the remote monitoring data communication device (e.g. modem) is faulty Check data connection
2 Remote monitoring data communication device (e.g. modem) defective Replace remote monitoring data communication device
