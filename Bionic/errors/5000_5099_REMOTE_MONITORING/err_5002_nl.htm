5002 E_RM_NO_PHYSICAL_DATA_CONNECTION_TO_CC

Summary

 This section describes the 5000_5099_REMOTE_MONITORING error E_RM_NO_PHYSICAL_DATA_CONNECTION_TO_CC.

Description

The data connection from the elevator (remote monitoring data communication device, e.g. modem) to the control center hasn't physically established or was physically interrupted

Extra Info

0.) Unknown (not supported)
1.) Dialtone missing
2.) Line voltage too low
3.) No carrier
4.) Data speed negotiation failed

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Connection physically interrupted or bad Check data connection (e.g. phone line)
2 Data speed negotiation failed (e.g. at modem) Check configuration (e.g. country code)
