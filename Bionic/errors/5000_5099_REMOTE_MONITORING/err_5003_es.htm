5003 E_RM_DATA_LINE_TO_CC_BUSY

Summary

 This section describes the 5000_5099_REMOTE_MONITORING error E_RM_DATA_LINE_TO_CC_BUSY.

Description

The data connection from the elevator (remote monitoring data communication device, e.g. modem) to the control center hasn't physically established because the line was busy

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Control center not able to answer call of elevator (e.g. too busy) Do nothing. Wait for next calling attempt.
