2604 E_EEPROM_ACCESS_TO_UNKNOWN_FILE

Summary

 This section describes the 2600_2699_EEPROM error E_EEPROM_ACCESS_TO_UNKNOWN_FILE.

Description

The elevator control has tried to access a file which is not present in its EEPROM
