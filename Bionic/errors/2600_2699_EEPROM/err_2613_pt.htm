2613 E_EEPROM_TRAFFIC_CTRL_FILE_ERROR

Summary

 This section describes the 2600_2699_EEPROM error E_EEPROM_TRAFFIC_CTRL_FILE_ERROR.

Description

Corrupted data in the persistent memory (EEPROM) of the elevator control (CRC error at traffic control file)

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Faulty EEPROM Replace elevator control PCB if failure persists
