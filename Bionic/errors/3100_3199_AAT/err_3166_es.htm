3166 E_AAT_CWIB_TEST_CAR_NOT_AT_BOTTOM_FLOOR

Summary

 This section describes the 3100_3199_AAT error E_AAT_CWIB_TEST_CAR_NOT_AT_BOTTOM_FLOOR.

Description

The acceptance test 'Counterweight impact on Buffer' was tried to get initiated by the fitter while the car was not standing on the expected starting position (bottom floor)

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 The preparation for this acceptance test has failed (park the car on the bottom floor) due to any fault in the system (e.g. faulty shaft information system). Check previous reported errors in error log. Get rid of the corresponding problem.
2 Elevator main controller software problem Leave the test mode, reset the system and try again. If there's no success, update main controller software.
