3163 E_AAT_CIB_TEST_NOMINAL_SPEED_NOT_REACHED

Summary

 This section describes the 3100_3199_AAT error E_AAT_CIB_TEST_NOMINAL_SPEED_NOT_REACHED.

Description

The acceptance test 'Car impact on Buffer' failed, because the needed car nominal speed wasn't reached.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 The shaft is too short for the test to be performed Reduce the nominal speed of the elevator. Repeat the test.
2 The drive frequency converter (FC) software does not match with the elevator main controller software Update drive frequency converter (FC) software.
