3172 E_AAT_SGC_TEST_UNEXPECTED_STOP

Summary

 This section describes the 3100_3199_AAT error E_AAT_SGC_TEST_UNEXPECTED_STOP.

Description

The acceptance test 'Safety Gear Car' got interrupted unexpectedly by the user.

Extra Info

ndef

Source ID

ndef

Causes and Actions
ID Possible causes of message Actions to be taken
1 Mal manipulation Keep the recall button (DRH) pressed well during test
2 The safety circuit has opened while the test was running Check safety circuit contacts (e.g. KSS)
