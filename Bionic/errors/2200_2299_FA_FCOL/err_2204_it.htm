2204 E_FA_DRIVE_WRONG_SD_LOGIC

Summary

 This section describes the 2200_2299_FA_FCOL error E_FA_DRIVE_WRONG_SD_LOGIC.

Description

E_FA_DRIVE_WRONG_SD_LOGIC

Extra Info

Nessuna.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 (solo Variodyn) La sequenza logica del segnale SD non è corretta (SD non diventa inattivo dopo transizione KSA. Possibili cause: Variodyn difettoso, cablaggio difettoso, uscita KSA difettosa, ingresso SD difettoso. L'ascensore può presentare insufficiente precisione di arrivo.
