2133 E_SHAFT_MIX_KSE_AND_KSE_D_U

Summary

 This section describes the 2100_2099_SHAFT error E_SHAFT_MIX_KSE_AND_KSE_D_U.

Description

La maniobra del ascensor recibió señales del sistema de información del hueco que no eran señales del tipo esperado.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 El cableado del sensor de posición del final del hueco está defectuoso. Comprobar el cableado y la correcta conexión del sensor correspondiente (KSE, KSE_U, KSE_D).
