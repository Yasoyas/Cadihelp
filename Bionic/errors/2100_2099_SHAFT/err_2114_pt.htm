2114 E_SHAFT_POSITION_LOST_WHILE_STATIONARY

Summary

 This section describes the 2100_2099_SHAFT error E_SHAFT_POSITION_LOST_WHILE_STATIONARY.

Description

O sistema de informação de caixa indicou que a cabina se encontra fora da zona de porta (cabina não se encontra no piso) apesar de a cabina dever estar parada no piso

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Falha na detecção da zona de porta do sistema de informação de caixa Verificar a detecção da zona de porta do sistema de informação de caixa (por ex. PHSx, ajuste mecânico, sensor com defeito, etc.)
2 Cabina saiu da zona de porta Verificar meios de suspensão (por ex. cabos para detectar alongamento demasiado elevado)
Verificar travão da transmissão

Field Feedback
ID Location Actions to be taken
1 Controller Premir Reset (reinicializar) na placa SCIC
