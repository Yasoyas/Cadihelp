2137 E_SHAFT_SPEED_SUPERVISION_AT_ROUGH_POS_CHANGE

Summary

 This section describes the 2100_2099_SHAFT error E_SHAFT_SPEED_SUPERVISION_AT_ROUGH_POS_CHANGE.

Description

The car was not decelerating (still travelling in normal speed) while it was reaching the shaft end (at KSE) because the wrong number of floors was counted. Failure in shaft information system.

Extra Info

ndef

Source ID

ndef

Causes and Actions
ID Possible causes of message Actions to be taken
1 Bad alignment of floor sensors (KS/KS1) with magnets in shaft Check alignment of magnets.
Check position of floor sensors. Check for lost magnets.
Check magnet to sensors distance.
2 Bad floor sensor signal transmission Check general electrical wiring of the floor sensor signal (e.g. connections at KS/KS1, at SDIC, ..)
3 Floor sensor (KS/KS1) defect Replace corresponding floor sensor
4 Faulty input on the interface board (PCB) for the floor sensor Replace interface board (SDIC)
5 One or more floor indication magnet of the shaft information system is/are mounted with wrong magnetic orientation Check entire shaft for the correct magnetic orientation (N,S) of these magnets
