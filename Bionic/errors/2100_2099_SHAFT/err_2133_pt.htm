2133 E_SHAFT_MIX_KSE_AND_KSE_D_U

Summary

 This section describes the 2100_2099_SHAFT error E_SHAFT_MIX_KSE_AND_KSE_D_U.

Description

O comando do ascensor recebeu sinais do sistema de informação de caixa que não são sinais do tipo de sistema de informação de caixa previsto.

Extra Info

Source ID

Causes and Actions
ID Possible causes of message Actions to be taken
1 Falhas na cablagem para o sensor de posição de fim de caixa. Verificar cablagem e correcta ligação do sensor correspondente (KSE, KSE_U, KSE_D).
