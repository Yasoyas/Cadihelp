2163 E_SHAFT_UPPER_FLAG_EDGE_NOT_SET

Summary

 This section describes the 2100_2099_SHAFT error E_SHAFT_UPPER_FLAG_EDGE_NOT_SET.

Description

E_SHAFT_UPPER_FLAG_EDGE_NOT_SET

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 While setting the RAM shaft image at the application startup, a missing upper flag edge position is detected. Manually erase the shaft image file on the persistent data medium and run the synchronization trip.
