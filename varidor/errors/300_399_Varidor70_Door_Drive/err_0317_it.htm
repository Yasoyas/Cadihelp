0317 IAC_UNKNOWN_LEVEL

Summary

 This section describes the Varidor70 Door Drive error IAC_UNKNOWN_LEVEL.

Description

[V70 Door Controller SW]
A Command with unknown or incorrect level information was sent.

Error response: Wait till the doors open, and then stop all door motions
Easy Setup: Bargraph-RED-LED-2 [ON]

Extra Info

If a command with unknown level/floor information is handled, the SW cannot find the corresponding mass for the floor, and without that information of mass, the door motions cannot be done.

Source ID

This error is generated from the IAC Panel

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. Unknown floor info a door command 1. Check the debug logs
2. Reset the Door Controller
2 2. Incorrect LONV70 Firmware 1. Connect CADI-GC to the lift controller and check for a valid version of the LON Firmware on the LONV70
2. Download valid LONV70 Firmware
3 2. Faulty or loosely fitted LONV70 Card 1. Remove and refit the LONV70 card
2. Replace the LONV70 card
