0333 SLV70_OBSTACLE_DETECT

Summary

 This section describes the Varidor70 Door Drive error SLV70_OBSTACLE_DETECT.

Description

[V70 Door Controller SW]
An Obstacle was detected during self learning process, this is a warning and not an error

Error Response: None
Easy Setup: None

Extra Info

None

Source ID

This error/warning is generated from Self Learning Task

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. Obstacle in the door ways during Self Learning 1. Remove the Obstacel from the door ways and start Self Learning again
