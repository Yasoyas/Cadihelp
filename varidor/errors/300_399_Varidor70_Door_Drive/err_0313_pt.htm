0313 HW_FLASH_WRITE

Summary

 This section describes the Varidor70 Door Drive error HW_FLASH_WRITE.

Description

[V70 Door Controller SW]
It was not possible to write onto the Flash.

Error response: Wait till the door is open and the stop all further door motions
Easy Setup: Bargraph-RED-LED-8 [ON]

Extra Info

This error occurs at the start up when the Flash is being checked for consistency. This error also occurs in the Param Backup task, when it encounters Flash writing problems.

Source ID

This error is generated from the Param

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. Flash is gone bad 1. Reset the Door Controller
2. Replace the Door Controller PCB
