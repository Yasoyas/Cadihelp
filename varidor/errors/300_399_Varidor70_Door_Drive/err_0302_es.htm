0302 TRC_PAR_ACCESS

Summary

 This section describes the Varidor70 Door Drive error TRC_PAR_ACCESS.

Description

[V70 Door Controller SW]
Trip Curve Converter/Generator parameter access failed.

Error response: Stop the door immediately

Easy Setup: Bargraph-RED-LED-7 [ON]

Extra Info

The most common parameters which might cause this errors are
a. Safety Standard
b. Energy Limits
c. Emergency Parameters
d. Mass
e. Curve Type
f. Jerk,Acceleration,Speed Limits both Open & Close Directions
g. Door Type

Source ID

This error is generated from Trip Curve Generator

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. Parameter not defined or incorrect 1. Check validity of parameter definitions described
2 2. Parameter access failed 2. Try resetting the door controller
3 3. Flash is gone bad 3. If the error "PAR_NOVRAM_MISSING" occurs too, then change the Door Controller PCB and boot the door again.
