0314 HW_WATCHDOG

Summary

 This section describes the Varidor70 Door Drive error HW_WATCHDOG.

Description

[V70 Door Controller SW]
This is not an error, but a warning.
This warning comes after the door controller had had a reset because of a Watchdog or a self reset

Error response: None, just to notify that there was a Watchdog initiated reset
Easy Setup: Bargraph-RED_LED-8 [ON]

Extra Info

This could be beacuse of a runaway task. Watchdog will reset the system if it has not been notified within a preset period of time. This warning is generated at startup of the system.

Source ID

This warning is generated from General Init

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. Watchdog caused a reset, the reasons are explained in the Error Description above None
2 2. Errors in Peripheral units can cause seld resets to reactivate the Peripherals None
