0309 HW_POSREF

Summary

 This section describes the Varidor70 Door Drive error HW_POSREF.

Description

[V70 Door Controller SW]
Error in positon referencing
This error is raised when the position sensor unit goes bad or is removed or someway disconnected.

Error response: Stop all door motions and stop all further door motions, till the error stops, once the Sensor Unit is reconnected the door can resume the motions
Easy Setup: Bargraph-RED-LED-5 [ON]

Extra Info

Once the Position Sensor Unit is re-connected the door resumes the usual operation. This error is generated at the startup or whenever a parameter update takes place

Source ID

This error is generated from Sensor Unit

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. Position Sensor Unit cable is disconnected or loose 1. Check the cable for damage.
2. Disconnect and Reconnect the cable
2 2. Position Sensor Unit is gone bad 1. Replace Position Sensor Unit
