0331 BLOCKED_OPENING_DIRECTION

Summary

 This section describes the Varidor70 Door Drive error BLOCKED_OPENING_DIRECTION.

Description

[V70 Door Controller SW]
The door is blocked in the opening direction.

Error Response: Stop all door motions, and stop all further door motions
Easy Setup: Bargraph-RED-LED-8 [ON]

Extra Info

This error is generated in response to the blocked state detected in the Mechanical supervisor module. When, applied force is unable to move the door in the open direction.

Source ID

This error is generated from SafeDev SW Module

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. Obstacle in the door ways 1. Remove the Obstacle in the door way if any
2. Reset the Door Controller
2 2. Panels Jammed beacuse of dirt on the panel gangway 1. Remove or clean the dirt in the panel gang way
2. Reset the Door Controller
