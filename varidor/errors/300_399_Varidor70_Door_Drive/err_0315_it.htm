0315 IAC_MOD_UPD_FAIL

Summary

 This section describes the Varidor70 Door Drive error IAC_MOD_UPD_FAIL.

Description

[V70 Door Controller SW]
Unable to update the Speed Mode in the Trip Curve Generator.

Error response: Stop all door motions and stop all further door motions
Easy Setup: Bargraph-RE-LED-7 [ON]

Extra Info

Door Mode is set per command, every command has a door mode associated with it. Mostly this error will occur if a door command with an unsupported door mode is sent. The other reason this error could occur is when the speed mode dependent parameters cannot be accessed/read from the Flash.

Source ID

Trip Curve Generator, when the door mode needs to be changed. This error is generated from only one location.

Causes and Actions
