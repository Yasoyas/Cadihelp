0307 HW_PAR_ACCESS

Summary

 This section describes the Varidor70 Door Drive error HW_PAR_ACCESS.

Description

[V70 Door Controller SW]
Parameter Access failed when the controller was trying to do a motion. The error could also be raised if the read parameter values are not correct, or when a certain changed parameter is set back to the parameters.

Error response: Stop all door motions, and dont do any more door motions
Easy Setup: Bargraph-RED-LED-8 [ON]

Extra Info

This error generally occurs when the force is being applied in a motion. The parameters because of which the error could rise are
a. d0
b. DriveNewtonmeterPerVolt
c. iGear
d. DrivePoleLength
e. DriveEncoderIncr
f. ZeroSpeed
g. StalledSpeed
h. OscillatingSpeed

Source ID

This error is generated from the HWAct

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. Invalid Parameter Values 1. Check the valdity of the parameters described in the error description
2 2. Cannot Access Parameters 1. Reset the Door Controller
2. Replace the Door Controller PCB
3 3. Flash is gone bad 1. Reset the Door Controller
2. Replace the Door Controller PCB
