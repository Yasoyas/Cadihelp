0324 ESK_COMMUNICATION_ERR

Summary

 This section describes the Varidor70 Door Drive error ESK_COMMUNICATION_ERR.

Description

[V70 Door Controller SW]
No communication between Easy Setup and Door Controller. This is a warning and not an error.

Error response: None
Easy Setup: Bargraph-RED-LED-8 [ON]

Extra Info

None

Source ID

This error/warning is generated from ESK SW Module

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. ESK HW is gone bad 1. Reset the Door Controller
2. Check the ESK HW
3. Replace the ESK HW
2 2. ESK Cables 1. Check the ESK Cables and connectors
2. Consider Shielding the cables for short circuits.
3. Replace the ESK HW
