0329 NORMAL_RESET

Summary

 This section describes the Varidor70 Door Drive error NORMAL_RESET.

Description

[V70 Door Controller SW]
A Reset was done. This is a warning and not an error

Error Reponse: None
Easy Setup: None

Extra Info

This error is generated at the startup, which indicated what was the cause of last reset. There could be two causes,
a. Deliberate Reset
b. Watchdog Reset

Source ID

This is error/warning is generated from General Init

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. A reset had taken place None
