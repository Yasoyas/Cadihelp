0327 VLON_RECEIVE

Summary

 This section describes the Varidor70 Door Drive error VLON_RECEIVE.

Description

[V70 Door Controller SW]
VLON receiving error, this is an internal SW error

Error response: Wait till door is open, then stop all door motions
Easy Setup: Bargraph-RED-LED-2 [ON], LC-Comm-RED-LED [ON]

Extra Info

This error occurs of the message from the incoming lon queue could not be read

Source ID

This error is generated from the LON SW Module

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. Memory Problems 1. Reset the Door Controller
2 1. LON Firmware is not OK 1. Check the validity of the LON Firmwaer via CADI-GC
2. Update the LON Firmware if required
3 2. LONV70 Card has gone bad 1. Replace the LONV70 Card
