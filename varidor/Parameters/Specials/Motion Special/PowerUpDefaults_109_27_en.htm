Power Up Defaults

Summary

 This section describes the parameters in the parameter window Power Up Defaults of the parameter group Motion Special in subsystem Specials.

Motion Limits Standard Mode

motion_limits_standard_mode defines the standard mode after power up for the limits of speed, acceleration etc. Normally it is set to Normal (4) There are 5 modes: "Nudging mode" to ensure nudging limits are hold: motion_limits_standard_mode = 1 "Night mode" to minimize noise: motion_limits_standard_mode = 2 "Soft mode" to minimize noise, but at a higher speed: motion_limits_standard_mode = 3 "Normal mode" with the optimum trade-off between speed, noise and reliablity: motion_limits_standard_mode = 4 "Turbo mode" using optimum speed parameters for rush hour operation: motion_limits_standard_mode = 5
