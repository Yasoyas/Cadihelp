Transmission Ratio

Summary

 This section describes the parameters in the parameter window Transmission Ratio of the parameter group Door Drive in subsystem Specials.

Belt Transmission Ratio

Transmission ratio of the belt gear. i_gear = speed_motor/speed_belt_wheel. For a direct rotating drive, i_gear is set to 1.0 . For a linear motor i_gear has to be set to 1.0 .

 














Belt Wheel Diameter


For Belt drives: Diameter of the belt wheel in m. (e.g. 0.0661m for a Varidor 30)
The effective diameter d0 has to be taken, where d0=z*t/pi.
z is the number of teeth of the belt wheel
t is the distance between the teeth on the belt and
pi is 3.1416 .
E.g. for a BAT10 belt, z is 20 and t is 10mm, resulting in a d0 of 63.63mm.
For a linear motor drive, d0 has to be set to 1.0 .
