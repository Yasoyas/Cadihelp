HoldingForceAtClosedPos

Summary

 This section describes the parameters in the parameter window HoldingForceAtClosedPos of the parameter group Closing Force in subsystem Specials.

F1

This force F1 ramp is used to minimize closed loop position control error. It helps to close the door completely before trip curve has completed.
The force ramps up from 0.0 to F1 within 0.5 seconds and starts at calculated blind zone (t0).
Ramp-up stops when trip curve has completed (t1) latest and force F2 is applied.

t0: Calculated BlindZone
t1: Reference trip curve has finished

Calculated BlindZone= BlindZone + ClosedPosition

 














F2


This force F2 is used to close the door in case of misalignment. It moves the door towards closing simultaneously while the active clutch system is closing. Especially needed with the car front sealing which gives additional friction to the door while clutch closing.

It is applied as soon as the reference trip curve has completed (t1) and stops if clutch are closed (t2) to start force ramp-up F3.
In case of no misalignment the door is fully closed at this time.

t1: Reference trip curve has completed
t2: Door Locked i.e. Clutch closed





















F3


This force F3 is used to press the panels together. This helps to pre-form the sealing rubber to get the doors completely closed especially in case of car front sealing.
Force ramps up with 12.5 N/s starting as soon as door is locked i.e. clutch closed (t2) and hold for 3 seconds.
After this force F3 is started to ramp down with 25 N/s to force F4 (t3) as long as the door position has not moved back for > 0.5 mm.

Error 337 HOLDING_FORCE_AT_CLOSED_POSITION
is generated if F3 can not be reduced to < 70N

t3: F3 ramped down as long as door position from F3 does not change > 0.5mm to F4





















F4


This force F4 is used to save power consumption.
It is applied all the time until the door position is changing > 1mm from the average door position while force F3 is applied. In this case force F3 is ramped up with 417 N/s and hold for 3 seconds to ramp down as described on F3.

Error 337 HOLDING_FORCE_AT_CLOSED_POSITION
is generated if F4 to F3 cycle has occurred for more than 10 times before a door command has arrived.

t4: If average door position while F3 has changed > 1.0mm by any reason, F4 is applied to close the door again





















HoldingForceAtClosedPosition


HoldingForceAtClosedPosition is a force profile consisting of F1, F2, F3 and F4 to close the door in case of misalignment and with the option car front sealing properly.
Detailed description see on force F1 to F4.

ON: 	Force profile= enabled (F1 to F4 have effect)
OFF:	Force profile= disabled (F1 to F4 have no effect)
	(i.e. same behaviour as on SW V70 2.40;
	target closed position= closed pos -3mm)
F3_Only:	Part of force profile= enabled (F1, F2 have no effect)
