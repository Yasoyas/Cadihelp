Door Settings

Summary

 This section describes the parameters in the parameter window Door Settings of the parameter group Settings in subsystem General.

Commissioning No

serial number of the door drive, max 32 Bit long

 














Number of Floors


Number of Levels in this installation. If Levels is set to 1, no level dependant commissioning is performed.





















Door Drive Type


Name of door for lift control. For Varidor 70 this is "VD70    " .





















Type Of Door


Door type e.g. center opening with 2 panels: C2, Telescopic door with 2 panels: T2, 
Center telescope door with 4 panels: C4





















Left Opening


Door is a left opening telescope door, when set to 1.
The direction (sign) of force reference and position measurement is inverted internally in this case.





















BKE [m]


Max. car door width in m
This parameter is floor undependant
(BKE = Breite Kabine Eingang)





















Speed Selector Opening


Opening Speed Selector
Set the motion speed selector for door opening. Each speed mode e.g. nudging can adjusted by the speed selectors. The speed mode is set by Lift Control.

50%    Input 0 
70%    Input 1
100%  Input 2
120%  Input 3
140%  Input 4





















Speed Selector Closing


Closing Speed Selector
Set the motion speed selector for door closing. Each speed mode e.g. nudging can adjusted by the speed selectors. The speed mode is set by Lift Control.

50%    Input 0 
70%    Input 1
100%  Input 2
120%  Input 3
140%  Input 4





















Safety Code Type


Sets the standard to select the right kinetic energy limits for closing movement.
With the selection Automatic, the MaxClosingForce will be calculated during the selflearning trip

0 = EN81,
1 = ANSI,
2 = CSA,
3 = Automatic








This parameter is a setting parameter.
