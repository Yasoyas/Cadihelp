Mechanical Settings

Summary

 This section describes the parameters in the parameter window Mechanical Settings of the parameter group Door Mechanism in subsystem General.

BKE

Max. car door width in m
This parameter is floor undependant
(BKE = Breite Kabine Eingang)

 














HT


Nominal door height in m





















BT Raw


Nominal door width as input from ESI





















HT Raw


Nominal door height as input from ESI





















Cladding Raw


Cladding thickness
This parameter value used for calculation the door mass. The calculation is made by Door Controller itself. 
Value 0: No Cladding
Value 1: Cladding 1.5 mm
Value 2: Cladding 1.5..4 mm
Value 3: Cladding 4..8 mm
Value 4: Cladding 8..12.5 mm





















Mass


Effective moved masses in kg. The learned value for the mass of the complete door mechanism transformed to the fast panels edge for the given level. In kg.
For a C2 (Center) door, this is the sum of the weights of 2 car panels and 2 landing panels plus
weights of the motor, the carriers, the belt and the counterweights of the landing door.
For a C4 (Center-Telescope) door the sum of the panel weights has to be reduced to 62.5% in the calculation.
The maximum value 500.0F is 500kg, minimal value 42.4F is 43kg.





















Basic Drive Weight


Weight of carriers, belt and motor secondary.
The parameter is used to calculate weight estimates prior to learning operations.
.





















Panel Weight


Estimated weight of 1 square meter of complete panel design (bearing structure, cladding and thermal insulation).
The parameter is used to calculate weight estimates prior to learning operations by
calculating m     eff        = BasicDriveWeight + HT*BT*PanelWPerM2 .
