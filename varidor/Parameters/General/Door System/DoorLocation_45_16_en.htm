Door Location

Summary

 This section describes the parameters in the parameter window Door Location of the parameter group Door System in subsystem General.

Door Location Deck

Location of Door Drive in Car Normally the location of the door drive in the car (1. side(Zugang 1) or 2. side (Zugang 2)) is defined by a jumper on the lift-control-interface PCB (e.g. the VLON70 or the LONDIM). This allows to specify either Access side 1 or Access side 2. No adjustments have to be made for this case to the door drive, because reading that jumper and forwarding that information is the default.

For some special configurations (double decker car, three accesses to car) this is not sufficient. For these cases one has to set the Location parameter to one of the values defined below and the door will provide that information to the lift control during configuration. This feature works with a serial/field bus interface only,
but not with a parallel interface.

0: LON_DOOR_NO_DECK, Jumper info is relevant only
1: LON_DOOR_DECK_1
2: LON_DOOR_DECK_2

This parameter is a setting parameter.
