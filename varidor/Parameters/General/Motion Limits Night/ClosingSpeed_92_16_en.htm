Closing Speed

Summary

 This section describes the parameters in the parameter window Closing Speed of the parameter group Motion Limits Night in subsystem General.

Max Speed

max_speed_closing_night is maximum value for speed during closing in mode.

 














Max Acceleration


max_acc_closing_night is maximum value for acceleration during closing in mode.





















Max Deceleration


max_dec_closing_night is maximum value for during in mode.





















Jerks


jerks_closing_night is maximum value for during in mode.





















Curve Type


curve_shape_type_closing_night selects shape of speed curve for closing in night mode.
Input 0: ==> quiet motion curve 1
Input 1: ==> quiet motion curve 2
Input 2: ==> quiet motion curve 3
Input 3: ==> normal motion curve 
Input 4: ==> time optimised motion curve 1
Input 5: ==> time optimised motion curve 2
Input 6: ==> time optimised motion curve 3 

See Curve Type Number definition t0111tt0.htm.





















Locking Zone


Locking_zone is the position where locking may start during closing, i.e. where the clutch may start to release the lending door lock during night speed.
