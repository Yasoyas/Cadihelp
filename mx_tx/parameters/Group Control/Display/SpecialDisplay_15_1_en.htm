Special Display

Summary

 This section describes the parameters in the parameter window Special Display of the parameter group Display in subsystem Group Control.

Resolution

Resolution of physical position display. Defines the distance (in mm), when a display update has to occur. The definition is used to generate the message VXT_OUT_DISPLAY / VXT_PHYSICAL_POSITION.

Formula for the floor value of message VXT:PHYSICAL_POSITION:
Floor value = 'distance of car from bottom floor' * 8mm / 'P_Resloution'

 














Customer Display


Name for the floors/stops (maximum 4 characters) which shall be displayed on the position indicators for each referenced display position. The floor name consists of one to a maximum of four ASCII characters. The ASCII values must be right-aligned and the unused leading characters have to be filled with blanks (char '20') otherwise the indicator will not show the correct characters.

Normally the ASCI values defined in the parameter FLOOR_NAME (Position Display) are used for the floor name representation. The parameter is typically used when other information is to be displayed on the position indicators (e.g. express zones, 'TUEM selector')

Message Display


List of ASCII strings.  It is used to display special messages, such as active services etc.
