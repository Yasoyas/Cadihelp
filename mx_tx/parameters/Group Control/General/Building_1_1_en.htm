Building

Summary

 This section describes the parameters in the parameter window Building of the parameter group General in subsystem Group Control.

Machine Room Number

Defines which machine room the elevators in the group belong to (used for smoke detection). It consists of 8 bytes, one byte for each elevator in the system.
The parameter must be defined for as many elevators as there are elevators in the group.

 

¨A9[ 06 09 01 01 01 01 00 00 00 00 00 ]

P_STR_MACHINE_ROOM_DEF = 1 1 1 1 0 0 0 0 0
Triplex group with common machine room (1)

Hoistway Number

Defines which hoistway the elevators in the group belong to (used for smoke detection). It consists of 8 bytes, one byte for each elevator in the system.
The parameter must be defined for as many elevators as there are elevators in the group.

 

¨AA[ 06 09 01 01 01 02 02 00 00 00 00 ]

P_STR_HOISTWAY_DEF = 1 1 1 2 2 0 0 0 0
Quadruplex group with separate hoistways for elevators 1/2 and 3/4.
