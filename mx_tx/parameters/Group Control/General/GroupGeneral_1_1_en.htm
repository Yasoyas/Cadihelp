Group General

Summary

 This section describes the parameters in the parameter window Group General of the parameter group General in subsystem Group Control.

Group Number

The parameter is used to differentiate groups when several groups are connected together through a building bus

 














Number of Lifts


Number of lifts in the group





















Number of Floors-Group


Number of floors served by elevators in the group ( = highest floor number in the group)





















Main Floor


Specify which (group) floor level is the main floor (Haupthalt). Normally all elevators a group have the same main floor.





















Control Mode


Control mode (Steuerungsart)

1: DE     Simple Push Button Control / Druckknopf Einfach Steuerung (no more available)
2: KA     Collective Down Control / Kollektiv Abwärts Steuerung
3: KS     Collective Selective Control / Kollektiv Selektiv Steuerung
4: PI      Pick-Up Control / Pick-Up Steuerung
5: M10    (Embedded) Miconic 10 Control / Miconic 10 Steuerung
6: DD     Double Decker Control / Doppeldecker Steuerung
7: DDS    Double Decker Shuttle Control / Doppeldecker Shuttle Steuerung

For 2x2 KA control: Set the parameter 'control mode' to KA (2) and set the parameters for subgroup operation to 2x2 KA configuration.
