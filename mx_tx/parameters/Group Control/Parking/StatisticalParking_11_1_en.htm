Statistical Parking

Summary

 This section describes the parameters in the parameter window Statistical Parking of the parameter group Parking in subsystem Group Control.

Min.Parking Distance

Minimum parking distance (in number of floor) for statistical parking. This number is added and subtracted to the parking floor number to define the parking zone.

E.g. Parking floor is 5, parameter value is 2 => parking zone 3 to 7.

 














Min.Nbr of Calls


When the traffic (in calls per 2 minutes) is under this threshold the statistical parking is not active and the predefined parking is used instead.





















Weight on Floor


Allows to give more importance to special floors (lobby, canteen,…) for statistical parking.
