Option Box

Summary

 This section describes the parameters in the parameter window Option Box of the parameter group Terminals in subsystem Miconic 10.

Destination Call Buttons

Definition of the function assigned to the dedicated call button(s) on M10 Terminals with option box.
The additional buttons on the M10 option box are used to give destination calls for specific destinations. The parameter defines the destination floor value (in group floor notation 1..x) for these dedicated buttons. A maximum of eight dedicated buttons on a terminal is foreseen. The definition of the dedicated buttons with this parameter is used on all terminals but is overwritten by the programming of the button on the M10 terminal. (Refer also to local programming procedures of the dedicated buttons on the M10 terminal.)

 ¨69                     * PCT_GP_M10_DEVICES
  {01 [02 02 0005] * P_M10D_CALL_DISPLAY_DUR

  ¨0B [06 02 01 03] * P_M10D_DEDICATED_BUTTON 1 floor 3

  ¨0B [06 02 03 08] * P_M10D_DEDICATED_BUTTON 3 floor 8
  ¨0B [06 02 04 0B] * P_M10D_DEDICATED_BUTTON 4 floor 11
