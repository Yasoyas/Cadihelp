Names General

Summary

 This section describes the parameters in the parameter window Names General of the parameter group Supervisory Display in subsystem CADI-SMLCD.

Group Name

Specify a name for the group displayed on a supervisory display.

 














Lift Name


Defines the name of the lift to be displayed on the supervisory display (TUEM). Default value: 'A' for lift 1, 'B' for lift 2, etc. Maximum length is 10 characters. If less characters used the remaining characters have to be zero.





















Services Name


Defines the string (characters) shown on the service interface (CADI, SMLCD, Lobby monitoring system...) for the currently active service (function). Per service the following values are defined in this PCT parameter:

1. Service identification number (number)
2. Presentation identification number (number)
3. Service name ( maximum 4 ASCII characters)

Characters not uset for the service name (e.g. 'N' ) shall be filled with 0 (zero not '0'). The Presentation identification number is the display attribute but is not used at the moment.









Example: 
+------------+------------+-------------------+
| SERVICE_ID | PRESENT_ID |   SERVICE_NAME    |
+============+============+===================+
|  COL 1     |   COL 2    |COL3|COL4|COL5|COL6|
+------------+------------+----+----+----+----+
|  1         |  49        | 4E   00   00   00 | => N
+------------+------------+----+----+----+----+
|  3         |  50        | 4A   41   42   00 | => JAB
+------------+------------+----+----+----+----+
|  8         |  49        | 4A   4E   46   46 | => JNFF
+------------+------------+----+----+----+----+
