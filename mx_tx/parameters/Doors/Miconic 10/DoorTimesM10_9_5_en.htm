Door Times M10

Summary

 This section describes the parameters in the parameter window Door Times M10 of the parameter group Miconic 10 in subsystem Doors.

Min.Hold Open Boarding

Minimum door hold open time for boarding. If the addition of door open time increments for each boarding passengers is lower than this minimum time, the door will stay open for this minimum time.

 














Min Hold Open Exiting


Minimum door hold open time for exiting. If the addition of door open time increments for each exiting passenger is lower than this minimum time, the door will stay open for this minimum time.





















Open Long Increment


Amount of time added to the door hold open time for each passenger under or equal to the threshold.





















Open Short Increment


Amount of time added to the door hold open time for each passenger above the threshold.





















Long to Short Time Threshold


Thresholds for switching from long door hold open time increment to short increment.





















Max. Door Open Time


M10 maximum door open time.
If the calculating door hold open time for a boarding or an exiting is higher than the value of this parameter, then the value of this parameter is used instead of the calculated one. If the boarding coincides with an exiting, the test for the maximum is done separately for the exiting and the boarding. It means in that case that the total door hold open time can be two times the value of this parameter.
Depending on the mode defined in P_M10C_DOT_MODE, this parameter is also used when deciding if an allocation can be done when boarding at floor where car is standing door open.
