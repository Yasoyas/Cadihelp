Shaft

Summary

 This section describes the parameters in the parameter window Shaft of the parameter group Travel Control in subsystem Drive.

KSERE Distance

According to SIA 11.02.38. Travel up to 1.5 m underneath the lowest mounted element.

 














KSE Distance


Distance from KSE switches to the bottom/top floor level. KSE switches are used for (depending on type of shaft information system):
- automatic positioning of car before starting the learn trip.
- finish of learn trip at top floor.(has to match number of floors in PCT)
- Synch. trip 
- Normal Terminal Slowdown monitoring (NA/ASME code) / for NTSL the KSE switches must be placed at a distance:
KSE_DISTANCE = ((NOM_SPEED*NOM_SPEED) / (2 * A_MAX)) + (NOM_SPEED * 0.05)
(NOM_SPEED in mm/s, A_MAX in mm/s/s)





















GAP Length


Distance between the two KSE magnets in the upper hoistway area ( only used for TX4 )

values for GAP_LENGTH depending on shaft info setting:
{0: PHS_PHNRD_PHUET}  =  300
{1: KS_KS1_KUET} =  300
{2: MOSIS} =  300
{3: KUET_AGSI}  =  0
{4: SI_IG}  =  0





















Fin. Length


Length of the floor vane.

values for FIN_LENGTH depending on shaft info setting:
{0: PHS_PHNRD_PHUET}  =  200
{1: KS_KS1_KUET} =  200
{2: MOSIS} =  200
{3: KUET_AGSI}  =  240
{4: SI_IG} = 200

Parameter P_ANT_RELEVEL_DISTANCE is calculated based on the value of P_ANT_FIN_LENGTH.

On-Level Distance


The drive sends a ON/OFF message (VXT_CABIN_IS_LEVEL) to the control. This message is used for special control functions ( e.g. ON/OFF level signal in the car ). As long as the car is closer to a floor than the distance defined with the parameter 'ON_LEVEL_DIST' the message will be 'ON level'.





















Min Distance


Defines the levelling accuracy. If the car is within this distance from the floor level it is considered at floor level. A relevelling trip is initiated if the car is not within this distance from the floor level (smallest trip distance).

If the parameter value is set low there will be more re-leveling trips to bring the car to the floor level within the defined limit. If the value is too high the relevelling travel will never occur.

The parameter 'Relevel Distance' defines the maximum relevel zone.





















Relevel Distance


Maximum distance for which re-leveling travel will occur





















Max Door Zone


Note: When the elevator enters the destination floor, at this distance before the stopping point the message ‘’Door open enabled” ("Freigabe Tür öffnen") is generated.
In older elevators the same function is produced using KUET magnets, which also allow bridging of the doors. In MICONIC-TX installations enabling of door opening is software controlled and the bridging zone is determined by the length of the floor vane (PHUET photo-electric cell in the vane).
It is possible for the door zone to be greater than half the floor vane (max_door_zone > 100). This means that door bridging can be enabled before the PHUET enters the vane. This saves time which would otherwise be lost waiting for the system to react and for the door contacts in the safety circuit to open.





















Shaft Length


Defines the distance from the bottom to the top floor level. The distance is measured and stored after the learning trip is finished.





















Ref. Shaft Image


Elevator position counter (unit = increments) value for the bottom floor on systems with incremental car position encoder . This number is used to avoid negative numbers for the position counter for any trips below the lowest floor. It must be set to a value bigger than zero . 

Note: In the case of a group of elevators where the lowest floor is not identical (e.g. if the number of stops in the lower part of the hoistway are not the same) this parameter is set such that at any given height the position counters all produce the same reading.

Normally the value is set to 10'000 for elevators with incremental car position encoder.

For the calculation of ref_shaft_image the following formula is used:

= (TACHO_FACTOR * 'Distance from group bottom floor') + 10000

Unit for 'Distance from group bottom floor' is m (meters).





















Counterweight Balance


Required car load to obtain balance between car and counterweight.

Example: Value 40 for P_ANT_CWGT_BAL means that the total counterweight weight equals 40% of the nominal load capacity of the car plus the weight of the empty car.
In North America, the actual value must be determined by Applications Engineering.





















Selector Translation Table


The selector_translation_table (STT) converts the drive related stop (floor) number value (selector/position) to the control (group) related stop (floor) number. 
The drive only 'knows' about the levels (floors) where it stops and counts these floors 1, 2, 3, 4....etc. The control however knows which elevators in the group serve which floors/entrances. This definition is done with the table 'served_floors'. The floor counting on the control side is always group related (1, 2, 3...etc.).
The selector translation table translates between these 2 numbering systems 
(when a selector or position telegram is sent from the drive to the control the floor value is calculated from drive numbering to group numbering).
The selector translation table must have as many entrys as the number of floors served by the lift. 
(The table may have more entrys but then the values for the exceeding entrys are 0.)








example: Lift A and B serving floors as follows:

A	B

5	4
4	3
3	2
2	
1	1

On the control side the floor number for a common floor is the same as for lift A and lift B.
The selector translation table for lift B is defined as follows (lift B has 4 floors)

STT[1] = 1
STT[2] = 3
STT[3] = 4
STT[4] = 5

In the above example, when the lift B travels from floor 1 to floor 3 the selector value will change directly from 1 to 3.













Cabin height


Defines the height of the car. Distance from floor level (with car at floor) to the top of the car. The parameter is used for hoistway access control.





















Hoistway Access Zone  Top


Maximum distance the car travels in down direction with the hoistway top access switch JACT (unit = mm).





















Hoistway Access Zone Bottom


Maximum distance the car travels in up direction with the hoistway bottom access switch JACB (unit = mm).





















Other Dual Shaft Elevator


Elevator identification number of the other elevator/lift in the same hoistway for which the monitoring (limitation) of the car synchron travel is required. With value 0 the function is switched off.





















Min Car Distance Synchro Travel


Required minimum distance between cars when both cars travel in the same direction and at a speed above 'Maximum_Synchro_Speed' [mm].
Default value: 2xHK
