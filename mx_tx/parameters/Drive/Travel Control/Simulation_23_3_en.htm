Simulation

Summary

 This section describes the parameters in the parameter window Simulation of the parameter group Travel Control in subsystem Drive.

Simulation Flag

For activation of the internal Travel Control simulation which is independent of the control.
The simulation is meant for drive testing (without the controller). During this simulation trip commands are generated but the door(s) will not open.

0: Simulation switched off.
1: Simulation switched on.

 














Simulation Type


Type of travel control internal traffic simulation generating trip commands:

0: ANY_FLOOR_TRAFFIC: Trip to random floors
1: ONE_FLOOR_TRAFFIC: Trip to next floor in current direction. Change of direction at top or bottom floor.
2: END_FLOOR_TRAFFIC: Trips between top and bottom floors





















Simulation Interval


Waiting time after trip is finished until next destination command is generated during internal simulation. ( see also P_ANT_SIM_FLAG )

Note: If this value is 0 (or very low) it is possible that more than 240 trips per hour are carried out because the door(s) do not open. This may cause thermal overload of the drive the system.
