DC-Drive

Summary

 This section describes the parameters in the parameter window DC-Drive of the parameter group Travel Control in subsystem Drive.

Extra starting load

Drive extra Starting

 














Field weakening start


For SSDC drives we must reduce the motor field current as we approach full speed to limit the motor voltage.  This parameter specifies the speed at which we start reducing the field current.  Values typically range from 70 to 100 % of full speed.  The actual value must be specified by the Applications Engineer based on motor characteristics, A2, J3 and the rate of change of the field supply.

For VF-70 systems in North America which have ride quality problems, we have the option to change the switching frequency at low speed, which will reduce vibration in the motor.  Below the speed specified by this parameter ( scale: 2000 = 1 m/s) the switching frequency is set by switch_freq2 instead of switch_freq.
Unless there is a problem on a job, this should be disabled by setting to 0.





















Field weakening stop


For SSDC systems, since we reduced the motor field current during acceleration, we must increase it again during deceleration.  This parameter gives the % of full speed at which we must start increasing it.   The typical value varies from 80% to 100%. The actual value must be computed by Applications Engineering based on the motor characteristics, J5, and the field supply rate of change.

                   For VF-70 systems in North America, when a car has ride quality problems, we reduce motor vibration by changing the switching frequency.  This parameter specifies the switching frequency to be used at low speed (below the speed specified by speed_liwith_sf2).  It must be between 80 and 240 (33.3 khz - 11.1 khz) for the feature to work.
