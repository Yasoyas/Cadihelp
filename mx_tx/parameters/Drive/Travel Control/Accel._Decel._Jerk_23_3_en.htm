Accel./Decel./Jerk

Summary

 This section describes the parameters in the parameter window Accel./Decel./Jerk of the parameter group Travel Control in subsystem Drive.

A Max.

Maximum value acceleration and deceleration rate (mm/s2)

The above listed values for geared installations have been taken from the
document EZ-AN 92707 of D. Budic.
The Field personnel cannot change the values of a2, a6 over these values with
CADI / LCD.

In North America, the machine, drive and power supply are sized to provide accel
and decel at a2 and a6. We do not want the field to be able to adjust it above this
value, so we set a_max = a2.

 














A Relevel


It is recommended that a higher value be selected so that the elevator quickly reaches the relieving speed. High acceleration values at low speed values are not perceptible in the car.





















A Standard


The system does not allow an increase above the value of the invariable parameter A_MAX.





















Acceleration  A2


Acceleration rate for normal trips. (mm/s2)

comments: 
In buildings with a requirement for high travel comfort this value can be set lower. 
In North America, this value is generated by Applications Engineering based on job specifications. Typical values are:

Ride comfort / Geared:    800
Fast floor time / Geared: 1000
Ride comfort / Gearless: 1000
Fast floor time / Geared: 1200





















Deceleration  A6


Deceleration rate for normal trips. (mm/s2)

comments: 
In buildings with a requirement for high travel comfort this value can be set lower. 
For installations in North America the same value is used as for A2





















J Relevel


It is recommended that a high value be selected so that the elevator quickly reaches the releveling speed. High acceleration values at low speed values are not perceptible in the car.





















J Standard


The system does not allow an increase above the value of the invariable parameter j_max.





















First Jerk Curve  J1


The value must be smaller than  j_max.





















Second Jerk Curve  J3


The value must be smaller than  j_max.





















Third Jerk Curve  J5


The value must be smaller than  j_max.





















Fourth Jerk Curve  J7


The value must be smaller than  j_max.





















AKNV


The parameter AKNV is used for the calculation formula for control of the emergency stop deceleration on AC-gearless maschines.
The field personnel can change the value between 1100...1600 mm/s2 with CADI  / LCD.





















Acceleration for NTSD


Deceleration rate used for the function 'normal terminal slowdown' NTSD. The deceleration rate is used to calculate the NTSD trigger point on MoSIS.
