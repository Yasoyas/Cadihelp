Speed

Summary

 This section describes the parameters in the parameter window Speed of the parameter group Travel Control in subsystem Drive.

V4

Constant speed reference (for fourth phase of speed curve).

The drive internal parameter for speed limit v_max is calculated from P_ANT_V4:
v_max = V4 * 1.1

The drive internal parameter move_shutdown is calculated from P_ANT_V4:
move_shutdown = ( ( ( v4 + 3000) / 1000 ) * 100 )

 














V Relevel


A practical value may be at any point between 10 and 100 mm/s. At very low values releveling travel will be very long, but will be barely perceptible. At very low values (less than 5 mm/s) and with gear driven elevators with 2.5 m/s there is a risk that the inverter will hardly react at all to very low reference values (gear friction very high).
At very high values it may occur that the speed is never reached and only the jerk and acceleration values are critical in re-leveling.





















V Learn


It is possible to increase the speed for learning trips and it is useful to do so if the travel height is considerable. The same restrictions apply, however, as for v_synch.





















V Synch


It is possible to increase the speed for synchronization trips and it is useful to do so if the travel height is considerable. As the braking distance for these trips is determined by the KSE distance, v_synch can only be increased slightly. 
If the KSE distance is 2000 mm long, v_synch must not exceed 1500 m/s, otherwise the elevator will hit the emergency limit switch on synchronization trips.





















V Recall


This value is limited internally to 630 mm/sec. This value can also be reduced or increased from the electrical recall control station by switching to recall and holding the JHM button depressed whilst pressing the DRH-U or DRH-D button for a few seconds.





















V Insp


This value is limited internally to 630 mm/sec. This value can also be reduced or increased from the inspection control station by switching to inspection and holding the JHC button depressed whilst pressing the DREC-U or DREC-D button for a few seconds.





















V Reduced


Parameter is used for function 'Dual Speed Slowdown' . It defines the maximum speed of the car when it is travelling within the hoistway zone where reduced speed applies. The reduced speed zone is defined with the parameter 'Reduced Speed Zone Bottom'. When the parameter 'V_Reduced' is set to 0 the function is disabled.





















Reduced Speed Zone Bottom


Lower limit of zone in hoistway within which the car shall run below the speed defined with P_ANT_REDUCED_SPEED. The value defines the distance in [mm] from the bottom floor level to the lower end of the reduced speed hoistway zone.





















Air Shutter Speed Limit


Speed limit [mm/s] for the control of the ambient air channel shutter equipment. Once the car exceeds the speed defined by this parameter the control switches the relay to open the ambient air channel shutters (RAACS).





















V Max Synchro Travel


For the monitoring of 'Car Synchron Travel'. Synchronous travel of cars allowed below this speed limit. See also parameter
'MIN_CAR_DISTANCE_SYNCHRO_TRAVEL'.
