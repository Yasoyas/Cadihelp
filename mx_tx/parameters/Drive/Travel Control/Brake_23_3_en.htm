KB Feedback Type


Type of the brake on machine. Defines the expected behaviour of KB/KB1 contacts.

Values: 
{0 =  NO_BRAKE}
{1 =  BRAKE1} : not used
{2 =  BRAKE2} : not used
{3 =  BRAKE3} : not used
{4 =  BRAKE4} : not used
{5 =  BRAKE5} : not used
{6 =  BRAKE6} : not used
{7 =  BRAKE7} : not used
{8 =  MAGNET_BRAKE1} : Magnet Brake with two active closed contacts KB/KB1. Both contacts open when brake closed. (wiht interface to MMB type brake module) 
{9 =  MAGNET_BRAKE2} : Magnet Brake with active open and active closed contacts KB/KB1. KB closed when brake closed/KB1 open when brake closed. (wiht interface to MMB type brake module) 
{10 =  MAGNET_BRAKE3} : not used
{11 =  HYDRO_BRAKE1} : Hydraulic brake with active open and active closed contacts KB/KB1. KB closed when brake closed/KB1 open when brake closed. (wiht interface to MMB type brake module) 
{12 =  HYDRO_BRAKE2} : Hydraulic brake with two active closed contacts KB/KB1. Both contacts open when brake closed. (wiht interface to MMB type brake module) 
{13 =  HYDRO_RES1} : spare
{14 =  CM_BRAKE1} : Magnet Brake with two active closed contacts KB/KB1. Both contacts open when brake closed. (wiht interface to non-MMB type brake module) 
{15 =  CM_BRAKE2} : Magnet Brake with active open and active closed contacts KB/KB1. KB closed when brake closed/KB1 open when brake closed. (wiht interface to non-MMB type brake module)





















KB Time


Max. waittime for KB feedback





















SGRB Time


Time for pickup the brake with high voltage. After sgrb_time the voltage on the brake will be reduced.





















Brake Start Delay


Time delay after brake opening until the trip starts. During this time the control holds the car on the floor by sending speed reference values = 0 to the inverter. This time is necessary to allow the elevator to start without brake slide.





















Brake Stop Delay


Delay until the brake closes after the car has arrived at floor level (typically used during commissioning/maintenance).

Early Brake


With P_ANT_EARLY_BRAKE set to 1 the travel control opens the brake during the start preparation as soon as possible (after the door(s) start(s) to close). Once the brake is open and the doors are still closing until the 'start_ok' command from the control is received, the travel control regulates to the velocity 0. The drive takes off from the floor with overbridged door contacts in order to eliminate start delays caused by bouncing of this contacts.
Set P_ANT_EARLY_BRAKE to 0 to disable the function. In this case the travel control software will open the brake only when the door(s) are locked and starts to accelerate from the floor with the door overbridging unit disabled.





















Earlybrake Delay


This time should be set only when the parameter Early_brake is defined as TRUE.
The idea of this time is to prevent the car of perceptible jerks when the brake has to be closed in case of reversing the doors. The time should be defined such that the IVXVF opens the brake (and regulates to velocity 0) when the doors are 3/4 closed.
