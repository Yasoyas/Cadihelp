DMC Settings

Summary

 This section describes the parameters in the parameter window DMC Settings of the parameter group Settings in subsystem Drive.

DMC Type

DMC control type

0: off
1: on

 














Suspension Rope


Identification of suspension rope





















Number of Suspension Ropes


number of suspension ropes





















Compensation Rope


Identification of the compensation rope. Used by the software to select a complete set of compensation rope parameters required by DMC.
6 digit number, where the 4 last digits are equal to the 4 last digits of the Schindler Id.No. value defined in the TK.








1)	steel rope DRAKO wire rope 300T +16.0 N 1 570 RS 
	Schindler Id.No. = 998374, SUSR = 008374
2)	steel rope DRAKO wire rope 300T +22.0 N 1 570 RS
	Schindler Id.No. = 59100018, SUSR = 010018
