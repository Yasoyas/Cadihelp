Magnetek-Supervisor

Summary

 This section describes the parameters in the parameter window Magnetek-Supervisor of the parameter group Motor Control in subsystem Drive.

Current Limit

Current limit in percent of the nominal current. (magnetek function #1).

Defines the maximum current the drive will supply for a short time as a percentage of the capability of the drive. If the drive is rated for 190 Amps, and the motor needs 380 Amps to accelerate up with full load, then the current limit should be set at (380/190) * 100 = 200%

 














Overspeed


This sets the level for the overspeed monitor in the drive.  (magnetek function #12).
The drive will shut down if the motor turns at more than this percentage of contract speed. It is normally set to 110.





















V Sense


This defines the armature voltage level at which the encoder loss and reverse encoder monitors start working. (magnetek function #14).
These monitors are turned off at very low speeds, and turn on at this percentage of full speed armature voltage. It is normally set to 25.























T Sense


Defines the sensitivity of the encoder loss monitor.  (magnetek function #15).

If the car is running and the encoder feedback is lower than it should be by this percentage, the car will shut down. It is normally set to 5.























I Decay Ramp


This controls the rate at which the current is ramped down when the drive is turned off.  (magnetek function #85).
This is set to 0.2 and should not be changed without special permission from engineering.





















Analog Speed Ref Zero


Delay time before turning off the output contactor when the drive is turned off. (magnetek function #86).

This is set to 0.3 and should not be changed without special permission from engineering.





















Overload Time


This is used to change the current vs time characteristics of the over-current monitor.  (magnetek function #83).
This is set to 90 and should not be changed without special permission from engineering.





















Overload Level


This is used to change the current vs time characteristics of the over-current monitor. Above this level the overcurrent monitor starts to operate.  (magnetek function #84).
This is set to 1 and should not be changed without special permission from engineering.
