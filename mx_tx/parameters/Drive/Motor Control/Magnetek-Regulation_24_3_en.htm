Magnetek-Regulation

Summary

 This section describes the parameters in the parameter window Magnetek-Regulation of the parameter group Motor Control in subsystem Drive.

System Inertia

Defines the inertia of the entire elevator system,in terms of the time it would take to accelerate the car from standstill to contract speed at rated torque. This value must be set correctly before attempting to fine tune the response or stability parameters. On new installations this value should be computed by applications engineering. For modernization there is a proceedure for estimating it in the Installation and Adjustment manual. (magnetek function #41).
This value must be set correctly before attempting to fine tune the response or stability parameters.
To compute this, take the rated current of the selected drive ( 100A, 190A, 300A) and look in the machine OI to find the torque produced by the hoist motor at this current (ft-lb)
Find the total inertia of the elevator reflected back to the motor armature as calculated by VT/TAP. VT/TAP output was in units of lb-ft-s**2; multiply this by gravity (32.2 f/s/s) to get inertia in appropriate units ( lb ft**2).
Calculate the time it would take to accelerate the car from zero to contract speed if this torque were applied:
t (seconds) = system inertia ( lb ft**2) * motor armature rpm at contract speed / 308 / motor torque at drive rated current (lb ft)

 














Lo Speed Bandwidth


This defines the responsiveness of the drive SPEED regulator. (magnetek function #40).
Too large values result in jitter, too small results in underspeed or overspeed errors, since the car lags behind the pattern.
Do not try to fine tune this until the parameter inertia is correct.





















Stability


This controls the amount of dampening of the speed regulator (similar to acceleration feedback).  (magnetek function #42).
Do not try to fine tune it until the parameter inertia is correct.





















Tacho Rate Gain


Used to compensate for jitter from the tacho.  (magnetek function #107)

This should not be required to adjust for elevators with digital encoders. It is normally set to 0





















Current Reg Crossover


This sets the responsiveness of the CURRENT loop regulator.  (magnetek function #8).

If too large, the motor current will jitter; if too small, the motor will be sluggish.























Use Self Tune Values


Defines if the drive uses the values for 'motor resistance', 'motor inductance' and 'field time constant' measured with the self-tune function or the values entered manually (or defined in the PCT).  (magnetek function #2).
It must be set to zero until a self tune is done, since there are no measured values available. After the self-tune is done it should be set to 1 so the measured values are used. If the parameter cannot be set to 1, then the self-tune was not successful.
