Motor (Asynchron / Synchron / DC)

Summary

 This section describes the parameters in the parameter window Motor (Asynchron / Synchron / DC) of the parameter group Motor Control in subsystem Drive.

Id Motor

Identification of the motor. The motor parameters are derived from the parameter table on the inverter based on the value for the motor setting. This parameter needs to be edited only if the motor parameters selected are not commissioned individually.

Value: 4 last digits of the Schindler Id.No.
Example: the motor VM132-C4A240 is identified with the Id.No. 147964. Therefore, the correct value for the Motor Id is 7964.

The value 0 corresponds to NO_SETTING. A negative value indicates which value the parameter had originally i.e. before at least one motor parameter was edited and will be set automatically by software..
All motor parameters are defined in DR-92069_Vxx.
Motor list

 














Motor Type


Type of Motor. Used to define whether motor parameter estimation has to be executed or not.

Values [-]:
{0.0= SCHINDLER_VM_FM }	all Schindler frequency converter motors with the identifiers FM.... or VM....
{1.0= OTHER}		all other induction motors (asynchronous motors)
{2.0 = SYNCHRONOUS}	synchronous motors
{3.0 = DC_MOTOR}		DC motor





















f Motor Nominal / f Motor Nominal / P Motor Nominal


AC motors: Rated frequency of the motor [Hz]
DC motors: Rated motor power [kW]





















U Motor Nominal / U Motor Nominal / U Motor Nominal


AC motors: Rated Motor Voltage (line to line voltage, RMS); is used to normalize parameters and input/output variables of the vector control.

DC motors: value to be set according to the motor plate (no Motor Id's available for DC motors)

Designation  according to EE-DR 93376:	UMN





















I Motor Nominal / I Motor Nominal / I Motor Nominal


Rated-Motor Current (A)

For DC motors the value must be set according to the motor plate (no Motor Id for DC motors)





















Isa Motor Nominal / Flux / --


AC Induction (Asynchronous) Motors: Magnetizing motor current (RMS) 
Designation  according to EE-DR 93376:	IM0
Higher value cause a higher motor voltage, which leads to incorrect functioning of the two point regulator (refer to error handling) and higher motor temperatures. Smaller values may cause too high motor currents which may lead to inadequate acceleration power and to an error ( refer to error handling).

AC Synchronous Motors: Flux
Permanent Magnet Flux

DC Motors: Field current of the motor. Value must be set according to the motor plate data (no Motor Id for DC motors)





















Tr / Zero Position / --


AC Induction (Asynchronous) Motors: Rotor time constant.
Used for the calculation of the transformation angle and the flux. Higher values will increase the motor voltage, which may lead to incorrect operation of the two point regulator (refer to the error handling) and increased motor temperatures. Smaller values cause higher motor currents leading to inadequate acceleration power and to an error (refer to error handling). The value is set by the commissioning test Estimate Motor Parameters for motors without a known motor id.

AC Synchronous Motors: Zero Position
Angle between the permanent magnet north pole and zero positon of the motor encoder. The value is determined with Zero Position Test during commissioning.

DC Motors: Motor armature resistance [Ohm]
The Value must be set according to motor TK (no Motor Id for DC motors available)





















Xh / I Motor Peak / I Motor max


AC Induction (Asynchronous) Motors: Main reactance of the motor equivalent switching circuit. It is used for standardization of parameters and input and output size of the vector regulator. Value is derived from P_ANT_MOT_SETTING on the inverter. The value is set by the commissioning test Estimate Motor Parameters for motors without a known motor id.

AC Synchronous Motors: Maximum Motor Current ('IMotPeak')

DC Motors: Maximum armature current





















Xr / (--/Xsd) / La


AC Induction (Asynchronous) Motors: Rotor inductance of the motor equivalent switching circuit and is used for standardization of parameters and input and output size of the vector regulator. Value is derived from P_ANT_MOT_SETTING on the inverter.

AC Synchronous Motors: depending on the type of synchronous motor the parameter is either not used (-) or corresponds to the synchronous d - inductance (Xsd)
Condition for "not used": Not used, if only Xs is available, i.e. if there is no extension "d" or "q" (e.g. PMS 230, SM 250)
Condition for Xsd: The parameter corresponds to Xsd, if Xsd and Xsq are given (e.g. PMS 420)

DC Motors: Armature inductance
The value must be set according to the motor TK (no Motor Id for DC motors available)





















Xs / (Xs/Xsq) / --


AC Induction (Asynchronous) Motors: Stator inductance of the motor equivalent switching circuit is used for standardization of parameters and input and output size of the vector regulator. Value is derived from the setting P_ANT_MOT_SETTING on the inverter.

Synchronous Motors: 
Depending on the type of synchronous motor the parameter corresponds either to the synchronous inductance (Xs) or to the synchronous q inductance (Xsq).
Condition for Xs: The parameter corresponds to Xs, if only Xs is available, i.e. if there is no extension "d" or "q" (e.g. PMS 230, SM 250)
Condition for Xsq: The parameter corresponds to Xsq, if Xsd and Xsq are given (e.g. PMS 420)

DC Motors: Field inductance (not used yet).
The value must be set according to the motor TK (no Motor Id for DC motors available)





















Rs / Rs / --


AC inverters/motors: Stator resistance of the motor.

DC inverters/motors: Field resistance (not used yet)
The value must be set according to the motor TK (no Motor Id for DC motors available)





















Nominal Torque / Nominal Torque / --


Rated torque of the motor [Nm]. Used to calculate the starting torque (load measurement) and feedforward torques (direct-traction).





















Pole Pairs / Pole Pairs / n Motor Nominal


AC motors: Number of pole pairs of the motor

DC motors: Rated motor speed [1/min], used to calculate the rated torque and for scaling of speed. Value must be set according to motor plate (no Motor Id for DC motors).
