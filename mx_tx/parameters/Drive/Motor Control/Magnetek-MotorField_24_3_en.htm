Magnetek-Motor Field

Summary

 This section describes the parameters in the parameter window Magnetek-Motor Field of the parameter group Motor Control in subsystem Drive.

Rated Field Volts DC

Motor field voltage (DC) required to get full field current. (magnetek function #52).

 














Full Field Current


Defines the motor field current to be used during acceleration and deceleration when no field weakening is applied.  (magnetek function #50).





















Weak/Running Field Current


Defines the motor field current which will be used when running at full speed.  (magnetek function #49).

It is typically set to about 90% of full field current.





















Standby Field Current


Defines the motor field current to be applied when the car is parked for a short time. (magnetek function #53)
It is typically set to about 40 % of full field current. After 6 minutes it will be turned off fully. 





















Field Change Rate Fast


This controls the rate at which the motor field current ramps from OFF or STANDBY up to FULL. It is typically set to 2.  (magnetek function #56).





















Field Change Rate Slow


This controls the rate at which the motor field current ramps between FULL and WEAKENED.  (magnetek function #58).

The speed at which field weakening starts is defined by the parameter field_weakening_start (as a percentage of contract/nominal speed).  The field is fully weakened when the car reaches full speed.  The time also depends on the contract speed (V4), the acceleration rate (A2) and the jerk rate (J3). Can be computed as:
t = SQRT (  ((A2 **2) / (J3 **2))  +  2 * V4 * ( 1 - field_weakening_start/100 )  / J3 )  - A2 / J3





















Field L/R


Motor field time constant (magnetek function #51).
This value will be adjusted in the field using the self-tune procedure.





















Measured Field Time Constant (L/R)


Measured L/R  (magnetek function #615).

Time constant of the motor field windings that was measured in the self-tune process. It is used to run the car if the parameter use self tune is set to 1. 
It is for display only and is not editable. Unit = s (Seconds)





















Field Response


This controls the gain of the current servo in the motor field regulator. It is typically set to 5.   (magnetek function #54).





















Measured FieldCurrent


Measured motor field current (magnetek function #612).

This displays the actual motor field current reading.
It is for display only and is not editable. Units = A (amps)





















Overnight Field Current


Overnight Motor Field Current in % of nominal field current. (magnetek function #48).

This is the % of standing field current provided by the motor drive immediately after powerup, and approximately 2 minutes after the lift sits idle.
[ Note: the available field current values are 'Full', 'Running', 'Standing' and 'Overnight' ]





















Field Source AC Voltage


AC voltage for motor field in VAC   (magnetek function #55).

This is the AC input voltage used for the AC to DC motor field conversion.





















Field Resistance


Resistance of motor field coils [Ohms]   (magnetek function #60).

This is the measured DC resistance of the motor field coils.
