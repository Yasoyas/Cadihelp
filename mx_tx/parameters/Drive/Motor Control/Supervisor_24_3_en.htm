Supervisor

Summary

 This section describes the parameters in the parameter window Supervisor of the parameter group Motor Control in subsystem Drive.

Over Speed Factor

Overspeed factor. Used for overspeed monitoring. ( see also error 0705 'Speed difference' )

 














Over Current Factor


Maximum admissible motor current for the duration of constant speed. The factor relates to the rated motor current "IS_NOM".

The effective limits are:

Gearless induction motors and SGB:	over current fact * 1.15 =  1.38
Planetary gear:			over current fact * 1.2 =  1.44
Worm gear:			over current fact * 1.25 = 1.5
Synchronous motors (typically underslung):	over current fact * 1.25 = 1.5
(refer to error description)





















Is Diff Limit


Maximum admissible deviation between the reference current and the actual current in the rotor flux oriented reference frame. Scaling: The number 1024 corresponds to the rated motor current.( see also error 0713 'current difference' )





















Inverter Temp Limit


Maximum temperature of the heat sink.





















U DC Link Drift Limit


Maximum DC link voltage mid-point drift





















U DC Link LOW Limit Idle


Lower DC link voltage limit at trip phase standing (refer to remark) with net (mains power) connection using SGR/SH. Used for undervoltage monitoring (refer to error description). Values [V]:

{276}	inverter input voltage: 230 V
{446}	inverter input voltage: 380 V
{470}	inverter input voltage: 400 V
{487}	inverter input voltage: 415 V
{540}	inverter input voltage: 460 V
{563}	inverter input voltage: 480 V

remarks:	definition of standing:
- all inverters except the ones with FCR:	motor switched off
- inverters with FCR (line converter):	FCR and motor are switched off





















U DC Link UPP Limit Idle


Upper DC link voltage limit at standing (refer to remark of parameter 'DC Link Min Voltage Idle'), with net connection using SGR/SH. Used for overvoltage monitoring (refer to error description)

Values:	
{364} with inverter input voltage: 230 V
{602} with inverter input voltage: 380 V
{634} with inverter input voltage: 400 V
{657} with inverter input voltage: 415 V
{729} with inverter input voltage: 460 V
{760} with inverter input voltage: 480 V

remarks:	parameter-relation: see Inverter Input Voltage





















U DC Link LOW Limit Ride


Lower limit of DC link voltage during trip. Used for undervoltage monitoring. (refer to error description)

Values:
{255} with inverter input voltage: 230 V
{422} with inverter input voltage: 380 V
{444} with inverter input voltage: 400 V
{460} with inverter input voltage: 415 V
{510} with inverter input voltage: 460 V
{533} with inverter input voltage: 480 V

remark:	parameter-relation: see Inverter Input Voltage 





















Dist Diff Limit


Maximun admissible travel distance into wrong direction





















Motor Temperature Limit


Maximum allowed operating temperature of the hoisting motor [C]. 

{140.0}	for all Schindler FM/VM/PMS machines (see also Type motor)
{90.0}	FMB130-(LANC)  
{120.0}	FMB130-(MLS) and Basic
{140}	all other (default value)

Remark: High limit temperature of the hoisting motor. The value is set by the inverter depending on the Motor Id. For unknown motors the default value is used (140degC).
