Inverter

Summary

 This section describes the parameters in the parameter window Inverter of the parameter group Motor Control in subsystem Drive.

Inverter

Setting value for the frequency converter. Each value identifies a specific inverter type. Based on the setting value the converter software selects the correct values (predefined) for all derived values (target configuration on motor control). The derived parameters from the inverter setting value are all parameters in the parameter group 'inverter' (e.g. inverter_type, line_converter_type, brake_chopper_type, current_sensor_type, etc.)

If this parameter is not used for commissioning of the inverter, the inverter parameters (normally derived from the setting value) must be defined individually in the PCT.

The value 0 corresponds to “NO_SETTING”. A negative value indicates which value the parameter 'inverter setting' originally had and that at least one parameter on the inverter was modified. The inverter software sets the negative value automatically when a parameter is modified .
The parameter 'Inv Current Ampl' is selected for the corresponding Motor Control PCB (effective as from PVEC VecSys Software V2.05, older versions select 'Inv Current Ampl' for the PIOVEC boards QA and QB).

Remark valid for VF120 and VF170: The values of the parameters 'Inv Current Ampl', 'Inv Current Scale' and 'Current Sensor Type' are set according to the OCP current measurement (effective as from PVEC VecSys software V2.05, older versions according to the LEM current measurement).

Parallel Converters: use the setting of the single converter connected in parallel.

Frequency converters VF11BR, VF22BR, VF33BR and VF44BR may correct parameter 'Inverter Id' according to the detection of the power unit and select the corresponding parameters automatically (hardware recognition).

Contactor Mode


Defines the behaviour of the Inverter Contactor (SGRW/SH).
Bit 1 = 0: SH used for EN81 safety interface X15 (contactor monitoring) / SH turns off after every trip
Bit 2 = 0:  Mains separation/discharging with SGRW and SH after 2hrs (to save energy) 

Regardless of the parameter value separation from mains is done if the inverter is not available (Error State = BLOCKED or DEAD)

Values [-]:	
{0}	default
{1}	SH remains on after trip,  e.g. for configurations with output contactors (IVXVF)
{2}	Inverter remains charged using SGRW (with Error State = START_ALLOWED)
{3}, {1} and {2}	mains separation and safety interface disabled (if Error State = START_ALLOWED)

This parameter with PCT id. h5F and inverter comm. id 80(dec) was formerly named 'Inv Lock Time' but was not used by the inverter. With inverter SW versions VecSys V2.16 and VARIOsys V1.08 the parameter was renamed to 'Inv Contactor Mode'.

I Rem Upp Lim


Limit of discharging current representing capacitor’s leakage current. Parameter is used for test of capacitors for pulse forming. Value [mA]:

100:	VF11BR, VF22BR, VF33BR, VF44BR, VF88BR, VF30BR, VF50BR, VF85BR

200:	VF100BR, VF120BR, VF122BR

250:	VF120PF1

300:	VF170PF1, VF280PF1, VF155PF1, VF288PF1

The value is determinated according to:
I_rem = C_dcl [ { U(t1) - U(t2) } / delta_T ]
with t1 = 0.5s and t2 = 2.5s (discharging time)
