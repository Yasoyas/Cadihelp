Hoistway

Summary

 This section describes the parameters in the parameter window Hoistway of the parameter group Motor Control in subsystem Drive.

Counterweight Mass

Mass of counterweight [kg]

 














Hoistway  Efficiency


Efficiency of hoistway [%]





















Nominal Speed (Setting)


Nominal speed of car.
Parameter for the fourth phase in the speed curve (V4).
Note: The system does not allow an increase above the value of v_max = V4 * 1.1.

The following parameters are set or calculated internally based on the value of the nominal speed:
set A2, A6, J1, J3, J5, J7, V_INSP, V_RECALL, V_SYNCH, V_LEARN to default values (MX-GC: special values); 
NA only: A_MAX = A2 ;





















Reeving Factor (Setting)


Type of suspension. ( 1:1 / 2:1 etc.)

1:  ROPING 1:1
2:  ROPING 2:1
4:  ROPING 4:1
6:  ROPING 6:1
8:  ROPING 8:1
(also known as 'Reeving Factor' / CADI uses the term 'Roping'.)





















Load Measurement Type (Setting)


Type of load measurement device/system.

{0} NO_LMS [COS] Installation without load measurement
{1} KL_LMS [COS] Installation with load contacts
{2} CAR_DMS [COS Installation with DMS (strain gages) placed under the car platform
{3} HP_DMS [COS] Installation with DMS placed on hitch plate (dead end or crosshead)
{4} HP_LC5K [COS]	Installation with 3 load cells placed on hitch plate each rated to 5000 £ (2400 Kg)
{5} HP_LC10K[COS]	Installation with 3 load cells placed on hitch plate each rated to 10000 £ (4800 Kg)
{6} HP_LVDT Installation with LVDT sensor (output 4-20 mA) placed on hitch plate
{7} BRK_DMS Installation with DMS sensor placed on brake construction
{8} HP_DGS Installation with Sensors from DIGISENS placed on hitch plate
{9} DD_CAR_DMS Installation with DMS (strain gages) placed under the car platform with double decker elevators






















Nominal Load (Setting)


Nominal car load [kg]. For double deck elevators this is the sum of the nominal loads for the lower and the upper deck. E.g. if a DD elevator is specified with a nominal load of 2 x 1600kg then the value for P_ANT_NOM_LOAD is 3200kg.

On first generation VX/TX systems, nom_load was defined twice (also in the LIFT subgroup of the PCT). For TX-GC2/MX only the P_ANT_NOM_LOAD is required.

The calculation of the drive internal load limit values for the load-measuring system is based on the parameters NOM_LOAD and CODE_TYPE
for NA:
kl_h_load = NOM_LOAD / 2 ,
kl_max_load = NOM_LOAD * 1.1 ,
kl_x_load = NOM_LOAD * 1.3 ,

others (EUAP):
kl_h_load = NOM_LOAD / 2 ,
kl_x_load = NOM_LOAD * 1.1 ,

these limits are independent of load limits defined in PCT_GP_RS_GENERAL





















Counterweight Balance


Required car load to obtain balance between car and counterweight.

Example: Value 40 for P_ANT_CWGT_BAL means that the total counterweight weight equals 40% of the nominal load capacity of the car plus the weight of the empty car.
In North America, the actual value must be determined by Applications Engineering.





















Shaft Length


Defines the distance from the bottom to the top floor level. The distance is measured and stored after the learning trip is finished.
