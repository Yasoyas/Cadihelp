Magnetek-Elevator

Summary

 This section describes the parameters in the parameter window Magnetek-Elevator of the parameter group Motor Control in subsystem Drive.

Rated Speed

Contract/nominal speed of the elevator. (magnetek function #17).

 














Motor RPM


Motor RPM when the car runs with nominal speed.   (magnetek function #11).

RPM =  Car speed (m/s) * 60 * roping ( 1 if 1:1, 2 if 2:1)  * gear ratio (1 if gearless) / sheave diameter (m) / Pi





















Encoder PPR (Setting)


Defines the number of pulses per revolution for the motor encoder.  (magnetek function #10).





















Encoder/ MTR Ratio


Ratio of the motor encoder RPM to motor RPM.  (magnetek function #16 'Encdr/ MTR Ratio').

For rim tachos (encoders), this is the diameter of the sheeve where the tacho rolls divided by the diameter of the wheel on the rim tacho ( ratio = sheave diameter / roller diameter).   For tachos mounted to the end of the motor armature ( including geared machines), set this parameter to 1.





















Nominal AC


This specifies the AC line voltage feeding the drive.  (magnetek function #9).





















Low Speed Threshold


The load difference value is used to compute the pre-torque current used at the start of a trip.  (magnetek function #64).

The system knows the difference in weight between the car and counterweight. With the car load value the actual load difference is calculated to determine the required torque. Once the load weighing is calibrated, adjust this value to eliminate roll-back. 
When it is set to zero, pre-torquing is turned off.





















Acceleration Limit


Defines the acceleration limit in the drive.  (magnetek function #21).
If the speed pattern changes faster than this, the drive will shut down. Normally set to a_max + 100 mm/s/s





















Analog Out 0 Select TB1 (45)


Defines what signal is routed to analogue output #0 (TB1 pin 45 and TP41). (magnetek function #95)

0: Output signal = Speed reference
1: Output signal = Trace Buffer 0

This analog test point may be connected to a chart recorder or voltmeter for debugging purposes. Normally the parameter is set to 0.






















TB1-25 Multiplier


Test point #0 multiplier. (magnetek function #97).

Used to change the scaling on the analog output at TB1 pin 45. It is normally set to 1.





















Analog Out 1 Select TB1 (46)


Defines what signal is routed to analogue output #1 (TB1 pin 46 and TP44). (magnetek function #96).

0: Output signal = Speed feedback
1: Output signal = Trace Buffer 1

This analog test point may be connected to a chart recorder or voltmeter for debugging purposes. Normally the parameter is set to 0.





















TB1-26 Multiplier


Test point #1 multiplier. (magnetek function #98)

Used to change the scaling on the analog output at TB1 pin 46. It is normally set to 1.
