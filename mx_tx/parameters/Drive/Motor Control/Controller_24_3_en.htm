Controller

Summary

 This section describes the parameters in the parameter window Controller of the parameter group Motor Control in subsystem Drive.

P Gain Speed

Proportional amplification for the speed controller. P Gain Speed is a factor that is multiplied by the internally calculated proportional gain of the speed controller. The internal calculation mainly depends on the mass moment of inertia and the motor parameters.

{0.5} direct-traction (gearless and SGB), induction motors (asynchronous motors)
{1.0} direct-traction (gearless), synchronous motors
{1.0} all geared drives

The value is set by the converter software depending on the parameters 'Type Gear' and 'Type_motor'.

Range of values:
for gearless and SGB drives, induction motors: 0.3 - 0.8,
for gearless drives, synchronous motors: 0.6 to 1.8
for geared drives: 0.6 to 1.8.

(On direct traction drives with long travel heights and high masses this is a commission dependent parameter.)

 














Ti Speed


Time constant of the speed controller.

Default values [s]:	
{0.12} for induction motors (asynchronous motors)
{0.02} for synchronous motors

Range of values: 
induction motors: 0.06 to 0.16 s.
synchronous motors: 0.01 to 0.03

The value is set by the converter software depending on the parameter 'Type_motor'





















P Gain Current


Proportional amplification factor for the current controller. 
Default value:1





















P Ti Current


Proportional amplification factor for the time constant of the current controller. Default value:1





















Zero Speed Amplification


Amplification factor of the speed controllers coefficients at zero speed.
Induction motors:
The proportional gain of the speed control is multiplied by a coefficient depending on the speed. At zero speed this coefficient is set to the value of 'Zero Speed Amp'. If the speed is larger than or equal to the parameter 'Speed Edge' then the coefficient is set to 1. If the speed is between 0 and Speed Edge, it is interpolated.

Synchronous motors:
The amplification factor is only active at the beginning of the trip at zero speed.
Values:
{1.0}	direct-traction, induction motors (asynchronous motors)
{2.5}	direct-traction, synchronous motors
{x} 	geared drives:
Gear Ratio <= 15: x = 1.0
Gear Ratio > 15: x = (1.0 + (Gear Ratio - 15) * 0.025)

remarks:	The value is set by the converter software depending on the parameters Gear Type / Gear Ratio / Speed Edge / Type_motor.





















Speed Edge


The proportional gain of the speed controllers coefficient is set depending on the speed between zero speed and the 'Speed Edge'. See parameter 'Zero Speed Amp'. The value relates to rated speed. Only used for worm gears.

Values:	
{0.0} for direct-traction
{0.1} for geared drives





















Boost Current


Proportional factor for motor current at low speeds (only for drives without incremental encoder).

values [-]:
{1.0} for inverters without incremental encoder (initial value)
{1.0} for all other cases. The parameter is not evaluated

Remark: The value is set by the software depending on the commissioning test 'Adjust Tacholess Control'.





















Static Slip Compensation


Static slip compensation (only for drives without incremental encoder). 

Values [-]:
{1.0}	inverters without incremental encoder (initial value)
{1.0}	all other cases. The parameter is not evaluated

Remark: The value is set by the software as a result of the commissioning test Adjust Tacholess Control.





















Dynamic Slip Compensation


Dynamic slip compensation (only for drives without incremental encoder). 

Values [-]:	
{3.0}	inverters without incremental encoder (initial value)
{3.0}	all other cases. The parameter is not evaluated (initial value)

Remark: The value is set by the software during commissioning as a result of the test 'Adjust Tacholess Control'.
