Answer - Timing

Summary

 This section describes the parameters in the parameter window Answer - Timing of the parameter group Phone Settings in subsystem Servitel LC.

Answer Window 1

Servitel-LC allows an existing telephone line to be shared with the customer. The customer telephone is actuated parallel to the modem. The Servitel Control Center can, if an Answer Window is defined, make one call with connection to the modem, within that time. The answer window is limited by two times. If the answer window is set on 00:00 - 00:00 , this function is turned off, and it is assumed that a separate telephone line is available for the modem.
The customer can call at any time. Collisions have to be expected when using the customer telephone with customer telephone and modem in parallel circuit.

Example:
Answer Window 1: 00:15 to 01:00 (times are in 24 hr format)
Answer Window 2: 02:35 to 02:40

One telephone call from 0:15 until 01:00 is answered by Servitel-LC, i.e. with one call the modem will respond. During that time period the Servitel Control Center can carry out Test Trips, status inquiries and other operations. From 01:00 until 00:15, i.e. during most of the time, the modem never responds. The same rule applies for Answer Window 2.

Default:
00:00 - 00:00 (i.e. No customer phone. Modem answers every call.)
