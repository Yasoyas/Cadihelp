Phone Settings

Summary

 This section describes the parameters in the parameter window Phone Settings of the parameter group Phone Settings in subsystem Servitel LC.

0: Regional Center

Regional center.

Possible characters for the Phone Settings:
0-9 (numbers)
A-D (additional digits for tone dial)
#
*
, (pause, depends on register S8 )
P (pulse dialing)
T (tone dialing)
W (wait for second dial tone)
% (wait for PABX dial tone)
@ (wait for stop signal)
! (flash, earth button)
^ (call tone switch off)
Refer the modem manual for more details.

Own Phone Number


Own phone number of LC modem





















Substract 1


If a Phone number contains at the beginning of the string the same string as in Substract 1, the modem will not dial the beginning of the Phone number.
Example: 
Phone Number Regional Center 042344523
Substract 1: 042
Modem dials: 344523





















Substract 2


If a Phone number contains at the beginning of the string the same string as in Substract 2, the modem will not dial the beginning of the Phone number.
Example: 
Phone Number Regional Center 042344523
Substract 2: 042
Modem dials: 344523
