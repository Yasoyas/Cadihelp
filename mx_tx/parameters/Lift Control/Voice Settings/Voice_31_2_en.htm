Voice

Summary

 This section describes the parameters in the parameter window Voice of the parameter group Voice Settings in subsystem Lift Control.

Volume CM Speaker

To set the volume of the loudspeaker connected to the CRM.
NOTE: Do not set speaker values outside the suggested ranges.
Speaker: 0 …11 














Volume MRM Speaker


To set the volume of the MRM loudspeaker. 
NOTE: Do not set speaker values outside the suggested ranges. 
Speaker: 0 …11

Sensitivity of MRM Mic.


To set the sensitivity of the microphone in the MRM. 
In case of an connection via the analoge telefone line, this value does also influence the volume of the incomming signal! 
NOTE: Do not set microphone values outside the suggested ranges. 
Microphone: 0 … 8 





















Sensitivity of CM Mic.


To set the sensitivity of the microphone connected to the CRM. 
NOTE: Do not set microphone values outside the suggested ranges. 
Microphone: 0 … 8 





















Car Voice Floor Announcement


Floor annoucements in car with voice:

0: off
1: only when elevator is in handicapped mode (EN81-70)       ( Default )
2: always

Not used for Z-Line II





















Car Voice Door Announcement


Door annoucements in car with voice:

0: off
1: only when elevator is in handicapped mode (EN81-70)       ( Default )
2: always

Not used for Z-Line II





















Delay Synthvoice


The synthetic voice announcement is repeated after this delay. 





















Lenght Intercomconnection


After this time the intercom connection between CRM and MRM will automatically be terminated. 





















Car Voice Direction Announcement


Voice annoucements in car for pre-announcing of travel direction:

0: off
1: only when elevator is in handicapped mode (EN81-70)     ( Default )
2: always

Not used for Z-Line II





















Car Voice Service Announcement


Voice annoucements in car for special services (Overload, Final Timer etc.):

0: off
1: only when elevator is in handicapped mode (EN81-70)
2: always        ( Default )

Not used for Z-Line II
