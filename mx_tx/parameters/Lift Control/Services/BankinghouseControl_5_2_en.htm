Bankinghouse Control

Summary

 This section describes the parameters in the parameter window Bankinghouse Control of the parameter group Services in subsystem Lift Control.

Recall Floor

BANK_FLOOR: floor where the bank counter is. It is used in service Bankinghouse Control BS1/BS2.

 














Open Doors


For service Bankinghouse Control BS1/BS2 which door sides have to be opened at parking floor.

door side mask:
 1 = ZS1     3 = ZS1 + ZS20
 2 = ZS20    ...
 4 = ZS30    14 = ZS20 + ZS30 + ZS40
 8 = ZS40    15 = ZS1 + ZS20 + ZS30 + ZS40





















Timeout Off


BANK_TIMEOUT is the reservation time to start the direct trip in service Bankinghouse Control BS1/BS2.
