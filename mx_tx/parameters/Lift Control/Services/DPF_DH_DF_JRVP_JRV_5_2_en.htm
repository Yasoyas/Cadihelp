DPF / DH / DF / JRVP / JRV

Summary

 This section describes the parameters in the parameter window DPF / DH / DF / JRVP / JRV of the parameter group Services in subsystem Lift Control.

DDF Timeout

Used for VIP service (function DFx).

Conventional controls KS/KA/PI:
Defines the maximum time for the passenger to enter the car on the DDF floor and give the command for the direct trip (DPF). If no command is entered the VIP service is switched off. (default value 3min)

M10 controls:
Defines the maximum door open time until the direct trip (DPF) is initiated by the control. The passenger may close the door before by pressing the DT-S button. (default value 10s)

 














DPF Timeout Off


Activation time limit of priority trip service DPF. After activation of priority service the priority service is switched OFF again when this time elapses if no car call is entered (with the car at rest) for the priority trip.

Default value: 30s





















DH Timeout Off


For service DH specify the time to wait after the service has been switched ON to test if it can be switched OFF.





















JRVP Timeout


OFF delay time for independent service parking functions (JRVP).

To be defined for NPE application





















JRV Side Open Door


Defines the door side to open after activation of JRV mode when both doors are closed. Only used when front and rear door is configured.

1: door on side 1 opens
2: door on side 2 opens
3: both doors (side 1 and 2) open








¨B3[ 02 02 0000 ]    * P_STR_JRV_DOOR_OPEN = 0













JRV Car Call Registration Mode


Defines the car call registration mode for car calls entered on COP. 
In mode 'single' only the last entered car call remains registered.In mode 'all' all car calls register.

0: Register all car calls
1: Register single car call








¨B4[ 02 02 0000 ]    * P_STR_JRV_CAR_CALL_REGISTRATION_MODE = 0
