Gong on Floor

Summary

 This section describes the parameters in the parameter window Gong on Floor of the parameter group Gong in subsystem Lift Control.

Amplitude on Floor

Defines the volume of the arrival gong individually for each landing/entrance (per floor and side). Used for arrival gong interface boards LONDAG/LONIOS.
Five different volumes are available. Default value is 3 'Loud'.

Available volumes:
0 = Not Used
1 = Quiet
2 = Normal
3 = Loud
4 = Very Loud

Arrival Gong Mode

Gong mode for arrival gong GA (for conventional controls).
Specifies when the arrival gong for boarding sounds.

0: Drive/car starts to decelerate to boarding floor (standard for KA/KS)
1: Drive/car starts to serve hall/floor call (standard for PI)
2: Hall/floor call registration (early announcement) - NOT AVAILABLE
3: Door opens on boarding floor
