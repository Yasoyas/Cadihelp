Melodies

Summary

 This section describes the parameters in the parameter window Melodies of the parameter group Gong in subsystem Lift Control.

Melodies

Defines the gong melodies for arrival gong interface board LONDAG/LONIOS.

P_GONG_GA_UP … P_GONG_IN_CAR_BUZZER: Gong melodies for LONDAG’s. Tone 0 terminates the gong melody.
P_GONG_GA_EXT_UP … P_GONG_GA_EXT_UP: Gong melodies for external (driven by generic IO’s) gongs.

Default Melodies:

Gong melody identifier(field 1) Gong tone identifier (field 2…9)
--------------------------------+----------------------------------
1 (P_GONG_GA_UP) | 7, 0, 0, 0, 0, 0, 0, 0
2 (P_GONG_GA_DOWN) | 7, 5, 0, 0, 0, 0, 0, 0
3 (P_GONG_FLOOR_PASSING) | 15, 0, 0, 0, 0, 0, 0, 0
4 (P_GONG_IN_CAR_BUZZER) | 8, 0, 0, 0, 0, 0, 0, 0
5 (P_GONG_GA_EXT_UP) | 1, 0, 0, 0, 0, 0, 0, 0
6 (P_GONG_GA_EXT_DOWN) | 1, 0, 1, 0, 0, 0, 0, 0

 














Tone Duration


Defines the waiting time to start next gong tone. Used for arrival gong interface board LONDAG/LONIOS.









Examples:

- Timing diagram for the gong melody P_GONG_GA_DOWN:

        start tone 1    start tone 2        end of gong sequence
   ____________|__________|____________________|___________________________
                <   1 s  > <        2 s       > 


- Timing diagram for gong melody P_GONG_GA_EXT_DOWN:

              gong tone 1           gong tone 2        end of gong sequence
              _________       ____________________
   __________|         |_____|                    |_____|__________________
              <  1 s  > 0.5s  <       3 s        > 0.5 s <

Gong Tone


Defines the gong tones, which are used to generate gong melodies. Up to 16 different gong tones can be defined.

Default Tones: 

 Gong tone identifier | Gong tone frequency
 ---------------------+--------------------
         1            |           440
         2            |           550
         3            |           660
       4..16          |             0
