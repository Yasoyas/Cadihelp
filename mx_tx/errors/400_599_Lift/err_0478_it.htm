0478 OVERLAY failure

Summary

 This section describes the Lift error OVERLAY failure.

Description

The message indicates a status change of the overlay Master and/or Slave PC (M10 overlay) detected by the control software. This message is generated when the communication with the Overlay Master/Slave PC (M10) fails.

Extra Info

Interpretation of values:

0101(hex): PC1 ok, PC2 failed, Communication to overlay OK
0011(hex): PC1 failed, PC2 ok, Communication to overlay OK
0000(hex): Communication Failure

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Failure of master or slave overlay (PC) Check overlay PC and replace if necessary
2 Communication link to master or slave overlay (PC) disconnected Check connection between control and overlay PC. Reestablish communication
3 Overlay software (application) no longer responding or shut down Check operation of overlay application
4 Problem with interface board on master/slave overlay PC Replace interface board
