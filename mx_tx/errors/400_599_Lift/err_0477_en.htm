0477 Fatal Error OFF

Summary

 This section describes the Lift error Fatal Error OFF.

Description

Message indicates deactivation of the Fatal service. See value of extra information for further details about:

- Cause of fatal service reset (manual / time)
- current service before activation of fatal service
- error number which caused the activation of the fatal service (reason)

Extra Info

Interpretation (decoding) of the 8 digit hex. value (0xABCDEFGH)

Value of 'GH'(Bit 0 to Bit 7): Fatal Reset cause
0000 0000(bin)/00(hex): Fatal service OFF after Reset Timer expired
0000 0001(bin)/01(hex): Fatal service Off after Manual Reset

Value of 'EF'(Bit 8 to Bit 15) : Service Id. number of the active (current) Service before Fatal service ON (see error 476)
For Example:
01(dec)/01(hex): Normal Service 'N'
10(dec)/0A(hex): Inspection Service 'INS'
14(dec)/OE(hex): Unavailable Service 'UNV'
27(Dec)/1B(hex): Sab Service

Value of 'CD' (Bit 16 to Bit 23): Value of Lower Byte of the error number that caused the fatal service.

Value of 'AB' (Bit 24 to Bit 31): Value of Upper Byte of the error number that caused the fatal service.

For Example: For Error number 300(dec) ‘AB’ will be 01(hex) and 'CD' will be 2E(hex)

Source ID

Identifies the software task which requests the lift control to take the system out of fatal service.

1: Init Task
45(dec): Lift Service Task
90(dec): Door Control Front Door (Module RC1)
91(dec): Door Control Rear Door (Module RC2)
117(dec): Travel Control

Causes and Actions
ID Possible causes of message Actions to be taken
1 not applicable (the message 'Fatal Error OFF' is not an error)
