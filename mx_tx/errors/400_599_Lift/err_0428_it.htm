0428 End Monitor Expired

Summary

 This section describes the Lift error End Monitor Expired.

Description

After sending the start command (Start_OK) to the drive a timer is started. When this timer is over and the trip not yet finished then this message will be generated.

Extra Info

Number of retries during the same trip

Source ID

Not important

Causes and Actions
ID Possible causes of message Actions to be taken
1 Check the safety circuit
2 Check the door contacts
