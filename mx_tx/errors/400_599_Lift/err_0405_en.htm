0405 Permanent Active Call

Summary

 This section describes the Lift error Permanent Active Call.

Description

Permanently active call button. After serving a call the acknowledge lamp or LED of the call button can not be cleared. The call input will be ignored for a period of time (3 min).
Remark: this monitoring function can not be done on interface with separate input and output circuitry.

Extra Info

Floor number of the call button I/O

Source ID

Call type:
01 = Car-call side 1
02 = Car-call side 20
03 = Floor-call side 1
04 = Floor-call side 20

Causes and Actions
ID Possible causes of message Actions to be taken
1 Defective call button or button is pressed continuously Check the call button.
2 Check the wiring of the call button.
