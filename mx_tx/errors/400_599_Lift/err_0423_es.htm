0423 PCT define

Summary

 This section describes the Lift error PCT define.

Description

Software Error "LPCT create". Parameter group missing in PCT. The software expects the following parameter groups to be present in the PCT:


PCT_GP_RS_GENERAL, PCT_GP_RS_TIMEOUT, PCT_GP_RS_COST_CALC, PCT_GP_RS_DOOR_TIME_S1, PCT_GP_RS_DOOR_TIME_S20
PCT_GP_RS_DOOR_MODE, PCT_GP_RS_PARKING_ZONE
PCT_GP_RS_SERV, PCT_GP_RS_SERV_NAME
PCT_GP_RS_PEAK, PCT_GP_RS_SUBGROUP, PCT_GP_LIFT
PCT_GP_SYSTEM, PCT_GP_RC_1, PCT_GP_RC_20
PCT_GP_ANT, PCT_GP_M10_CONTROL, PCT_GP_M10_DEVICES
PCT_GP_VOICE, PCT_GP_BUS_SYSTEM

Extra Info

None.

Source ID

None.
