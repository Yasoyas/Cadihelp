0406 ExceptionMC

Summary

 This section describes the Lift error ExceptionMC.

Description

Software Warning "EXCEPTION of 68000 microprocessor". The system jumps to the exception routine and causes a watchdog reset.

Extra Info

Reason for the exception (MSByte) and the address where the exception occured (Byte3 - Byte1).

Source ID

Task_Id where the exception occured. When '0', no task involved.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Faulty EPROM or pcb. Change EPROMs/FLASH or Lift Control Processor board. If the problem can not be solved contact the field support hotline or the VX-Center.
