0404 Start Monitor Expired

Summary

 This section describes the Lift error Start Monitor Expired.

Description

The control generates the error 'trip start monitor expired' when a trip was
not initiated within the predefined timeout after sending a trip command.
Possible reasons are:

a) The doors refused to close or communication with the door(s) was disrupted.
b) The drive refused to or could not travel or communication with the drive was disrupted
c) The door(s) refused to lock or communication with the door(s) was disrupted.
d) An unknown fault occured during one of the trip preparation phases.

Extra Info

01 = Drive print dead while starting
02 = Drive error while starting
03 = Door print dead. Doors couldn't close
04 = Door error. Doors couldn't close
05 = Unknown error while starting
06 = Door print dead. Doors couldn't lock
07 = Door error. Doors couldn't lock
08 = Unknown error while locking

Source ID

Not relevant

Causes and Actions
ID Possible causes of message Actions to be taken
1 Faulty safety circuit Check the door contacts.
2 Check the safety circuit.
3 Connection to subsystem lost Check the bus connection and test corresponding subsystem to verify its operation and if necessary to exchange it
