0433 Chipcard not valid

Summary

 This section describes the Lift error Chipcard not valid.

Description

Il software ha rilevato una scheda chip non valida (dati della scheda chip non validi).
Commenti: i controllori di tipo MX disattivano l'ascensore per il normale funzionamento una volta rimossa la scheda chip.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Dati della scheda chip non validi o scheda chip guasta Ordinare una nuova scheda chip dal fornitore.
