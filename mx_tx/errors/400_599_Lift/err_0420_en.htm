0420 SCT/PCT-Checksum

Summary

 This section describes the Lift error SCT/PCT-Checksum.

Description

CT CHECKSUMME ERROR. The configuration data of the PCT / SCT are no longer consistent or have been partially destroyed, or the download was not correct. Reliable operation is no longer guaranteed.

Extra Info

Faulty configuration data group
0x1 = PCT input/outputs
0x2 = PCT parameter
0x4 = PCT Download Checksum Error
0xFC = SCT

Source ID

Function which detected the error.

Causes and Actions
ID Possible causes of message Actions to be taken
1 During the checksum test of the configuration data PCT/SCT an illegal change was detected. Examine the EEPROM.
2 Please contact the VX-Center.
3 The download of the configuration data PCT/SCT was not successfull. When download checksum error occurs, examine the connection to the configurator.
