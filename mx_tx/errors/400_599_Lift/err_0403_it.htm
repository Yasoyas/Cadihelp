0403 PowerUp

Summary

 This section describes the Lift error PowerUp.

Description

La CPU sul pcb di controllo principale si reinizializza.
Commenti: verificare se nel registro sono presenti i codici di errore CPU compresi tra 1000 e 1100

Extra Info

Nessuna.

Source ID

00 = Prima accensione[Altre informazioni: (solo per il gestore di errore Power_up) 01 = prima accensione dopo l'installazione di nuovo SW o configurazione 02 = accensione dopo doppia reimpostazione] 01 = accensione per reimpostazione manuale o ragione sconosciuta 02 = accensione per reimpostazione CADI 03 = accensione per preparazione di nuovo download. 04 = accensione provocata da gestore di errore:[Altre informazioni: (solo per il gestore di errore Power_up) 01 = raggiunto il limite di reimpostazione 02 = reimpostazione dopo "Fatal error block system short" 03 = reimpostazione dopo "Fatal error block system long" 04 = reimpostazione dopo "Fatal error block system permanent"] 05 = Watchdog accensione [Altre informazioni: (solo per watchdog di errore Power_up). Corrisponde a una maschera di bit che indica quali attività erano in corso quando si è verificata la reimpostazione. Bit0 = HCBBit1 = HFCBit2 = SPG1 = ok0 = questa attività non ha risposto entro l'intervallo di tempo consentito.]

Causes and Actions
ID Possible causes of message Actions to be taken
1 Accensione dopo problema interno (es. watchdog, gestione d'errore software) Consultare il registro errori per individuare la ragione della reimpostazione
2 Ragione esterna per l'accensione (es. pressione del pulsante di reimpostazione, perdita totale di alimentazione) L'Id origine contiene informazioni sulla ragione dell'accensione
