0405 Permanent Active Call

Summary

 This section describes the Lift error Permanent Active Call.

Description

Pulsante di chiamata permanentemente attivo. Dopo una chiamata, la spia di conferma o il LED del pulsante di chiamata non si spengono. L'input di chiamata verrà ignorato per un periodo di tempo (3 min).

Extra Info

Numero di piano del pulsante di chiamata (I/O)

Source ID

Tipo chiamata:
01 = chiamata cabina
102 = chiamata cabina
2003 = chiamata piano
104 = chiamata piano 20

Causes and Actions
ID Possible causes of message Actions to be taken
1 Pulsante di chiamata difettoso o premuto costantemente Verificare il pulsante di chiamata.
2 Verificare il cablaggio del pulsante di chiamata.
