0490 Table_Overflow

Summary

 This section describes the Lift error Table_Overflow.

Description

Error 490 is no longer used. Please refer to error 903 'Table Overflow'.

The software detects a table overflow in the IO handler or an overflow of the buffer (200 messages) that is provided between the target and the remote monitoring system (typically the error is generated when there is a burst of updates sent to the Servitel system).

Extra Info

The extra info identifies the unique function call that led to the table overflow. It defines the table, the index value and the specific function in the code that caused the overflow.

Source ID

Identifies the source that caused the data overflow. The number identifies the STR task.

23 = SAP_TASK
81 = GIO TASK
82 = IOC TASK
83 = GIOT TASK
84 = IOH TASK
88 = UDS TASK
