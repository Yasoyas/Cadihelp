0426 Wrong SBT Data

Summary

 This section describes the Lift error Wrong SBT Data.

Description

Software Error "invalid service sequence data (SBT)".

This error is generated when the Service Behaviour Table in the SCT does not have a valid value. Normally this should not happen. Indicates a possible inconsistency between target software and SCT (services) version.

Extra Info

For troubleshooting purposes (development).

Source ID

None.
