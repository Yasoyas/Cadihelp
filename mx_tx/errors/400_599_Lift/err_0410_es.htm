0410 PIO-Error

Summary

 This section describes the Lift error PIO-Error.

Description

One of the components, which are connected to the Lift Control, reports a fault.

Extra Info

Points out the false component.
01 = IOBUS detects no periphery
02 = PCB supply below error limit
03 = PCB supply below warning limit
Meaning: The function of the indicated component is disturbed. (f.e. IOBUS cable defect)

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Check the indicated components and if necessary change them
2 Check the wiring of the described component
