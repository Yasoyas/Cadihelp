0504 Battery Dead / Missing

Summary

 This section describes the Lift error Battery Dead / Missing.

Description

The battery test indicates that the battery is either completely discharged, dead, damaged, or disconnected (battery voltage too low).

The battery voltage level is tested once every day and after a manual reset provided normal voltage supply (Group24VDC/Lift24VDC to ASILOG/ASIX) is on.

Extra Info

None

Source ID

None

Causes and Actions
ID Possible causes of message Actions to be taken
1 Battery not connected Check the wiring and connections to the battery
2 Leave the controller powered on, see if the battery is charged eventually
3 Low battery voltage Measure the voltage of the battery.
It should be about 12VDC
4 If the error occurs again, check the battery and replace it if necessary
