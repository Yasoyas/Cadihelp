1503 Defect_MoSis_Sensor

Summary

 This section describes the Travel Control Ext error Defect_MoSis_Sensor.

Description

During the trip, When the MOSIS floor magent sensor is missing or not working , then that floor magnet will not be detected and this error will be logged. Monitoring of floor sensor is done only in auto trips, so this error will be logged only when the auto trip is running.

Extra Info

The position value of missing floor

Source ID

Idnetification number of source file

Causes and Actions
ID Possible causes of message Actions to be taken
1 Sensors made of steel causes problems of detection at low speeds. Change sensor to Aluminium type.
2 Magnet on the floor dropped off. Put the floor magnet back in place.
