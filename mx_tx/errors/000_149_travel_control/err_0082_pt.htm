0082 NoSupply12V

Summary

 This section describes the Travel Control error NoSupply12V.

Description

The emergency power supply is not connected to the Travel Control print.
Comments: The Travel Control PCB has total functionality with only 24V. If a reset is carried out and 12V are not connected, a synchronization trip is started.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Check connection 12V.
2 Check emergency power module.
