0012 DirectionErr

Summary

 This section describes the Travel Control error DirectionErr.

Description

The car moved more than 2.5 cm in the wrong direction.

In case of the Australian Standard (AS1735): The car tried to start a trip in a direction, which is not allowed by the direction limit switches (KSEB-D / KSEB-U).

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Incorrect load input Check the load measurement information.
2 Wrong parameter values Check the parameters on Motor Control.
3 Incorrect tacho polarity Check the polarity of the position tacho.
4 Direction limit switches(AS1735 only) Check the direction limit switches (KSEB-D / KSEB-U), the inputs and wiring to the controller, and the actual position of the car.
5 Variodyn E only: Manual trip on cabin roof in wrong direction in shaft end area Check tacho. Reset Synchronization trip.If necessary perform a learn trip.
