0126 Wrong hardware

Summary

 This section describes the Travel Control error Wrong hardware.

Description

L'hardware installato non coincide con quello che il PCT richiede.

Extra Info

1= PCT specifica A17.1-2000 che richiede la scheda relé ASIBNA2.
2= PCT specifica A17.1-2000 che richiede la scheda CRIPNA3.
3= PCT specifica A17.1-2000 che richiede la scheda CRIPEV3.

Source ID

None.
