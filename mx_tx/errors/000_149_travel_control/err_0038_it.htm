0038 TargetLost

Summary

 This section describes the Travel Control error TargetLost.

Description

The target (destination) floor received is outside the range known by Travel Control. Therefore no direction decision was taken.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Incorrect parameters Check the parameter shaft_para.max_floor.
2 Incorrect selector translation table Check also the selector translation table.
3 Problems in the controller Establish why the controller wants to send the drive to go to a none existent floor
