0070 SW_Warning

Summary

 This section describes the Travel Control error SW_Warning.

Description

The software detected an abnormal situation which is not considered to be severe for the operation of the elevator. Warning messages are also used to provide additional information related to other messages logged. For further details refer to comments for the extra info and the source identifier.

Extra Info

1 or 2: Flash write error
3: Flash read error
4: Message is sent together with error 94 'door lock fault' (door open with KTS closed)
5: Message is sent together with error 94 'door lock fault' (door open with KTC closed)
