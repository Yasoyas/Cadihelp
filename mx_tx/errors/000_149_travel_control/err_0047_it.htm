0047 ErrReleveling

Summary

 This section describes the Travel Control error ErrReleveling.

Description

Durante una corsa di rilivellamento, la cabina si sposta al di fuori della tolleranza di rilivellamento di 75 mm

Extra Info

Nessuna.

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Verificare il controllo motore.
2 Il dispositivo di misurazione del carico invia un valore di carico errato Verificare le informazioni di carico del dispositivo di misurazione del carico. Tarare nuovamente il dispositivo.
3 Oscillazione verticale della cabina durante il caricamento o lo scaricamento della cabina stessa Provare a incrementare i valori dei parametri J_RELEVEL, A_RELEVEL, V_RELEVEL (parametri di movimento).
