0097 ETSL_DeviceError

Summary

 This section describes the Travel Control error ETSL_DeviceError.

Description

Condizione di funzionamento non sicura rilevata dal dispositivo ETSL (errore interno).
Commenti: questo evento viene generato in base ai segnali di stato (ingressi paralleli) provenienti dall'ETSL. Un contatto separato del dispositivo ETSL viene utilizzato per interrompere il circuito di sicurezza (a prescindere dai segnali di stato).

Extra Info

Nessuna.

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Errore interno di PHVKSE Disinserire e reinserire l'alimentazione sul PHVKSE, ripetere l'ultima corsa e verificare il sistema ETSL. Se viene di nuovo visualizzato lo stesso messaggio di errore continua a essere visualizzato, sostituire il PHVKSE.
2 Se l'errore si verifica solo saltuariamente, controllare l'ambiente elettrico per verificare che non vi siano problemi di EMC (forti interferenze elettromagnetiche non consentite provenienti da altri componenti difettosi come contattori ecc.)
