0127 MoSis_Error

Summary

 This section describes the Travel Control error MoSis_Error.

Description

Error on MoSis. Problem with configuration of/for MoSis detected.

Extra Info

0x00000000: Software detects a CAN interface board (EBCAN) but MoSis is not supported/configured by the software (based on deployment).

0x00000001: Software detects mismatch between MoSis and control (number of learnt floors). Occurs after replacing MoSis device or after exchange of main controller board (GCIOB).

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Wrong hardware configuration (extra info: 0) Check hardware (interface boards)
2 Mismatch (extra info: 1) Do a learn/measuring trip.
