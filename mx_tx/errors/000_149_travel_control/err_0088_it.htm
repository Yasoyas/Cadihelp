0088 TachoFault

Summary

 This section describes the Travel Control error TachoFault.

Description

Valore di posizione ricevuto dal rilevatore di posizione saltato, oppure nessun impulso del rilevatore rilevato durante la corsa.
Commenti: problema grave con il rilevatore di posizione

Extra Info

Nessuna.

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Collegamento errato del rilevatore di posizione. Verificare il collegamento del rilevatore di posizione al controllo.
2 Rilevatore di posizione danneggiato. Verificare il rilevatore di posizione.
3 Problemi meccanici Verificare se il rilevatore funziona correttamente.
