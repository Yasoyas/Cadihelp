0058 SH_NoStop

Summary

 This section describes the Travel Control error SH_NoStop.

Description

Monitoring of auxiliary contacts of SH and SH1 ( resp. RH and RH1 ) detected that contactor(s) did not drop off after a trip within the predefined time.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Bad wiring Check the auxiliary contacts of SH, SH1, RH, RH1
2 Incorrect parameter in Travel Control Check the value of parameter 'SH_SHUTDOWN'
3 Faulty inputs on interface board (pcb) Replace interface board
