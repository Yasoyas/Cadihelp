0011 FC_Block

Summary

 This section describes the Travel Control error FC_Block.

Description

Motor Control state is 'blocked'.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Refer to Motor Control related diagnostics
2 Variodyn E: BARMAG error red LED on ER113 Reset frequency converter. If not successful, check internal error-log of frequency converter with a terminal emulation.
Short Manual:
1. To start press 's' / 2. Press 'f' and read error (s. below) /
3. To quit, press F10 /
4. Reset frequency converter again.
3 Internal frequency converter error messages: Possible Causes
4 U_ZK < UZK_MIN Power net voltage too low. Missing net phase.
5 U_ZK > UZK_MIN Chopper resistor not connected, wrong specificied (Ohm) or defective. Improper parameter setting.
6 I > I_MAX Defective freq. converter VF20/30. Short-circuit or ground-short on engine wiring. Improper parameter setting.
7 TIBR > 5 S Interface board does not reset start signal. Check brake function.
8 NET > OFF Malfunction of freq. converter VF20/30.
9 KEINE IMPULSE Check tacho power supply (Pin 1 and 2 on IG100-connector). Defective VF20/30 control board. Slippage of tacho rope. Defective tacho device.
10 ZEIT V K-FAHRT Instruction correction trip has not been reset.
11 VBET < 1.15 x VKN/2 Value VKN set too high. DELTA_U too low. Slippage of tacho rope. Wiring defects. Defective freq. Convert. Mechanical defect on elevator system.
12 VBET > 1.15 x VKN Value VKN set too low. FC_MAX_R set too high. MM_IMP set too high. Mechanical defect on elevator system.
13 TEMP > T_MAX Permanent exceeding of nominal current. Cooling blowers not connected or defective. Air stream restrained. Wrong direction of rotation.
