0057 MCR_NoStop

Summary

 This section describes the Travel Control error MCR_NoStop.

Description

Drive (Motor Control) does not send message FC_IDLE or ERR_NON_FATAL after trip is completed which means the motor control itself did not finish the trip. ( internal event name: FIN_NO_FC_IDLE )

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Check the serial link to Motor Control (pcb).
2 Check the serial interface board on Motor Control.
3 Incorrect parameter in Travel Control Check the value of parameter 'FC_SHUTDOWN'
