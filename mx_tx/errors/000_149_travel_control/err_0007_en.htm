0007 SH_Fault

Summary

 This section describes the Travel Control error SH_Fault.

Description

The auxiliary contacts of SH / SH1 (or RH / RH1) opened while the car was moving.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Safety circuit open Check the safety circuit
2 Wrong parameter value If the error occurs when the car is approaching the target floor it may be caused by the door pre-opening circuit. Reduce value for parameter MAX_DOOR_ZONE.
3 Bad wiring Check the auxiliary contact wiring of SH, SH1 (RH, RH1)
