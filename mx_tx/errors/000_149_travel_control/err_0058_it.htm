0058 SH_NoStop

Summary

 This section describes the Travel Control error SH_NoStop.

Description

Il monitoraggio dei contatti ausiliari di SH e SH1 (o RH e RH1 ) ha rilevato che i contattori non si sono disattivati entro il periodo di tempo predefinito dopo una corsa.

Extra Info

Nessuna.

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Errore di cablaggio Verificare i contatti ausiliari di SH, SH1, RH, RH1
2 Parametro errato nel controllo corsa Verificare il valore del parametro 'SH_SHUTDOWN'
