0010 FC_Fault

Summary

 This section describes the Travel Control error FC_Fault.

Description

a) The frequency converter requested an emergency stop.
b) The control detected a communication problem to the frequency converter.

Comments: The Motor Control monitors the contactor SH. When the the safety circuit opens during a trip Motor Control may detect this before Travel Control. So this event can also be caused by a safety circuit problem. Please refer also to the error description for error 'SH_Fault'.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Drive internal problem Check the event log of Motor Control for detailed information.
2 Communication to frequency converter (temporarily) interrupted Check the wiring and plugs of the connection cable between the FC and the control.
