0055 NoMCR_Com

Summary

 This section describes the Travel Control error NoMCR_Com.

Description

Comunicazione con il PCB del controllo motore non stabilita o interrotta.

Extra Info

Nessuna.

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Nessun collegamento tra controllo corsa e controllo motore Verificare se il cavo è collegato al pcb sul controllo motore (inverter) e sul controllo corsa.
2 Verificare se il pcb del controllo motore è danneggiato.
3 Provare ad accedere al controllo motore con CADI.
4 In questo modo verrà verificata la connessione hardware.
