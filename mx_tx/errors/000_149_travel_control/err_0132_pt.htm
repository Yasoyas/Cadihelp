0132 MoSis_ETSL_Fault

Summary

 This section describes the Travel Control error MoSis_ETSL_Fault.

Description

Message from MoSis together with status change of ETSL relay.

Extra Info

The extra information is a 8 digit hexadecimal code with the following meaning per digit: STUVWXYZ

S=0: ETSL relay closed (normal)
S=1: ETSL relay open (reason for opening see other digits)
T=_: RESERVED FOR FUTURE USE
U=1: NTSD overspeed.
V=1: MoSis internal hardware or software error relating to the ETSL. Check the error-log (MoSis) for further info.
W=1: Relay test failure (EtslRelayFail)
X=1: Commissioned speed exceeded (MaxSpeed).
Y=1: Pre-commissioned limit of 1m/s exceeded (1mps)
Z=1: Exceeded ETS/ETSL curve towards terminal (Curve)

Source ID

None.
