0017 OverSpeedKSE

Summary

 This section describes the Travel Control error OverSpeedKSE.

Description

Overspeed detected while car is inside the upper or lower KSE position (KSE-U / KSE-D).

Extra Info

0: Overspeed detected with car in KSE zone.

10: For MX only (no KSE): Learn trip aborted because learn speed (V_LEARN) to high. Trip curve does not allow to stop the car normally within a distance equal to: FIN_LENGTH / 2.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Incorrect KSE input Check KSE parameters and position of magnets. Repeat the learn trip.
2 Variodyn E: Improper parameter settings (frequency converter) Check frequency converter parameter values Check KSE distance
