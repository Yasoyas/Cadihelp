0084 DMS_Spoiled

Summary

 This section describes the Travel Control error DMS_Spoiled.

Description

Strain gauge (DMS) is not working properly.
Software name of error: DMS_SPOILED

Extra Info

Identification of the load measuring system:
10(hex): First load measuring device (deck one)
20(hex): Second load measuring device (deck two)

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Bad connection of strain gauge to load measuring box Check the connection of the strain gauge devices to the load measuring box.
2 Faulty strain gauge device Replace strain gauge device (use strain gauge test tool for troubleshooting)
