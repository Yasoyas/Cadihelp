0083 NoSupply24V

Summary

 This section describes the Travel Control error NoSupply24V.

Description

No supply 24V connected to the Travel Control print (Variodyn E: PGO)
Comments: This event will appear every time the main switch JH is deactivated and Travel Control keeps running under 12V. The Travel Control runs in a very reduced mode, maintaining only the car position.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Check fuse 24V supply.
2 Check 24V connection.
3 Variodyn E:
4 Missing or defective 24V supply Check 24V supply
5 Missing WD (resistor-diode) device on the brake (electro-magnetical interferences) Install appropriate device properly
6 Net power failure
