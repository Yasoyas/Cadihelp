0138 Brake Not Set

Summary

 This section describes the Travel Control error Brake Not Set.

Description

ndef

Extra Info

ndef

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Brake does not close due to mechanical obstruction Check brake for correct function (sufficient mechanical torque applied to machine)
2 Problem with wiring of KB contacts to control Check wiring and operation of KB contacts
3 Faulty input circuit for KB contacts (both !) Check input status of KB contacts. Replace pcb if necessary
