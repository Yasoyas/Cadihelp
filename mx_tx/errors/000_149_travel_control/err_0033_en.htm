0033 RelvNewDest

Summary

 This section describes the Travel Control error RelvNewDest.

Description

While executing a re-levelling trip, a new destination is received. Note: A re-levelling trip should only take place if the load in the car changes so, that the ropes length change by more than 3mm.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Check if the drive repeats the re-levelling trip frequently. If this is the case check the shaft_parameter in Travel Control, specially the tacho_factor. If any of these values is changed, repeat the learn_trip.
2 A car becomes much lighter while at rest. None. A re-levelling trip is then normal. In that case this event does not mean abnormal operation.
