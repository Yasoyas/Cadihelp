0035 FC_Timeout

Summary

 This section describes the Travel Control error FC_Timeout.

Description

When preparing a new trip, Travel Control requests the status of the inverter (Motor Control). If the Motor Control does not respond as expected (e.g. within the predefined time), Travel control aborts the preparation of the trip. The preparation of a trip by travel control is divided into the following states (phase):

INIT_START_DRIVE
WAIT_CONTACTORS
WAIT_SH_CLOSED
ASK_FC_STATUS
ACTIVATE_FC
WAIT_VECSYS_SH_CLOSED (not used with DC drive)
START_DRIVE_END

Variodyn E only: VF20/30 interrupted starting trial that had been already begun.

Extra Info

Value indicates the cause of the trip abortion and the phase of the trip preparation:

10(hex0A): 'Preparation False' message from inverter in phase 'WAIT_CONTACTORS'
20(hex14): 'Preparation Wrong Trip Type' message from inverter in phase 'WAIT_CONTACTORS'
01(hex01): 'Preparation False' message from inverter in phase 'ASK_FC_STATUS'
21(hex15): 'Preparation Wrong Trip Type' message from inverter in phase 'ASK_FC_STATUS'
02(hex02): Timeout occurred in phase 'ASK_FC_STATUS' because no response from inverter
03(hex03): Timeout occurred in phase 'ACTIVATE_FC' because no response from inverter

Variodyn E: Distance covered away from start [mm].

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Motor Control not ready for new trip Check Motor Control.Check connection to Motor Control.
2 Incorrect parameter on Travel Control Adjust the time that Travel Control waits for Motor Control by the parameter START_UP_PARA.FC_DELAY
3 Variodyn E: Inaccurate parameter setting (frequency converter) Check (and adjust if necessary) frequency converter parameter.
