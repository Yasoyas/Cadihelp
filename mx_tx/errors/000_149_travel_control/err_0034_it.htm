0034 SH_Timeout

Summary

 This section describes the Travel Control error SH_Timeout.

Description

Durante la preparazione della corsa, il contatto ausiliario del contattore SH o SH1 (risp. RH o RH1 se montato) ha mancato di chiudere. La preparazione della corsa è interrotta.

Extra Info

Nessuna.

Source ID

Néant.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Difetto nel circuito di comando del contattore principale Controllare il cablaggio del contattore principale incluso il circuito di sicurezza e l'attivazione del contattore dal comando.
2 Contattore difettoso Controllare il contattore (contatto ausiliario) e il cablaggio verso il pcb.
3 Problema di timing Regolare il tempo di sorveglianza per il contattore principale regolando il parametro 'SH_DELAY' (parametro interno del comando)
