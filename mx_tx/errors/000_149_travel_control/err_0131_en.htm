0131 MoSis_KNE_Fault

Summary

 This section describes the Travel Control error MoSis_KNE_Fault.

Description

Message sent by MoSis together with status change of KNE relay.

Extra Info

The extra information is a 8 digit code with the following meaning per digit:xSTUVWXYZ

S=0: KNE relay closed (normal)
S=1: KNE relay open (reason for opening see other digits)
T=_: RESERVED FOR FUTURE USE
U=1: KNE open because of detection of unintended car movement
V=1: MoSis nternal hardware or software error relating to the KNE. Check the error-log (MoSis) for further info.
W=1: Relay test failure (KneRelayFail)
X=1: Car travelled past the end of the known shaft range (StrangePlace). MoSis was never at this position before (past the last floor).
Y=1: Travel past last known floor (Overtravel)
Z=1: Pre-commissioned state; no floors discovered (Precom)

Source ID

None.
