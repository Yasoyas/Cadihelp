0014 UnderSpeed

Summary

 This section describes the Travel Control error UnderSpeed.

Description

If the difference between the actual speed of the car calculated from the position encoder and the reference speed (velocity_ref) exceeds the calculated limit (speed tolerance) the trip is interrupted with an emergency stop. The allowed speed difference (tolerance) is calculated as follows: 6.25% of the reference speed with a minimum of 0.1m/s. The allowed speed difference is 0.1m/s up until a speed of 1.6m/s.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Incorrect parameter values Check the parameters both on Travel Control and Motor Control.
2 Incorrect tacho input from position tacho Check position tacho
3 Incorrect load input Check the load measurement device and the load parameters. These checks should also be made if the event occurs just at the beginning of the trip.
4 Speed reference value exceeds capabilites of the inverter; the inverter is not able to follow the trip curve. Check speed and acceleration parameters.
5 Car or counterweight machanically blocked Make sure car and counterweight can move in guide rails.
6 Rope slip on tracton sheave Check correct balancing of car/counterweight.
