0077 LM_PrintsErr

Summary

 This section describes the Travel Control error LM_PrintsErr.

Description

Unexpected behaviour of the load measurement device. See extra information for possible error causes.
Software name of error: LMS_DEVICE_FAULT

Extra Info

10(hex): First load measuring device (deck one) missing/faulty
20(hex): Second load measuring device (deck two) missing/faulty

64(hex): Load limit exceeded (20% of nominal load) for automatic zero load calibration of the (serial) load measuring device (performed at night).

Monitoring of load contacts. When a load contact is active the software checks if the contacts for the lower load limits are active too. If not an error message is sent.

Monitoring of load contacts for NA:
214(hex): KL-M is inactive with KL-X active
224(hex): KL-V is inactive with KL-X active
234(hex): KL-H is inactive with KL-X active
212(hex): KL-M is inactive with KL-V active
213(hex): KL-M is inactive with KL-H active
223(hex): KL-H is inactive with KL-V active

Monitoring of load contacts for EUAP:
114(hex): KL-M is inactive with KL-X active
124(hex): KL-H is inactive with KL-X active
134(hex): KL-V is inactive with KL-X active
112(hex): KL-M is inactive with KL-H active
113(hex): KL-M is inactive with KL-V active
123(hex): KL-H is inactive with KL-V active

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Check the LMS device and the bus connection to the control cabinet.
2 Incorrect calibration of load measuring system. Check (repeat) calibration of load measuring system.
3 With load contacts: Wrong status of load contact inputs. (e.g. KL-V active but KL-M inactive) Check proper function of load contacts KL-M / KL-H /KL-V / KL-MAX / KL-X.
4 Incorrect baud rate Check that the baud rate of the lift bus is the same on both Travel Control and LMS pcb
