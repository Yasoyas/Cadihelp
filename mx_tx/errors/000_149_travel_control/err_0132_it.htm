0132 MoSis_ETSL_Fault

Summary

 This section describes the Travel Control error MoSis_ETSL_Fault.

Description

Messaggio da MoSis insieme a cambio di stato di relè ETSL.

Extra Info

L'informazione extra è un codice a 8 caratteri con il seguente significato:xSTUVWXYZ

S=0: Relè ETSL chiuso (normale)
S=1: Relè ETSL aperto (per il motivo di apertura vedere gli altri caratteri)
T=_: RESERVED FOR FUTURE USE
U=1: sovravelocità NTSD.
V=1: Errore interno di hardware o software MoSis relativo al ETSL. Controllare registro errori (MoSis) per ulteriori informazioni.
W=1: Fallimento test relè (EtslRelayFail)
X=1: Superata velocità prestabilita (MaxSpeed).
Y=1: Superato limite prestabilito di 1m/s (1mps)
Z=1: Superata curva ETS/ETSL verso terminale (Curve)

Source ID

None.
