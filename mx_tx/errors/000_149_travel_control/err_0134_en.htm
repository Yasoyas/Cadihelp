0134 NTSD_Triggered

Summary

 This section describes the Travel Control error NTSD_Triggered.

Description

This error is generated when the NTSD (normal terminal slowdown) monitoring detects that the car speed is above the speed limit calculated from the car position and the limiting deceleration rate defined with the parameters A_NTSD [where parameter 'ETSL_TYPE' is set to 'MoSis_ETSL_NTSD']. When the NTSD is triggered 2 times in 1000 trips, a warning is sent. When NTSD is triggered 3 times in 1000 trips, an ERROR is sent. When it is triggered more than 3 times in 1000 trips, the system is blocked permanently. The total error count keeps accumulating until reset. The error type reported is not dependant on time, but on the number of trips that the elevator does.

None

Source ID

None

Causes and Actions
ID Possible causes of message Actions to be taken
