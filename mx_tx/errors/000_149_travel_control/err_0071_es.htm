0071 SW Error

Summary

 This section describes the Travel Control error SW Error.

Description

A severe error in the software was detected. The message indicates a major problem in the software. For further details refer to comments for the extra info. Normally no such error should occur in the field with an ofiicially released software.

Extra Info

Provides information (in addition to the source ID) about the error trap in the software which generated the error message:

14E(hex)/334(dec): Error in trip plan calculation
155(hex)/341(dec): Unknown value for CTP message found (calculate trip plan)
1B0(hex)/432(dec): Undefined trip type found (TP.current_trip)
200(hex)/512(dec): Undefined value for velocity state (speed calculation)
21A(hex)/538(dec): Unknown Command to motor control (inverter)
23F(hex)/575(dec): Unknown application message from motor control (inverter)
25B(hex)/603(dec): Unknown value in trip preparation 'start drive state'
323(hex)/803(dec): Unknown configuration message from motor control (inverter)
3B8(hex)/952(dec): Undefined value for deceleration state found (speed calc.)
4FC(hex)/1276(dec): Wrong application command for motor control (inverter)
55F(hex)/1375(dec): Wrong configuration command for motor control (inverter)

Source ID

Identifies the file in the core software (travel control) from where the message is generated. Refer to the list in the description for source id. for error number 70.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Contact R&D department in Ebikon.
2 Please report all the values logged in the event log as well as the Travel Control Local log entries. These are clues for the software specialist to detect the source of the problem.
