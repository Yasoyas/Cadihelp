0008 SB_Fault

Summary

 This section describes the Travel Control error SB_Fault.

Description

The SB auxiliary contact opened when the car was moving or an error on brake module occurred (e.g. MBB)

Extra Info

with MBB:
01 (hex)/ 1 (dec): (MBB_TASK_CREATE_ERROR) = State message
02 (hex)/ 2 (dec): (MBB_BUFFER_OVERFLOW_ERROR) = IO message
03 (hex)/ 3 (dec): (MBB_INT_WRONG_MSG) = Unknown message received
04 (hex)/ 4 (dec): (MBB_ZERO_MESSAGE) = Zero length message sent
05 (hex)/ 5 (dec): (MBB_UNKNOWN_COMMAND) = Unknown command received
06 (hex)/ 6 (dec): (MBB_KB_FAULT) = KB fault
07 (hex)/ 7 (dec): (MBB_KB1_FAULT) = KB1 fault
08 (hex)/ 8 (dec): (MBB_OPEN_TIMEOUT_CURRENT) = Open timeout , no current
09 (hex)/ 9 (dec): (MBB_CLOSE_TIMEOUT_CURRENT) = Close timeout, no 0 current
0A (hex)/10 (dec): (MBB_IGBT3_FAULT) = IGBT fault or other error
0B (hex)/11 (dec): (MBB_HW_FAULT) = Error on MBB HW
0C (hex)/12 (dec): (MBB_OPEN_STANDSTILL) = Brake open during stanstill
0D (hex)/13 (dec): (MBB_CLOSED_RUNNING) = Brake closed during trip
0E (hex)/14 (dec): (MBB_MBX_ERROR) = MBB mailbox error
0F (hex)/15 (dec): (MBB_MAX_MESSAGE_SIZE) = MBB too long message size
10 (hex)/16 (dec): (MBB_CURRENT_DIFF_ERROR) = MBB current difference error
11 (hex)/17 (dec): (MBB_MAINS_NOT_OFF) = Mains did not switch off
12 (hex)/18 (dec): (MBB_IGBT1_FAULT) = IGBT1 check not O.K.
13 (hex)/19 (dec): (MBB_VOLTAGE_DIFF_ERROR) = MBB voltage difference error
Brake Module MX:
14 (hex)/20 (dec): (MBB_KBR_FAULT) = RB fault (for MX-GC)
15 (hex)/21 (dec): (MBB_ZSB_FAULT) = Brake time relay fault (for MX-GC)

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Open safety circuit Check the safety circuit
2 Auxiliary contacts of SB/SB1 not working Check operation of auxiliary contacts of SB/SB1.
3 Check wiring of auxiliary contacts of SB/SB1
4 Relay RB not working Check operation of relay RB
5 Dust in the control cabinet or other difficult environmental conditions causing conductivity problems. Remove dust
