0022 MaskErased

Summary

 This section describes the Travel Control error MaskErased.

Description

The car is performing a trip in a trip type for which the control deletes the related trip type mask. This is a normal operation event. The trip under execution will be finished, but no new trip of the same type can be triggered until the trip type mask is set again.

Extra Info

None.

Source ID

None.
