0012 DirectionErr

Summary

 This section describes the Travel Control error DirectionErr.

Description

La cabina si muove di oltre 2,5 cm nella direzione sbagliata.
Nel caso dello Standard Australiano (AS1735): la cabina ha tentato di avviare una corsa in una direzione non consentita da uno o entrambi i finecorsa di direzione (KSEB-D / KSEB-U).

Extra Info

Nessuna.

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Input di carico non corretto Verificare le informazioni di misurazione del carico.
2 Valori di parametro errati Verificare i parametri sul controllo motore.
3 Polarit errata del rilevatore Verificare la polarità del rilevatore di posizione.
4 Finecorsa di direzione (solo AS1735) Verificare i finecorsa di direzione (KSEB-D / KSEB-U), gli ingressi e il cablaggio del controllore e la posizione effettiva della cabina.
5 Solo Variodyn E: corsa manuale sul tetto della cabina nella direzione sbagliata nell'area dell'estremit del pozzo Verificare il rilevatore. Reimpostare la corsa di sincronizzazione. Se necessario, eseguire una corsa di apprendimento.
