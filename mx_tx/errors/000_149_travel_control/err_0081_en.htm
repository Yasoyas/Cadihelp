0081 TripV_Tacho

Summary

 This section describes the Travel Control error TripV_Tacho.

Description

The car position (distance) tacho failed during the trip. Travel Control continues the trip to the nearest floor using the speed tacho feedback from the inverter/motor control.
Comments: The speed tacho information is provided by Motor Control as a handshake to each speed reference telegram (closed loop). This information is used by Travel Control to calculate position and speed of the car and to continue the current trip with the speed tacho only. The car stops at the nearest floor; normal stopping precision cannot be guaranteed. However the car will stop close enough to the floor so that the door(s) can open normally for passengers to leave the car.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Problem with the signal from the position tacho Check the position tacho as well as its connection to the interface PCB and the PCB hardware.
2 Bad/broken wire between position tacho and control cabinet Check wiring
3 Faulty position tacho Replace position tacho
4 EMC problems Check for EMC interferences caused by Schindler or 3rd party equipment
