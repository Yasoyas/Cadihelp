0040 NoPHS_Learn

Summary

 This section describes the Travel Control error NoPHS_Learn.

Description

For IG type position tacho:
At the beginning of the learn trip (after the preparation trip) the car should be within the fin of PHS Photocell (IN_FIN). If this is not the case the preparation for the learn_trip is aborted.
Comments: If the PHS connection to Travel Control is missing, the internal status is PHS=IN_FIN. Therefore, without PHS connected, the learn trip will start normally. The trip will abort when passing the first KSE magnet.
For AGSI type position tacho:
Trip will be aborted if learn trip is triggered outside door zone magnet (KUET).

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Incorrect shaft information With IG type (incremental) position tacho:
2 Check the KSE signal (distance, polarity of magnet, wiring).
3 Check wiring of PHS.
4 Check ground connection of photocell PHS.
5 With AGSI (absolute encoder) type position tacho:
6 Check the KUET signal (polarity of magnet, wiring).
