0096 ETSL_VaneError

Summary

 This section describes the Travel Control error ETSL_VaneError.

Description

Condizione di funzionamento non sicura rilevata dal dispositivo ETSL durante la scansione delle linguette.
Commenti: questo evento viene generato in base ai segnali di stato (ingressi paralleli) provenienti dall'ETSL. Un contatto separato del dispositivo ETSL viene utilizzato per interrompere il circuito di sicurezza (a prescindere dai segnali di stato).

Extra Info

Nessuna.

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Allineamento errato tra PHVKSE e linguette/banda metallica. Verificare l'allineamento.
2 Banda metallica deformata o contorta nel vano di corsa Verificare la tensione della banda metallica.
3 Spazio nella via di controllo senza linguetta di misura di fronte. Coprire lo spazio con qualcosa che interrompa la luce IR. Un solo nastro di plastica è probabilmente insufficiente.
4 Nessuna apertura dietro una linguetta di misura oppure apertura parzialmente coperta. Verificare le aperture.
5 Sporco sui componenti ottici. Pulire le lenti nello spazio del PHVKSE con un panno pulito.
