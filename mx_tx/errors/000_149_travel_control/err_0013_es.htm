0013 OverSpeed

Summary

 This section describes the Travel Control error OverSpeed.

Description

If the difference between the actual speed of the car calculated from the position encoder and the reference speed (velocity_ref) exceeds the calculated limit (speed tolerance) the trip is interrupted with an emergency stop. The allowed speed difference (tolerance) is calculated as follows: 6.25% of the actual reference speed with a minimum of 0.1m/s. The allowed speed difference is 0.1m/s up until a speed of 1.6m/s.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Incorrect parameter values Check the parameters both on Travel Control and Motor Control.
2 Incorrect tacho input from position tacho Check position tacho.
3 Incorrect load value from load measuring system causing wrong torque on machine Check the load measurement device and the load parameters. These checks should also be made if the event occurs just at the beginning of the trip.
4 Car moves faster than the detection speed limit while the drive/motor is switched off. May be caused by loss of traction at standstill or manual opening of the brake. Check if machine brake applies sufficient mechanical torque. Perform brake test according to acceptance test instruction.
