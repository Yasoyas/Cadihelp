0130 PCT_Drive_Para

Summary

 This section describes the Travel Control error PCT_Drive_Para.

Description

The value of a PCT Drive Parameter does not match with the current hardware or
is not allow the according to the Deploment Rules. Refer to the Extra Info for identification of the Drive Parameter.

Extra Info

Parameter Id. (hexadecimal) of parameter with wrong value.
for example:

0h4B : P_ANT CODE_TYPE
0h77 : P_ANT_SAFETY_TYPE

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 A PCT has been downloaded with a wrong parameter value, or Download PCT with correct value or modifiy parameter to the correct value with CADI or SMLCD.
2 The Parameter has been changed to an invalid value. Modifiy parameter to the correct value with CADI or SMLCD.
