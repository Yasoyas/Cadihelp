0131 MoSis_KNE_Fault

Summary

 This section describes the Travel Control error MoSis_KNE_Fault.

Description

Messaggio inviato da MoSis insieme a cambio di stato di relè KNE.

Extra Info

L'informazione extra è un codice a 8 caratteri con il seguente significato:xSTUVWXYZ

S=0: relè KNE chiuso (normale)
S=1: relè KNE aperto (per il motivo di apertura vedere gli altri caratteri)
T=_: RESERVED FOR FUTURE USE
U=1: KNE aperto per rilevamento di movimento involontario della cabina
V=1: Errore interno di hardware o software MoSis relativo al KNE. Controllare protocollo errori (MoSis) per ulteriori informazioni.
W=1: Fallimento test relè (KneRelayFail)
X=1: Cabina passata oltre la fine del vano conosciuto (StrangePlace). MoSis non è mai stato in questa posizione in precedenza (oltre l'ultimo piano).
Y=1: Escursione oltre ultimo piano conosciuto (sovracorsa)
Z=1: Stato pre-commissionamento; nessun piano individuato (Precom)

Source ID

None.
