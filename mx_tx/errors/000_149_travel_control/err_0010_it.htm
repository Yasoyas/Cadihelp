0010 FC_Fault

Summary

 This section describes the Travel Control error FC_Fault.

Description

Il convertitore di frequenza ha richiesto un arresto di emergenza. Commenti: Il Controllo motore sorveglia il contattore SH. Se il circuito di sicurezza apre durante una corsa, il Controllo motore può riconoscerlo prima del Controllo corsa. Pertanto questo evento può essere causato anche da un problema del circuito di sicurezza. Consultare anche la descrizione errori per SH_Fault.

Extra Info

Nessuna.

Source ID

None.
