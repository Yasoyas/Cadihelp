0127 MoSis_Error

Summary

 This section describes the Travel Control error MoSis_Error.

Description

Errore su MoSis. Rilevato problema con configurazione di/per MoSis.

Extra Info

0x00000000: Software rileva una scheda interfaccia CAN (EBCAN) ma MoSis non è supportato/configurato dal software (a seconda della dotazione).

0x00000001: Software rileva discrepanza tra MoSis e controllo (numero di piani appresi). Si verifica dopo la sostituzione del dispositivo MoSis o dopo sostituzione della scheda principale controller (GCIOB).

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Errata configurazione hardware (informazione extra: 0) Controllare hardware (schede interfaccia)
2 Discrepanza (informazione extra: 1) Effettuare una corsa di apprendimento/misurazione.
