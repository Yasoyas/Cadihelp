0007 SH_Fault

Summary

 This section describes the Travel Control error SH_Fault.

Description

I contatti ausiliari di SH / SH1 (o RH / RH1) si sono aperti mentre la cabina era in movimento.

Extra Info

Nessuna.

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Circuito di sicurezza aperto Verificare il circuito di sicurezza
2 Valore di parametro errato Se questo errore si verifica quando la cabina si avvicina al piano di destinazione, la causa potrebbe risiedere nel circuito di pre-apertura della porta. Ridurre il valore del parametro MAX_DOOR_ZONE.
3 Errore di cablaggio Verificare il cablaggio dei contatti ausiliari di SH, SH1 (RH, RH1)
