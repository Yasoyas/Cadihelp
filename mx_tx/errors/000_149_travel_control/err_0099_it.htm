0099 RSK_Fault

Summary

 This section describes the Travel Control error RSK_Fault.

Description

Problema con il feedback dei relè RSK / RSK1 (malfunzionamento del relè).

Extra Info

Nessuna.

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Guasto del contatto di feedback dei relè RSK / RSK1 Verificare il cablaggio e/o la funzionalità dei relè RSK / RSK1.
2 RSK/RSK1 non si disattiva come previsto Sostituire i relè RSK/RSK1 o il pcb su cui si trovano i relè.
3 RSK/RSK1 non si attiva durante la preparazione della corsa come previsto Verificare il circuito di sicurezza (ISKT4 &amp; ISKT5).

Field Feedback
ID Location Actions to be taken
1 Controller Controllare : RSK/RSK1; se necessario cambiare il print.
2 Shaft Controllare : circuito di sicurezza ISKT4 e ISKT5
