0053 NoSTR_Com

Summary

 This section describes the Travel Control error NoSTR_Com.

Description

The communication with the controller is broken.
Travel Control is still able to run if the safety circuit is closed. Use CADI to simulate calls on Travel Control. This procedure is useful to find out if whether the drive is healthy and eventually to evacuate passengers.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Check the hardware connections.
2 Check the healthiness of the lift bus members.
