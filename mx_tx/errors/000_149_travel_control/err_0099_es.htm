0099 RSK_Fault

Summary

 This section describes the Travel Control error RSK_Fault.

Description

Problem with feedback of relays RSK / RSK1 (malbehaviour of relay).

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Faulty feedback contact of the relays RSK / RSK1 Check wiring and/or functionality of relays RSK / RSK1.
2 RSK/RSK1 does not drop off as expected Replace relays RSK/RSK1 or pcb which holds the relays.
3 RSK/RSK1 does not turn on during trip preparation as expected Check the safety circuit (ISKT4 &amp; ISKT5).

Field Feedback
ID Location Actions to be taken
1 Controller Comprobar RSK/RSK1, quizás cambiar el print
2 Shaft Comprobar el circuito de seguridad ISKT4 y ISKT5
