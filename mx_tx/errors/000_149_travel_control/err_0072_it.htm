0072 Safety 110V

Summary

 This section describes the Travel Control error Safety 110V.

Description

Il circuito di sicurezza si è aperto prima della prima presa (ISKT1). Problema con l'alimentazione del circuito di sicurezza.
Commenti: questo errore può essere registrato dopo lo scollegamento dell'alimentazione principale. Altri ingressi delle prese del circuito di sicurezza (ISKT2 ecc.) non generano messaggi di errore corrispondenti.

Altre informazioni per l'alimentazione del circuito di sicurezza (xx):
00: nessuna alimentazione del circuito di sicurezza

Per sistemi di comando con pcb SKS vengono fornite le seguenti informazioni aggiuntive:
01: la tensione del circuito di sicurezza sui relè è troppo elevata. Nel caso di un guasto HW (malfunzionamento della regolazione della tensione), l'alimentazione del circuito di sicurezza viene disinserita se la tensione sui relè è troppo elevata. Il livello di tensione è compreso tra 27,0 e 28,0 volt.
02: la tensione del circuito di sicurezza sulla sorgente è troppo bassa. Nel caso di un guasto HW (malfunzionamento della regolazione della tensione), l'alimentazione del circuito di sicurezza viene disinserita se la tensione di uscita è troppo bassa. Il livello di tensione è compreso tra 19,0 e 20,0 volt.
03: la tensione di uscita del circuito di sicurezza è troppo elevata. Nel caso di un guasto HW (malfunzionamento della regolazione della tensione), l'alimentazione del circuito di sicurezza viene disattivata se la tensione di uscita è troppo elevata. Il livello di tensione è compreso tra 60,1 e 60,7 volt.
04: sovracorrente del circuito di sicurezza. Nel caso di un cortocircuito verso GND o di corrente di dispersione verso GND, l'alimentazione del circuito di sicurezza viene disinserita. Il limite di sovracorrente è compreso tra 300 e 500mA.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Circuito di sicurezza non alimentato Verificare l'alimentazione del circuito di sicurezza.
2 In caso di alimentazione di tensione regolata: reimpostare la protezione da cortocircuiti per l'alimentazione del circuito di sicurezza.
3 Relè RKPH Verificare il relè di controllo fase RKPH (se disponibile).
4 Ingresso guasto sulla scheda di interfaccia per gli ingressi delle prese del circuito di sicurezza Verificare la scheda di interfaccia e i collegamenti al circuito di sicurezza. Se necessario, sostituire la scheda.
