0125 EV Event

Summary

 This section describes the Travel Control error EV Event.

Description

Information (event) reported by the Electronic Valve control. (Hydro only)

Extra Info

1F5 Hex (501 Dec): Dynahyd C8 = Command to stop at this floor arrived too late (up trip). If this happens often, notify Gettysburg.

1F6 Hex (502 Dec): Dynahyd C9 = Command to stop at this floor arrived too late (down trip). If this happens often, notify Gettysburg.

1F7 Hex (503 Dec): Dynahyd T1 = Communication software timeout. Replace Dynahyd software chip. Replace CRIPEV board.

Source ID

None.
