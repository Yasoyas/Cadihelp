0005 SH_on_STDSTL

Summary

 This section describes the Travel Control error SH_on_STDSTL.

Description

The contactor SH or SH1 is detected energised during standstill phase.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Check the auxiliary contacts of SH, SH1 (RH and RH1 if fitted)
2 SH or SH1 are operated by hand None.
