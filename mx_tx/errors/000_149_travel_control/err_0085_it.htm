0085 LM_ComErr

Summary

 This section describes the Travel Control error LM_ComErr.

Description

Comunicazione interrotta con la scatola LMS.
Commenti: TX4: con CADI in controllo corsa, sotto il menu EVENT LOGS accedere al sottomenu VCOM ERR CTRS per vedere le prestazioni del collegamento del bus dell'ascensore al controllo corsa.

Extra Info

Nessuna.

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Verificare il dispositivo LMS e il collegamento del bus all'armadietto di controllo
2 Velocità di trasmissione errata Solo TX4: verificare la regolazione della velocità di trasmissione sui pcb.
