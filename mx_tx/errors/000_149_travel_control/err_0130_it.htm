0130 PCT_Drive_Para

Summary

 This section describes the Travel Control error PCT_Drive_Para.

Description

Il valore di un Parametro di comando PCT non coincide con l'hardware attuale o non è consentito dalle norme di applicazione. Consultare le Informazioni Extra per identificare il Parametro di Comando.

Extra Info

Riconoscimento (esadecimale) del parametro con valore errato.
per esempio:

0h4B : P_ANT CODE_TYPE
0h77 : P_ANT_SAFETY_TYPE

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Un PCT è stato scaricato con un valore di parametro errato, o Scaricare PCT con valore corretto o modificare il parametro al valore corretto con CADI o SMLCD.
2 Il parametro è stato modificato in un valore non valido. Modificare il parametro al valore corretto con CADI o SMLCD.
