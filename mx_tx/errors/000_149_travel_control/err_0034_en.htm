0034 SH_Timeout

Summary

 This section describes the Travel Control error SH_Timeout.

Description

During trip preparation the auxiliary contact of the contactor SH or SH1 ( resp. RH or RH1 if fitted) failed to close. Trip preparation is aborted.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Defect in control circuit of main contactor Check the wiring of the main contactor including safety circuit and triggering of contactor from drive.
2 Faulty contactor Check the contacor (auxiliary contact) and the wiring to the pcb.
3 Timing problem Adjust the monitoring time for the main contactor by adjusting the parameter 'SH_DELAY' (drive internal parameter)
