0055 NoMCR_Com

Summary

 This section describes the Travel Control error NoMCR_Com.

Description

Communication with Motor Control PCB not established or lost.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 No link between Travel Control and Motor Control Check if cable is connected to pcb on Motor Control (inverter) and Travel Control.
2 Check if pcb of Motor Control is damaged.
3 Try to access the Motor Control with CADI.
4 This will check the hardware connection.
