0057 MCR_NoStop

Summary

 This section describes the Travel Control error MCR_NoStop.

Description

Il controllo motore non invia il messaggio FC_IDLE o ERR_NON_FATAL dopo il completamento della corsa; ciò significa che il controllo motore non ha terminato la corsa (nome evento interno: FIN_NO_FC_IDLE).

Extra Info

Nessuna.

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Verificare il collegamento seriale al controllo motore (pcb).
2 Verificare la scheda di interfaccia seriale sul controllo motore.
3 Parametro errato nel controllo corsa Verificare il valore del parametro 'FC_SHUTDOWN'
