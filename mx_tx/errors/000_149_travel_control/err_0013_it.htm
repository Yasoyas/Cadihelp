0013 OverSpeed

Summary

 This section describes the Travel Control error OverSpeed.

Description

La velocit della cabina supera il valore di riferimento effettivo (il limite di tolleranza normale 6,25% )

Extra Info

Nessuna.

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Valori di parametro errati Verificare i parametri sul controllo corsa e sul controllo motore.
2 Ingresso non corretto dal rilevatore di posizione Verificare il rilevatore di posizione.
3 Ingresso di carico non corretto Verificare il dispositivo di misurazione del carico e i parametri di carico.
4 Queste verifiche devono essere eseguite anche quando l'evento si verifica all'inizio della corsa.
