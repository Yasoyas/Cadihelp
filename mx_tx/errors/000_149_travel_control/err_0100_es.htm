0100 RFE_Fault

Summary

 This section describes the Travel Control error RFE_Fault.

Description

Problem with the feedback contact of the relay RFE (malbehaviour of relay).
Comments: This error is only generated if the trip initiation circuit is with relay RFE.
The relay RFE is monitored during the trip preparation. Relay RFE has to drop after relays RSK & RSK1 have picked up.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 RFE is turned on and does not pick up within 5 seconds (Timeout) Check the wiring and the functionality of RFE.
2 Check all contacts/contactors in the RFE circuit.
3 RFE does not turn off Check RSPFE.
4 TX4: Wrong parameter 'System Type' Check the Travel Control - Elevator parameter 'System Type' (TX4 only).
