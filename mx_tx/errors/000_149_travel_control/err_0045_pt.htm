0045 NoFloorTop

Summary

 This section describes the Travel Control error NoFloorTop.

Description

When the car passes the upper KSE (KSE-U) during learn trip (up-direction), travel control expects to reach the uppermost floor after the distance defined with parameter 'kse_distance'. The car then slows down accordingly. If PHS is not IN_FIN at the end of the trip, the learn procedure is aborted.

Extra Info

None.

Source ID

None.
