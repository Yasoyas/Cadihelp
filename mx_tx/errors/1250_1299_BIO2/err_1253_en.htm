1253 BIO2_ISR_Communication_Error

Summary

 This section describes the BIO2 error BIO2_ISR_Communication_Error.

Description

An error has occurred in the communication between the BIO2 driver and the EBBIO.

Extra Info

0xCC??IIOO
CC = Error code / II = Input queue usage / OO = Output queue usage

Meaning of error code (CC):
08 = EBBIO uplink buffer full
09 = Short circuit on the bus

Source ID

Task Id.
