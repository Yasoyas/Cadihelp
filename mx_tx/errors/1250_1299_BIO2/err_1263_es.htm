1263 BIO2 Missing Node

Summary

 This section describes the BIO2 error BIO2 Missing Node.

Description

A node is missing on the BIO2 bus. There is a BIO2 node that has been recognised and does not response any more.
A BIO2 node does not response when it is scanned (node without critical IO) or it does not send the heartbeat signal any more (node with critical IO). It is possible that the missing node is defect, so it has to be exchanged by a new one. It also is possible that the missing node has been reteached. That means the node is actually working correct but the host can not receive any answers from the old address of the node. The host can not tell wheter a new node is installed or if an old one has been re-teached. In this case you have to make a freeze.

Extra Info

0x0000yyxx

xx = access side (01 = front side/02 = rear side)
yy = floor number

Source ID

Task Id.

Causes and Actions
ID Possible causes of message Actions to be taken
1 No communication to BIO2 board Check BIO2 node board on related floor
2 Loss of supply on BIO2 board Check BIO2 node board on related floor
3 Re-teach of BIO2 node Execute freeze command after verification of function of node.
4 Faulty hardware (BIO2 board) Replace boar.
