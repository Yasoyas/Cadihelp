1261 BIO2 Requestor Failure

Summary

 This section describes the BIO2 error BIO2 Requestor Failure.

Description

Error occurred in BIO2-requestor. The requestor table of the BIO2-BUS is full.

Extra Info

0x0010xxxx: requestor table full (queue id = xxxx)
0x0020xxxx: no request found (queue id = xxxx)

Source ID

160: BIO2_DISP_TASK_ID
