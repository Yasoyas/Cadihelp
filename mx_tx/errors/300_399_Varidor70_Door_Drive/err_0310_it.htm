0310 HW_DRV_SERIOUS

Summary

 This section describes the Varidor70 Door Drive error HW_DRV_SERIOUS.

Description

[V70 Door Controller SW]
Drive error from frequency converter.

Error response: Stop door motions now, and stop all further door motions
Easy Setup: Bargraph-RED-LED-4 [ON]

Extra Info

The error is a signal from the drive that there is some serious problem with the Drive Motor or the Drive Frequency Convertor(Inverter)

Source ID

This error is generated from the Drive

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. Motor or frequency converter error 1. Check the door drive system, motor and mechancial parts
2. Reset the Door Controller
2 2. Overheating/overloading of the door drive 1. Refer to Cause/Action list of DRIVE_OVER_TEMP
