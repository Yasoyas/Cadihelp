0300 CTR_PAR_ACCESS

Summary

 This section describes the Varidor70 Door Drive error CTR_PAR_ACCESS.

Description

[V70 Door Controller SW]
Door Controller could not access a Parameter.

Error response: Stop all door motions immediatly, and stop all further motions too.

Easy Setup: BarGraph-RED-LED-7 [ON]

Extra Info

The most common parameters which might cause this error are
a. Mass
b. Control Amplification
c. Drive Delay

Source ID

This error is generated from the PID Control SW Module
(PID Control = Proportional Integral Differential Control)

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. A parameter is not defined or incorrect 1. Check the validity of parameter definitions described
2. Try Downloading a fresh PCT with correct values
2 2. A Parameter access failed 1. Check the validity of parameter definitions described
2. Try Downloading a fresh PCT with correct values
3. Try Resetting Door Controller
3 3. Flash is gone bad 1. If the error "PAR_NOVRAM_MISSING" occurs too, change the Door Controller PCB and boot the door again
