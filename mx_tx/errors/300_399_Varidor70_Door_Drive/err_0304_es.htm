0304 TRC_CALC

Summary

 This section describes the Varidor70 Door Drive error TRC_CALC.

Description

[V70 Door Controller SW]
Error in Trip Curve Generator/Converter calculation

Error response: Stop the Door Motions Immediatly, and dont do any more motion
Easy Setup: Bargraph-RED-LED-7 [ON]

Extra Info

This error will occur when there is a problem with the calculation of trip curves. Since, its would be dangerous to do any motions without a proper trip curve, the door stops, unless it is reset.

Source ID

This error is generated from the Trip Curve Generator

Causes and Actions
ID Possible causes of message Actions to be taken
1 Trip calculation fails Reboot the door
