0322 ESK_PARAM_WRITE

Summary

 This section describes the Varidor70 Door Drive error ESK_PARAM_WRITE.

Description

[V70 Door Controller SW]
ESK could not write to a parameter. This is not an Error, but a Warning.

Error response: None
Easy Setup: Bargraph-RED-LED-8 [ON]

Extra Info

This error/warning is raised when certain parameters when modified via ESK and being set back to the parameters. When the parameters cannot be written, this error is raised

Source ID

This error is generated from the ESK SW Module

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. Flash is gone bad 1. Reset the Door Controller
2. Replace the Door Controller PCB
2 1. ESK is gone bad or ESK Cables are gone bad 1. Reset the Door Controller
2. Check the cables of the ESK
3. Replace the ESK HW
4. Consider shielding the ESK Cables
