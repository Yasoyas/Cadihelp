0321 ESK_INTERNAL

Summary

 This section describes the Varidor70 Door Drive error ESK_INTERNAL.

Description

[V70 Door Controller SW]
Error in the ESK controller.
This is not an error, instead its a Warning

Error response: None
Easy Setup: Bargraph-RED-LED-8 [ON]

Extra Info

This error is generated if the ESK is not in any of the pre-defined state. This is an internal SW error.

Source ID

This error is generated from ESK State Machine

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. Internal ESK SW error 1. Reset the Door Controller
2 2. ESK HW fault or loosely fitted 1. Disconnect and reconnect the ESK cables
2. Reset the Door Controller
3. Replace the ESK HW
3 3. ESK Cables short circuit 1. ESK's flat cable could be getting short somewhere with the Ebox body, check for this
2. Shield the flat cable for short circuit.
