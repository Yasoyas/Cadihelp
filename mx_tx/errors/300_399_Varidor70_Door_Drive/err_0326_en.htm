0326 VLON_TRANSMIT

Summary

 This section describes the Varidor70 Door Drive error VLON_TRANSMIT.

Description

[V70 Door Controller SW]
LON Module could not send a LON Message, this is an internal SW error.

Error response: Wait till the doors are open and then stop all door motions
Easy Setup: Bargraph-RED-LED-2 [ON] & LCI-Conn-RED-LED [ON]

Extra Info

This error could be raised because of memory problems with LONV70 and the OS

Source ID

This error is generated from the LON SW Module

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. Memory Problems wiht OS 1. Reset the Door Controller
2 2. LON Firmware is not correct 1. Check the validity of the version of the LON firmware on the LONV70 via CADI-GC
2. Update the LON Firmware
3 3. LONV70 Card is gone bad 1. Replace the LONV70 card
