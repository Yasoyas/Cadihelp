0301 CTR_CALC

Summary

 This section describes the Varidor70 Door Drive error CTR_CALC.

Description

[V70 Door Controller SW]
An attempt was made to set an unknown Control Mode
This is an internal SW Error.

Error response: Stop the door immediately, no more door motions allowed.

Easy Setup: Bargraph-RED-LED-7 [ON]

Extra Info

Presently there are 3 control modes
a. Position Control
b. Speed Control
c. Force Control

Source ID

This error is generated from the PID Control SW Module
(PID Control = Proportional Integral Differential Control)

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. An Attempt to set an unknown mode was done 2. Reset the door controller
