0328 EMERGENCY

Summary

 This section describes the Varidor70 Door Drive error EMERGENCY.

Description

[V70 Door Controller SW]
Drive is not in normal operation anymore and after 2 secs Door Contoller will reboot.

Error Response: Stop all door motions and stop all further door motions
Easy Setup: Bargraph-RED-LED-1 [ON]

Extra Info

This error is generated when the door is requested to do an Emergency Off. This can happen from several locations
a. Control task: when the HW_POS Error is generated
b. Safedev: when the door is in blocked state

Source ID

This error is generated from Control Task

Causes and Actions
ID Possible causes of message Actions to be taken
1 1. Obstacle in the door way 1. Remove the obstacle in the door way
2. Reset the Doot Controller
2 2. Refer to cause and action of Error HW_POS 1. Refer to cause and action of Error HW_POS
