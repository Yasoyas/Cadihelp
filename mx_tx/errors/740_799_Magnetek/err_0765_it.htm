0765 High temperature

Summary

 This section describes the Magnetek error High temperature.

Description

Magnetek error 908
The temperature inside the cabinet is too high.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Check machine room ambient temperature
2 Check that the fan on the drive is running
3 Check that the air flow into and out of the cabinet is not blocked.
