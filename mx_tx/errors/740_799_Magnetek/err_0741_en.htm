0741 Tach Loss

Summary

 This section describes the Magnetek error Tach Loss.

Description

Magnetek error 98
The armature voltage is high enough that the motor should be spinning, but the encoder is not generating pulses.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 The motor encoder installation may be broken Check for broken wires or a loose coupling.
2 The motor encoder may be broken Check if motor moves
3 Replace Encoder
