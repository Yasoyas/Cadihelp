0760 Line synch

Summary

 This section describes the Magnetek error Line synch.

Description

Magnetek error 903
The drive monitors its 3-phase power input to determine whento turn on the SCRs. If it does not detect 3-phase AC power it cannot operate.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Check for loose connections on input mainline wiring
2 Check for dips on mainline voltage.
3 Check voltage divider on Armature Interface board
