1228 NiNoInAppBuf

Summary

 This section describes the LON error NiNoInAppBuf.

Description

Generated when the Application Buffers in the Network Interface ( MIP on ILTP78, EBLON1 or EBLON2) overflow due to too many messages on the LON network. This condition is detected on the Neuron Chip.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Defective board Replace SIO or EBLON board.
2 Software failure Save error logs and report to supplier hotline or R&amp;D
3 Unauthorized equipment connected to LON bus Remove unauthorized equipment.
