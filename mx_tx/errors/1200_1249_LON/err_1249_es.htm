1249 NodePowerFail

Summary

 This section describes the LON error NodePowerFail.

Description

Generated when the voltage (normally 24VDC) to a LON node is too low (limit -20%)

Extra Info

Node type and location ID encoded as a 32-bit value. Please refer to description for error number 1209 for encoding information of extra info value.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Low voltage on LON node board. Check voltage on node.
