1231 PmipIqOverflow

Summary

 This section describes the LON error PmipIqOverflow.

Description

Generated when the MIP software on the Neuron Chip has received several messages that need to be transferred to the main processor, but the buffers in the LON driver software are completely full and overflowing. This error may indicate a serious communication problem.

Extra Info

IQ-size

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Defective board Replace SIO or EBLON board.
2 Software failure Save error logs and report to supplier hotline.
3 Unauthorized equipment connected to LON bus Remove unauthorized equipment.
