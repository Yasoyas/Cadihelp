1206 MysteryReset

Summary

 This section describes the LON error MysteryReset.

Description

Generated when the SIOL or SIOG is reset for an unknown or unlisted reason. (SIOL/SIOG ONLY)
TX5 reports CPU errors 1000 ... 1100 instead.

Extra Info

Reset cause ID

Source ID

SIOL or SIOG

Causes and Actions
ID Possible causes of message Actions to be taken
1 Software failure Save error logs and report to supplier hotline or R&amp;D
2 Defective SIO board Replace SIO board.
