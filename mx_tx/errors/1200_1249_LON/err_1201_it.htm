1201 PowerUpReset

Summary

 This section describes the LON error PowerUpReset.

Description

Generated when SIOL or SIOG powers up.
(SIOL/SIOG ONLY) TX5 reports either CPU errors 1000 ... 1100 or 403 instead.

Extra Info

None.

Source ID

SIOL or SIOG

Causes and Actions
ID Possible causes of message Actions to be taken
1 The main disconnect was turned ON No action required
2 The building had a power failure No action required
3 JNGG or JNGL tripped and was reset No action required
4 Loose power connection to the SIO Check for loose wiring
