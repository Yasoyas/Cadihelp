1233 SybsystemDead

Summary

 This section describes the LON error SybsystemDead.

Description

VCOM on SIOL/SIOG detected another print on group-bus or lift-bus stopped communicating
(SIOL/SIOG ONLY) TX5 reports error 402 instead.

Extra Info

Bits 15..8: cause=02=VCOM_PRINT_DEAD
Bits 7..4: Lift ID of subsystem
Bits 3..0: Logical address of subsystem
Example: 00000211 = RS Lift 1
Example: 00000212 = Travel Control Lift 1
Example: 00000221 = RS Lift 2

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Noise on group- or lift-bus Check bus terminations, grounding, cable shields. Make sure connectors are plugged in firmly.
2 Defective SIO board Replace SIO board.
3 Other defective board on group- or lift-bus Replace/remove other boards on lift bus or group bus to isolate problem.
4 Software failure Save error logs and report to supplier hotline.
