1224 CksErrGP_Konf

Summary

 This section describes the LON error CksErrGP_Konf.

Description

The SIO requests PCT configuration checksum information from the RS after every reset. This error is generated by the SIO when the checksum response from the RS indicates an error condition.

Extra Info

Error code response message from RS

Source ID

SIOL or SIOG

Causes and Actions
ID Possible causes of message Actions to be taken
1 Incorrect or corrupted PCT Download a correct configuration file.
2 Error on RS Save error logs and call EBI/RAND
