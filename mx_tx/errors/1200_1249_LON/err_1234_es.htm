1234 VcomUpcall

Summary

 This section describes the LON error VcomUpcall.

Description

VCOM on SIOL/SIOG detected an abnormal condition. This is usually catastrophic, and reliable operation can not be guaranteed.

Extra Info

Bits 15..8: cause=
03=VCOM_QUEUE_FULL, paramater is Queue-ID
04=VCOM_ADMIN_FULL
05=VCOM_UPDATES_FULL
06=VCOM_COMMS_Q_FULL
07=VCOM_BUSSED_STRESSED, paramater is stress-ID
08=VCOM_LOC_TABLE_FULL
09=VCOM_ADMIN_CONFLICT
Bits 7..0: paramater from VCOM.H
Example: 000003CB = VCOM_QUEUE_FULL, paramater=CB hex

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Noise on group- or lift-bus Check bus terminations, grounding, cable shields. Make sure connectors are plugged in firmly.
2 Defective SIO board Replace SIO board.
3 Other defective board on group- or lift-bus Replace/remove other boards on lift bus or group bus to isolate problem.
4 Software failure Save error logs and report to supplier hotline.
