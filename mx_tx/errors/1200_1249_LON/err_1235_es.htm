1235 KonfWipe

Summary

 This section describes the LON error KonfWipe.

Description

Generated when SIOL or SIOG erases its configuration and then reloads the PCT from RS. This is normal following a software upgrade, when swapping a SIOL with another lift, or after a double reset.

Extra Info

Reason for konf-wipe. See below (One or more may be set)
00000001 Master checksum
00000020 Domain ID
00000300 Subnet/node
00004000 Lift-ID changed
00050000 SW-Version changed

Source ID

None.
