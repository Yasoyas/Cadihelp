1217 DiffCksIO_Konfl

Summary

 This section describes the LON error DiffCksIO_Konfl.

Description

The SIO requests PCT configuration checksum information from the RS after every reset. This event is generated by the SIO when one of the I/O mapping configuration checksums on RS has changed changed and the configuration on the SIO is no longer correct.
The SIO will now download a new I/O mapping configuration.

Extra Info

New checksum value (16 bits)

Source ID

SIOL or SIOG

Causes and Actions
ID Possible causes of message Actions to be taken
1 First power-up None
2 A new PCT was downloaded to RS None
3 Download Configuration command None
4 Software on SIO replaced None
