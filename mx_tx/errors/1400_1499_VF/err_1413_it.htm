1413 KB_Fault

Summary

 This section describes the VF error KB_Fault.

Description

KB contacter error
The brake contacts KB or KB1 do not switch properly.
Software expects both KB and KB1 information

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Defective brake contacts Check contact adjustment
2 Malfunction of mechanical brake Check the brake
3 Observe the brake during opening - both sides should open symmetrically
