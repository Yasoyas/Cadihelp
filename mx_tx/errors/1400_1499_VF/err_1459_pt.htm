1459 Inv_BMG_Param

Summary

 This section describes the VF error Inv_BMG_Param.

Description

Parameter values stored in the frequency converter (BARMAG) do not correspond with their equivalents in the control software (inconsistence)

Extra Info

0x01: 0x0A: BARMAG parameter ID
0x01: FN
0x02: VKN
0x03: MM_IMP
0x04: S_KBR_2
0x05: FT_RU1
0x06: DELTA_U
0x07: FC_MAX_R
0x08: T_DIFF
0x09: T_AUF
0x0A: U_END

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Communication PGO - BARMAG not successful/impossible Check RS232 connection PGO - BARMAG
2 Communication PGO - BARMAG had been interrupted Perform double-reset on PGO and reset on BARMAG, wait approx. 5 minutes (!)
