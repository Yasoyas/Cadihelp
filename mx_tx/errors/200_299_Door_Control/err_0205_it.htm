0205 DoorContrErr

Summary

 This section describes the Door Control error DoorContrErr.

Description

Errore di controllo porta. Lo stato di una delle uscite a 24VDC della scheda di interfaccia della cabina non corrisponde allo stato previsto, oppure il livello di alimentazione della scheda di interfaccia è sceso al di sotto di un determinato punto.

Extra Info

01: Livello di alimentazione della scheda della cabina troppo basso.
02: Uscita VRVRT (velocità porta bassa)
03: Uscita VRET (ispezione porta)
04: Uscita RVEC (ventilatore cabina)
05: Uscita RLC (illuminazione cabina)
06: Uscita VRMGH (freno di stazionamento)
07: Uscita VKETSO (barriera ottica)
08: Uscita VENABLE (dispositivo di ponte porta)
09: Uscita VKSPEPO (radar lobby)

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Uscita difettosa o cortocircuito Verificare il cablaggio dell'uscita.
2 Sostituire la scheda di interfaccia sulla cabina.
