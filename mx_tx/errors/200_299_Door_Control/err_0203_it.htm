0203 ThermoDoorMot

Summary

 This section describes the Door Control error ThermoDoorMot.

Description

Il contatto termico KTHMT del motore della porta è attivo. Il movimento in corso della porta verrà completato e il freno magnetico rilasciato, tutte le uscite del comando della porta saranno disattivate e il controllo porta entrerà nello stato 'bloccato'. La cabina potrà ancora essere mossa manualmente.

Extra Info

Nessuna.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Porta bloccata / coppia eccessiva Verificare il comando della porta e che la porta possa muoversi senza ostacoli
