0209 DoorContact

Summary

 This section describes the Door Control error DoorContact.

Description

Per le porte manuali o semiautomatiche, la porta di piano manuale non si chiude a seguito di un comando di chiusura o blocco proveniente dal controllore.

Extra Info

Nessuna.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Dispositivo RTS guasto Esaminare il dispositivo e sostituirlo, se necessario.
2 Verificare l'uscita VSTS
3 Verificare il contatto del chiavistello e il cablaggio verso la scheda I/O
4 La porta di piano è stata lasciata aperta Nessuna (chiudere la porta)
