0204 DoorRevDevErr

Summary

 This section describes the Door Control error DoorRevDevErr.

Description

Errore del dispositivo di inversione della porta. L'errore viene generato dal controllo porta nelle seguenti situazioni:
a) lo stesso dispositivo di inversione è stato permanentemente attivo per due volte durante 5 cicli completi di apertura e chiusura.
b) un dispositivo di inversione è permanentemente attivo, pertanto anche dopo il timeout finale del timer la porta non può chiudersi.

Extra Info

01: DT-O permanentemente attivo.
02: RPHT permanentemente attivo.
03: Radar permanentemente attivo (REP).
04. Dispositivo di inversione di riserva permanentemente attivo.
05: Il limitatore di forza di chiusura KSKB è permanentemente attivo.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Dispositivo di inversione danneggiato o non regolato correttamente Verificare la funzionalità del dispositivo di inversione
2 Collegamento errato o ingresso guasto sulla scheda di interfaccia della cabina (es. PIOC/ICE/TIC/LONIBV/LONIC ecc.) Verificare il collegamento dalla scheda di interfaccia al dispositivo di inversione. Sostituire la scheda di interfaccia.
