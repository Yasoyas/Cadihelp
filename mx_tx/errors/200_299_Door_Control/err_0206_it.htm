0206 DoorBoltErr

Summary

 This section describes the Door Control error DoorBoltErr.

Description

Problema con il chiavistello. Il contatto di blocco della porta (KV/RKV) non si chiude a seguito di un comando di blocco della porta, oppure non si apre quando la porta viene sbloccata (per porte manuali o semiautomatiche).

Extra Info

Nessuna.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Contatto di blocco non funzionante Verificare il meccanismo e il contatto di blocco.
2 Verificare il cablaggio del dispositivo di blocco (motore / magnete).
