0207 CloseSeqErr

Summary

 This section describes the Door Control error CloseSeqErr.

Description

Errore di sequenza di chiusura della porta. L'errore viene generato dal controllo porta nelle seguenti situazioni:
a) un dispositivo di inversione è attivato durante x tentativi ripetuti di chiudere la porta sullo stesso piano e se ciò si è verificato due volte in cinque corse.
b) lo stesso dispositivo di inversione si attiva troppo frequentemente (8 volte = avviso, 12 volte = errore) durante la chiusura della porta.

Extra Info

Significato del valore:
01: DT-O innescato ripetutamente.
02: RPHT innescato ripetutamente.
03: Radar innescato ripetutamente.
04:Dispositivo di inversione di riserva permanentemente attivo.
05: Il limitatore di forza di chiusura KSKB viene attivato ripetutamente.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Dispositivo di inversione danneggiato o non regolato correttamente Verificare il dispositivo di inversione.
2 Collegamento errato o ingresso guasto sulla scheda di interfaccia della cabina (es. PIOC/ICE/TIC/LONIBV/LONIC ecc.) Verificare il collegamento dalla scheda di interfaccia al dispositivo di inversione. Sostituire la scheda di interfaccia.
