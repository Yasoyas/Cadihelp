0208 DoorDevErr

Summary

 This section describes the Door Control error DoorDevErr.

Description

Door device error. Door control malfunction. See extra info for details.

Extra Info

Identifies the causing component:

01: input from light barrier (PROGARD L etc.).
02: I/O-Bus error.
03: JHCT pressed.
04: Door motor power failed.
05: Door node communication failed.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Faulty device Examine the device and replace it if necessary.
2 Extra Info = 2 Check I/O-bus cable.
3 Check I/O-bus drivers.
4 Extra Info = 3 Check JHCT.
5 Check wiring of JHCT.
6 Extra Info = 4 Check the power supply on the door driver pcb (fuse).
7 Extra Info = 5 Check LON bus to door interface board.
8 Check LON node and its rotary switch settings.
9 Check LON bus to other nodes. (remove for test)
10 Check pcb EBLON.
