0210 Lift Guardian

Summary

 This section describes the Door Control error Lift Guardian.

Description

Door control detects faulty DT-O button or car light not on during a trip.

Extra Info

1 = the car light is off or flickers
- Warning after 3 trips with car light flickering
- Error after 3 trips with the car light permanently off

2 = defective DT-O
- Warning after 200 consecutive door cycles without DT-O being pressed
- Error after 400 consecutive door cycles without DT-O being pressed

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Car light is defective Check the lamp, light network, wiring, relays and car control outputs.
2 Sensor for car light defective or covered.
3 DT-O input signal is not working Check the button, the wiring, IO bus cable and the IO interface board. ( message is also logged when elevator is running from call simulator )
