0725 Overtemp KTHBR

Summary

 This section describes the MC PVEC error Overtemp KTHBR.

Description

Overtemperature on the brake chopper resistor (detected with PT100 / KTJ84 / KTHBR).

Extra Info

1: Overtemperature of BR (detected with klixon KTHBR)
2: No temperature signal from sensor BR / open
3: Shortcut of sensor BR
4: No feedback from FAN (converters with VARIOcon / VIP only)

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Fan faulty Check fan
2 Cable from sensor (klixon) broken or not connected Check cable
3 Temperature of machine room too high
4 Wrong dimensioning of drive system Verify drive calculation
