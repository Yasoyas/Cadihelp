0705 Speed Difference

Summary

 This section describes the MC PVEC error Speed Difference.

Description

The difference between the speed reference value from travel control and the actual speed feedback derived from the motor encoder exceeds the value defined by the parameter OVER_SPEED_FACT. The parameter OVER_SPEED_FACT relates to nominal speed.
Further error cause: In spite of a high torque no actual speed can be measured (speed control error).

Extra Info

1: underspeed
2: overspeed
3: speed control error
4: wrong sense of rotation detected during test 'sense of rotation' (direction test for asynchronous motors)
5: emergency stop during adjust tacholess control test

Source ID

None.

Causes and Actions

- the break does not open
