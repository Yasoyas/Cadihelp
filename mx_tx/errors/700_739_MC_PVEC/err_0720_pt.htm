0720 DCL Undervoltage

Summary

 This section describes the MC PVEC error DCL Undervoltage.

Description

Undervoltage in DC link.
The DC link voltage is measured by an AD-converter. The limit is determined by the supervisor parameter Udcl Low Lim Idle (see parameter description) during stand still and Udcl Low Lim Ride during the trip.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Undervoltage in mains Check mains
2 Parameter wrong Check the parameters Udcl Low Lim Idle and Udcl Low Lim Ride
3 Defective diode bridge Replace power unit
4 Line converter at the input of DC link (VF120 - VF280):malfunction of line converter Check line converter
