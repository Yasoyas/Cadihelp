0704 Encoder Failure

Summary

 This section describes the MC PVEC error Encoder Failure.

Description

Segnale di feedback dall'encoder del motore assente. Il segnale di errore dell'encoder viene generato sulla relativa scheda di controllo (es. PIOVECxx) e letto dal software.

Extra Info

1: La velocità rilevata è zero e il valore di riferimento è diverso da zero
2: L'ingresso del segnale dell'encoder sul convertitore di frequenza non funziona
3: Transazione dati non riuscita (encoder assoluto con interfaccia seriale)

Source ID

Nessuna.
