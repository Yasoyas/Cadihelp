0726 Overtemp MH

Summary

 This section describes the MC PVEC error Overtemp MH.

Description

Temperatura eccessiva del motore di sollevamento (rilevata con PT100 / KTJ84).

Extra Info

1: Temperatura eccessiva di MH
2: Nessun segnale di temperatura da MH / aperto
3: Cortocircuito del sensore MH

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Guasto della ventola MH Verificare la ventola
2 Cavo THMH di MH interrotto o non collegato Verificare il cavo
3 Temperatura del locale macchine troppo elevata
4 Dimensionamento errato del sistema di comando Verificare il calcolo del sistema di comando
