0713 Current Difference

Summary

 This section describes the MC PVEC error Current Difference.

Description

The control deviation of the current controller exceeds the value specified by supervisor parameter IS_DIFF_LIM.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Interrupted transmission of transistor signals Check cables
2 Defective igbts Replace power unit
3 Malfunction or interrupted transmission of current measurement. Check cables
4 If the error occurs at the end of the acceleration phase: faulty dimensioning of drive system If the error occurs at the end of the acceleration phase: reduce the nominal speed
5 Modernization: wrong motor wiring
