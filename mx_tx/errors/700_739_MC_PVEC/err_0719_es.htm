0719 DCL Overvoltage

Summary

 This section describes the MC PVEC error DCL Overvoltage.

Description

Overvoltage in DC link.
The DC link voltage is measured by an AD-converter. The limit is determined by the supervisor parameter Udcl Upp Lim Idle (--> see parameter description). It can only occur during stand still.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Overvoltage in mains Check mains
2 Parameter wrong Check the parameter Udcl Upp Lim Idle
3 Line converter at the input of DC link (VF120 - VF280): malfunction of line converter Check line converter
