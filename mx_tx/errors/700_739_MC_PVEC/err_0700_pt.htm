0700 Memory Failure

Summary

 This section describes the MC PVEC error Memory Failure.

Description

Incorrect memory access.

Extra Info

1: error after reading from or writing to RAM
2: PLD not programmed (only for pcb PIOVEC)

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Defective RAM on pcb PVEC (Extra Info = 1) Replace the pcb PVEC.
2 PLD on PIOVEC not programmed (Extra Info = 2) Replace the pcb PIOVEC or program its PLD
