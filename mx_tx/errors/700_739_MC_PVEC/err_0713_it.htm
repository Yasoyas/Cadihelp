0713 Current Difference

Summary

 This section describes the MC PVEC error Current Difference.

Description

La deviazione di controllo del controllore di corrente supera il valore specificato dal parametro di supervisione IS_DIFF_LIM.

Extra Info

Nessuna.

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Trasmissione interrotta dei segnali del transistor Verificare i cavi
2 Difetto Igbts Sostituire il gruppo elettrogeno
3 Malfunzionamento o trasmissione interrotta della misurazione di corrente. Verificare i cavi
4 Se l'errore si verifica al termine della fase di accelerazione: dimensionamento errato del sistema di comando Se l'errore si verifica al termine della fase di accelerazione: ridurre la velocità nominale
5 Modernizzazione: cablaggio errato del motore
