0701 SW Warning

Summary

 This section describes the MC PVEC error SW Warning.

Description

The software detected an exceptional behavior of either the software or of the hardware. These events are stored in the warning log only and never generate an error. The extra info generally can be decoded by developers of R&D-DR only.

Extra Info

140: Power up, SW-Reset
143: Error state set to dead (permanent blocked)
144: Warninglog deleted
145: Errorlog deleted

Source ID

None.
