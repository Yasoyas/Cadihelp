0717 Overtemp HS Limit

Summary

 This section describes the MC PVEC error Overtemp HS Limit.

Description

Temperature limit of the heatsink or of the brake chopper (if available) exceeded. (See supervisor parameter INV_TEMP_LIM --> parameter description).
A trip must be terminated by the Travel Control on the next floor.

Extra Info

1: overtemperature of heatsink
2: no temperature signal from heatsink
3: overtemperature of brake chopper
4: no temperature signal from brake chopper
11: Overtemperature phase U
12: Overtemperature phase V
13: Overtemperature phase W
21: No temperature signal from heatsink phase U
22: No temperature signal from heatsink phase V
23: No temperature signal from heatsink phase W

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Fan faulty Check fan
2 Cable from heatsink broken or not connected Check cable
3 Temperature of machine room too high
4 Faulty dimensioning of drive system
