0702 Communication Failure

Summary

 This section describes the MC PVEC error Communication Failure.

Description

Error on the serial communication link from the Motor Control to the Travel Control.

Extra Info

1: UART framing error (from the Motor Control to the Travel Control)
2: UART parity error (from the Motor Control to the Travel Control)
3: UART overrun error (from the Motor Control to the Travel Control)
4: overflow TX buffer (from the Motor Control to the Travel Control)
5: unknown source (from the Motor Control to the Travel Control)
6: telegram length to large (from the Motor Control to the Travel Control)
7: overflow RX buffer (from the Motor Control to the Travel Control)
8: overflow RX queue (from the Motor Control to the Travel Control)
9: CAN bus Warning (Parallel Converter)
10: CAN bus off (Parallel Converter)
11: CAN MSG to frame (Parallel Converter)
12: CAN frame to MSG (Parallel Converter)
13: CAN bus occupied (Parallel Converter)
14: No communication to master (Parallel Converter)
15: No communication to slaves (Parallel Converter)
17: No SSI communication (Parallel Converter)
20: CAN bus Warning (DriveBus)
21: CAN bus off (DriveBus)
22: CAN MSG to frame (DriveBus)
23: CAN frame to MSG (DriveBus)
24: CAN bus occupied (DriveBus)
30: I2C Bus not available
31: I2C operation error
32: I2C Bus Busy

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Extra info 1 - 3: EMC problems or communication cable faulty Check whether all screen cables are correctly grounded
2 Check the connection between the Motor Control and the Travel Control.
3 Extra info 4 - 8 / 14 / 15: software problems Report to R&D-DR
