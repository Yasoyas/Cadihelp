0708 SGR/SH Failure

Summary

 This section describes the MC PVEC error SGR/SH Failure.

Description

Il feedback di controllo o ausiliario del contattore SGR/SH è guasto.

Extra Info

1: contatto SGR/SH non ON quando VSGR è ON
2: contatto SGR/SH non OFF quando VSGR è OFF

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Cablaggio del segnale di controllo VSGR o del segnale di feedback SGR/SH interrotto o non collegato Verificare il relativo cablaggio in base agli schemi di cablaggio
2 Contattore SGR/SH guasto Verificare il contattore SGR/SH
3 TX-GC: SGR ha la funzione di SH e questo errore viene anche generato se il circuito di sicurezza si apre durante una corsa o un collaudo Verificare il circuito di sicurezza e il registro completo degli errori
4 Problema noto nel software VecSys. Aggiornare il software VecSys alla versione 2.14 (o successiva)
