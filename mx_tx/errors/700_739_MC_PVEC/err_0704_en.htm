0704 Encoder Failure

Summary

 This section describes the MC PVEC error Encoder Failure.

Description

The feedback signal from the motor encoder is missing. The encoder failure signal is generated on the related control board (e.g. PIOVECxx) and read by software.

Extra Info

1: Measured speed is zero and speed reference is not zero
2: Encoder signal input on frequency converter is faulty
3: Data transaction failed (absolute encoder with serial interface)

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Encoder cable not connected or not correctly plugged in. Check the cable and its connection between the encoder and the corresponding pcb (PIOVECxx).
Check the plug.
2 Motor encoder not correctly mounted Check the mounting of the encoder
3 Defective incremental encoder or encoder cable Replace encoder cable and/or encoder
