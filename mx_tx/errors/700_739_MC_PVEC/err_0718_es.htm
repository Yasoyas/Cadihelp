0718 DCL Overvoltage Trigger

Summary

 This section describes the MC PVEC error DCL Overvoltage Trigger.

Description

Overvoltage in DC link.
The overvoltage trigger signal is generated on the control print (PIOVECxx) and read by software, the voltage level is at about 875 V.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Diode bridge at the input of DC link:malfunction of the brake chopper or brake resistor Check brake resistor
2 Line converter at the input of DC link (VF120 - VF280):malfunction of line converter Check line converter
