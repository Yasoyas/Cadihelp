0712 Power Electronic Failure

Summary

 This section describes the MC PVEC error Power Electronic Failure.

Description

The feedback of the power electronics module(s) report an error.

Extra Info

Meaning of extra info value for VecSys 2.15 / VARIOsys 1.02 and higher:
x00: Any error in power module
x01(Bitmask Bit 0): Error in phase U / VF11BR: overcurrent
x02(Bitmask Bit 1): Error in phase V / VF11BR: supply voltage error
x04(Bitmask Bit 2): Error in phase W
x08(Bitmask Bit 3): Error in brake chopper

Meaning of extra info value for VecSys 2.14 / VARIOsys 1.01 and below (older):
x01: error in phase U
x02: error in phase V
x03: error in phase W
x04: error in brake chopper
x05: any error in power module

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Cable from power unit broken or not connected check cable and connectors
2 Power module faulty replace power unit
3 Fan faulty check that the fan runs during a trip, run fan test, check cables, replace fan
4 Brake resistor/chopper faulty (overvoltage trigger) check brake resistor and cables
5 Temperature in machine room or hoistway too high
6 Wrong dimensioning (calculation) of the drive system
7 For extra info value=0: thyristor not on (VF30BR only)
8 For extra info value=5: overcurrent / short circuit
