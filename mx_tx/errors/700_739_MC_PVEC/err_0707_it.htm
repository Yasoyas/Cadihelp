0707 SGRW Failure

Summary

 This section describes the MC PVEC error SGRW Failure.

Description

Il contatto di feedback ausiliario o di controllo del contattore di carica SGRW è guasto.

Extra Info

1: contatto SGRW non ON quando VSGRW è ON
2: contatto SGRW non OFF quando VSGRW è OFF

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Cablaggio del segnale di controllo VSGRW o del segnale di feedback SGRW interrotto o non collegato Verificare il relativo cablaggio in base agli schemi di cablaggio
2 Contattore SGRW guasto Verificare il contattore SGRW
