0706 Speed Ref Missing

Summary

 This section describes the MC PVEC error Speed Ref Missing.

Description

I telegrammi per i valori di riferimento della velocità inviati dal controllo corsa (controllore MX / TX / PTCM) sono assenti per almeno due periodi consecutivi (20 ms).

Extra Info

Nessuna.

Source ID

Nessuna.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Linea di comunicazione interrotta tra controllo corsa (es. pcb ASILOG/ASIX) e controllo motore (es. pcb PIOVECxx / PVEC) Verificare la comunicazione seriale (cavo e connessione ai pcb)
2 Scheda di interfaccia (pcb) difettosa Sostituire il relativo pcb
