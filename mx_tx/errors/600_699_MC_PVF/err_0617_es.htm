0617 Over speed warning

Summary

 This section describes the MC PVF error Over speed warning.

Description

1.) Car moves more than 20 cm during standstill
2.) Overspeed detected during PVF test trip (DRE_D or DRE_U active). Note: This monitoring function is inactive during normal operation, i.e. if PVF is running together with an interface board (Travel Control).

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 The brake is faulty. Check the brake.
2 Overspeed during trip: Must not occur in normal installations. Check if no test trip was performed.
