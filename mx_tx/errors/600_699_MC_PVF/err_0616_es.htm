0616 Over current warning

Summary

 This section describes the MC PVF error Over current warning.

Description

Over current detected during the constant speed phase of a trip. The max. allowed value = is_nom x over_current_fact.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 The motor consumes to much electrical energy: the motor brake does not open properly; the car is overloaded; wrong counterweight or car balancing. Check machine and mechanical brake.
2 Check if car &amp; counter weight are balanced.
3 Wrong PVF parameters Readjust PVF parameters.
4 Wrong motor tacho polarity. This error could occur during intallation travel, when the speed is very slow (error 18 is not detected, not enough distance in the wrong direction). Check motor tacho polarity (PVF X9),exchange wire X9.2 and X9.3 if necessary
