0602 Overtemperature

Summary

 This section describes the MC PVF error Overtemperature.

Description

Overtemperature check has detected power transistorsoverheated. When the car is at standstill, it will be blocked. If the car is in motion, the trip is completed, as if it remains in motion for over 30 sec, emergency stop is generated.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Overtemperature on the Inverter's heat sink. Check power transistors.
2 Wrong wiring of the input signal KTHVF Check wiring of the input signal KTHVF (X5.3)
3 Variodyn 35, 45, 70, 100/BR: Malfunction of the Brake Chopper, Brake Resistor. Check brake chopper and brake resistor
