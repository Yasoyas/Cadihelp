0650 External error cc1

Summary

 This section describes the MC PVF error External error cc1.

Description

This error occurs, if the input X3.1 (1_READY) on the PCC board is noton high level (high level = higher than 9 V). This input represents the current state of the converter CC1. Checked at trip start and during trip.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Converter CC1 detects an error Check converter CC1 (AEG converter: see error LEDs on PWR board)
2 Safety circuit opens during trip (produces normally 1 warning) Check Travel Control event log, if there are safety circuit problems, compare times
3 Bad input wiring Check wiring connections (PCC X3.1 or PWR X102.6)
4 Note: Parallel operation with AEG converter: Check error LEDs on PWR board Parallel operation with Schindler converter: Check error LEDs on UKE2 board
5 More information about how to check the error LEDs, see under error 614 !
