0662 DL overvolt trigg cc3

Summary

 This section describes the MC PVF error DL overvolt trigg cc3.

Description

DC link overvoltage CC3 detected if > 730 V. Checked during stand still, at trip start and every 10 ms during the tripby software.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Variodyn 280(Schindler): Malfunction of the front end converter (FCR) Check FCR
2 Variodyn 100/140/220 (AEG) has its own control circuits and this monitoring function is redundant. Ignore
3 Interferences or bad voltage measurement Measure voltage between PCC X?.? and X?.?. Should be UDCL/100
