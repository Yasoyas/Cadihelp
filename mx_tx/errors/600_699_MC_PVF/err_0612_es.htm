0612 Input contactor failure

Summary

 This section describes the MC PVF error Input contactor failure.

Description

1.) Input contactor (SGR, is equivalent to K16 in new AEG inverters with onlyone contactor) not responding to switch on / off signal.
2.) Input contactor (SGR, is equivalent to K16 in new AEG inverters) opens during trip.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Input contactor failure (Variodyn 35, 45, 70, 100/BR, 280 only) Check contactor coil (SGR)
2 Input relay failure (Variodyn 100, 140, 220 with one contactor) Check contactor coil (K16)
3 Wrongfeedback: X3.4 and/or X4.10 wrong wiring. Check auxiliary contacts and wiring.
4 If SGR coil is connected on UKE2 board (UKE2 X4.8): UKE2 detects an error and cuts off the SGR supply. Check UKE2 error LEDs.
5 Check signal VSGR (connection PVF X3.4 - UKE2 X4.9)
6 If the contactor SGR does not open after a switch off command, this error isgenerated and the error state is set to fatal, independent of the error filter (for EN81, from PVF SW version 4.20 on).
