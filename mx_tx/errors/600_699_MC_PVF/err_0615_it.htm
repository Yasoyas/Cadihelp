0615 SW encoder warning

Summary

 This section describes the MC PVF error SW encoder warning.

Description

Software detects no movement of motor. During a trip, the speed reference value is higher than the tolerance limit (0.12 m/s) and current speed calculated from the rotor angle increments is equal to zero.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Themotor is blocked, mechanical brake is closed or opens too slowly. Check adjustment of brake.
2 Speed controller is too slow Check control parameters (p_gain_speed, ti_speed) and elevator parameters (gear_rat, trac_dia, inertias is too small)
3 The motion of the car is blocked
4 The V-tacho opto-electronic bad or tacho disc is slipping. Check V-tacho (IG).
