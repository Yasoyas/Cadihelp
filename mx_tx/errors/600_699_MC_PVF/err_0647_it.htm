0647 Current difference warn cc1

Summary

 This section describes the MC PVF error Current difference warn cc1.

Description

1) Current difference on CC1 controller between actual and reference detected during trip.
2) Current value CC1 not zero during stand still (produces warning followed by error 11)

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Interrupted current loop on CC1: IGBT short circuit (Variodyn 280) or any other monitoring function (Variodyn 100/140/220: overcurr. overvolt, etc.) activated. Check IGBT, check error LEDs on PWR board (only AEG converter, VF 140 / 220)
2 The current controller reaches its limits, too much current needed. Check LEM wiring connections CC1
3 Wrong PVFP parameters Check parameters
4 Current difference error stored on PCC board during stand still (e.g. caused by interferences), detected at trip start Check error log, if error detected at the trip phase EXCITING_FIELD.
5 Then check PCC board
6 PVFP does not detect an opened safety circuit and tries to run. Check Travel Control event log if there are safety circuit problems
7 Bad contactor K2 Check contactor K2
8 Feed from LEMs of converter CC1 missing Check LEM wiring connection CC1
9 Interferences, bad shielding of the LEM converter cable Check shielding of the LEM cable
10 Motor cables mixed up Check motor cable wiring
11 Current value not zero during stand still (e.g. interferences) Check contactor K2 and Frequency converter
12 Warning as a consequence of error 46/54/62 or 48/56/64 or 50 See warning log
