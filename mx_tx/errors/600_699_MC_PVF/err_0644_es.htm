0644 Input contactor failure cc1

Summary

 This section describes the MC PVF error Input contactor failure cc1.

Description

Input contactor of the parallel converter (1_SGR) not responding to switch on or switch off signal.
That applies only to the parallel operation with Schindler converters(VF 280).If there is Parallel operationwith AEG converters(VF 140 / 220), the contactor feedback is not connected and this error should never occur (except PVF elevator paraInverter_Model is wrong).

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Input contactor failure Check contactor coil
2 Wrong feedback Check auxiliary contacts and wiring PCC X3.2
3 Wrong wiring, no driver signal Check driver signal VSGR on PVFP X3.4
