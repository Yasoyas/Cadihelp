0642 PCC failure

Summary

 This section describes the MC PVF error PCC failure.

Description

One or more of the three parallel controllers on the PCC board are not enabled (CC1: bridge 9 on the PCC board) and the PVF elevator parameter Inverter_Model is set to parallel operation. Checked at trip start.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Parallel controller not enabled Check bridges on PCC board
2 Wrong PVF parameter Check elevator para Inverter_model
