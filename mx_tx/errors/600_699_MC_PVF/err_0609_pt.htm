0609 Discharging DC link failure

Summary

 This section describes the MC PVF error Discharging DC link failure.

Description

This failure may occur during "Test current loop" and "Test DC link" functions started from CADI. The command to switch-off both charge- and input-contactors is issued but the DC link voltage does not decrease.With Variodyn 100, 140, 220 this error will always occur, because the DC link and contactors are not under control of the PVF board.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Input contactor or charge contactor malfunction, in which case this error appears together with error 612 or 613. Check input and charge contactors SGRW, SGR.
2 Wrong feedback from one of these contactors. Check auxiliary contacts on SGRW, SGR
