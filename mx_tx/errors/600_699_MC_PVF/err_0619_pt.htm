0619 Speed reference warning

Summary

 This section describes the MC PVF error Speed reference warning.

Description

1.) After field exciting phase and max. time delay (0.55 sec.), the telegram carrying speed reference value from the interface board does not come.
2.) During any other trip phase, the telegram carrying speed reference value from the interface board does not come for at least once within 20 ms.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Lost of communication. Check connection to Travel Control board.
2 Problem with the interface board. Check Travel Control board local event log, if any error or reset has been performed during trip.
