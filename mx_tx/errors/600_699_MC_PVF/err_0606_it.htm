0606 DC link undervoltage

Summary

 This section describes the MC PVF error DC link undervoltage.

Description

1.) DC link voltage below lower limit. During trip an error is produced. At stand still, only a warning is produced and the contactor SGR opens.
2.) VF 35, 45, 70, 100/BR: may be produced by current loop or capacitor pulse forming test with CADI.
This lower limit is given by the inverter parameter udcl_low_lim.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 The charge- or input-contactor is bad, or wrong parameters ("t_char_on / off", "t_inp_on / off"), in which case error 606 is combined with error 635, 608, 612 or 613 (VF35 - 100/BR only). Check contactors and parameters
2 DC link or main supply malfunction. The parameter t_dcl, time for charging DC link is too short. Check mains and fuses
3 Bad parameter Check parameter udcl_low_lim
