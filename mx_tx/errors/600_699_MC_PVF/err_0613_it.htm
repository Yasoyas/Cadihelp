0613 Charge contactor failure

Summary

 This section describes the MC PVF error Charge contactor failure.

Description

Charge contactor (SGRW) not responding to switch on / off signal.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Charge contactor failure (Variodyn 35 - 100/BR only) Check contactor coil
2 Wrong feedback. X3.3 and/or X4.11 wrong wiring. Check auxiliary contacts and wiring
3 Wrong parameters (t_char_on / off are too small) Check parameters t_char_on / off
