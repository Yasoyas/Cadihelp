0635 Contactor time exceeded

Summary

 This section describes the MC PVF error Contactor time exceeded.

Description

During each switching on or off of the charge- and/or input-contactor, their logical states are controlled by the monitoring function. If any of these states is not defined for longer than 10 Sec. this warning will be generated. Same as before.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Bad set of PVF parameters: "t_char on / off", "t inp on / off" (higher than 10 sec.) Set these parameters to the default value (0.2 sec.).
