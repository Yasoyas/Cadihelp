0608 Charging DC link failure

Summary

 This section describes the MC PVF error Charging DC link failure.

Description

Occurs if the DC link cannot be charged at the beginning of a trip or during a test function.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Input contactor or charge contactor malfunction, in which case this error appears together with warning 612 or 613. Check input or charge contactors (SGRW, SGR, see warning log).
2 Check coul of contactors.
3 Wrong feedback from one of these contactors, in which case this error appears together with warning 612 or 613. Check auxiliary contacts and wiring (PVF X4.10, X4.11).
4 Converter power supply problems, in which case this error appears together with error 606 Check converter power supply.
5 New AEG inverter (one contactor), parallel inverter (with AEG): front end converter could not be switched on, check converter and contacts RSK, RSK1
6 New AEG inverter (one contactor), parallel inverter (with AEG): Bad synchronisation with Travel Control, Error 608 as a consequence of an other error It is possible, that after an emercency stop Travel Control tries to start a new trip while K2 is open, so the dc link could not be charged.
7 Check previous errors, check Travel Control local event log if any Travel Control error occurred.
8 Parallel inverter with Schindler converter: Input or charge contactor malfunction, in which case this error appears together with warning 644 or 645 ( 651 or 652 / 658 or 659). Check input or charge contactors of the parallel converter (SGRWx, SGRx, see warning log), check auxiliary contacts and wiring (PCC X3.2, X3.4, ...).
9 Wrong PVF parameters, concerning input- and load-contactor timing. See error 606 (first cause)
