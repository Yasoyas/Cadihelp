0631 Transmit failure

Summary

 This section describes the MC PVF error Transmit failure.

Description

At least one byte to send is waiting to be sent. The interrupt is activated but RTS (Ready-to-send) status of the communication UART is not set (e.g. Travel Control performs a reset at that moment and could not communicate). After 1 sec warning 31 is produced.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 The communication line is blocked. Check communication cable (X8)
2 Disturbance or interference. Check shielding
3 Travel Control not responding. Could happen if Travel Control perfomes a reset while PVF wants to send a byte.
4 Bad UART Change board
