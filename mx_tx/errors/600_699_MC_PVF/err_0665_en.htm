0665 Overtemperature cc3

Summary

 This section describes the MC PVF error Overtemperature cc3.

Description

Over temperature check has detected power transistors on converter CC3 overheated. When the lift is at standstill, it will be blocked, if it is in motion,the trip will be completed.Ifit remains in motion for over 30 sec, an emergency stop will be generated.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Overtemperature on the heat sink of inverter CC3. Check power transistors.
2 Wrong wiring of the input signal Check wiring of the input signal 3_KTHVF (PCC X?.?)
