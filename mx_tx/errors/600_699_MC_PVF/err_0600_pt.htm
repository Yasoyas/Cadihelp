0600 Attempt to start not allowed

Summary

 This section describes the MC PVF error Attempt to start not allowed.

Description

This error can onlybe detected atthe start of the 1st trip and usually occurs when 1 or more fatal errors were detected during either the stand-still or boot-phase.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 One of more of the following errors were detected: 626, 627, 629, 630. Check the corresponding error description.
