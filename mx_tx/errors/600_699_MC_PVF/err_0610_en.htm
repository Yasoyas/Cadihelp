0610 Current loop not OK

Summary

 This section describes the MC PVF error Current loop not OK.

Description

1.) Feedback from LEMs missing (point a).
2.) VF 35, 45, 70, 100/BR only: CADI test loop function detects difference between IST and REF outside tolerance (point b to f).

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 The feedback cable X1 bad or not connected in its socket. Check feedback cable X1.
2 Interruption in the current loop,any of the components: power IGBTs, motor supply in cables, motor wiring. LEM malfunction or wrong installation.
3 The shortcut circuit monitor system at the driver board has triggered.
4 Wrong current measurement resistors on PVF. Check board, see CADI PVF Help under Current Measuring Resistors
5 Wrong PVF parameters "Is ADC12 res" or/and "Is ADC8 res". Check parameters
6 The cable X6, link from PVF to the driver boards, bad or wrong wired.
