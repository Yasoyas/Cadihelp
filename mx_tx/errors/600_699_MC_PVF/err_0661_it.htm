0661 Charge contactor failure cc3

Summary

 This section describes the MC PVF error Charge contactor failure cc3.

Description

Charge contactor of the parallel converter (3_SGRW) not responding to switch on or switch off signal.
That applies only to the parallel operation with Schindler converters (VF 280). If there is Parallel operation with AEG converters (VF 140 /220), the contactor feedback is not connected and this error should never occur (except PVF elevator paraInverter_Model is wrong).

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
