0604 DL overvolt trigg

Summary

 This section describes the MC PVF error DL overvolt trigg.

Description

DC link overvoltage detected if > 730 V. Checked at start and every 10 ms during the trip by software.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Variodyn 35, 45, 70, 100/BR: Malfunction of the Brake Chopper, Brake Resistor. Check Board TBR (VF70-94), brake chopper Skiip module (VF35, 45, 70-95, 100/BR)
2 Screws of the brake chopper badly tighten. Check screws of the brake chopper (IGBT).
3 Variodyn 100/140/220 has its own control circuits and this monitoring function is redundant. Ignore
