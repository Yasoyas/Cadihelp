0637 Overtemperature brake resistor

Summary

 This section describes the MC PVF error Overtemperature brake resistor.

Description

Overtemperature check has detected brake resistors overheated. When lift is at standstill (and call is given) it will be blocked, if in motion trip will be completed. If it remains in motion for over than 30 sec. an emergency stop follows.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Variodyn 35, 45,70, 100/BR: Overtemperature on the Inverter's brake resistor. Malfunction of the Brake Chopper, Brake Resistor. Check brake chopper and brake resistor
2 Wrong wiring of the input signal X4.9. Check connection of the signal KTHBR.
