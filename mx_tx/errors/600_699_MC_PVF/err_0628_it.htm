0628 Board battery failure

Summary

 This section describes the MC PVF error Board battery failure.

Description

After each reset and after eachtrip, the reliability of the system battery (capacitor) is approved.If this failure occurs, the battery (capacitor) should be exchanged as soon as possible, because with the power-off, all data from RAM will be deleted.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Old or too long operated battery. Replace battery or capacitor.
2 Check date on battery.
3 Wrong mounting or handling Check connections on PRUM board.
4 Discharged through bad packaging material.
