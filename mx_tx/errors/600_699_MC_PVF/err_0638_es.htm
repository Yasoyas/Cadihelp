0638 FCR no start

Summary

 This section describes the MC PVF error FCR no start.

Description

After a trip command, the PVF enables the FCR. If it does not start, this error will be generated. Only active, if elevator parameter Inverter_Model is set to 15 (FCR).

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 No internal FCRclearance due to undervoltage (mains), Skiip failure, ... Check FCR error log
2 Bad input wiring Check wiring connections PVF X4.5 (input)
3 Bad output wiring Check wiring connections PVF X3.7
