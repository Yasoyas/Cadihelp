1070 LiftIdSwitchInvalid

Summary

 This section describes the CPU error LiftIdSwitchInvalid.

Description

The lift-ID switch is set incorrectly
TX5/HX Note: The LiftID switch (On CRIP or ASILOG) is checked at every reset. It must be set to a value from 1 to 9, or A (A=10) according to which lift it is in a group of elevators. Switch settings 0 and B through F are not allowed, otherwise the default setting LiftID=1 is used instead.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Check that the switch setting matches the correct Lift ID (1-9, A)
2 Check that the switch really clicked into the correct setting, for example, sometimes the switch is halfway between 0 and 1
3 If the error persists, suspect a defective ASILOG or CRIP board
