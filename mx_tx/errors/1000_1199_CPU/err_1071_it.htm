1071 MissingEB

Summary

 This section describes the CPU error MissingEB.

Description

A required EB (Expansion Board) is missing
All ‚EB expansion boards (Like EBLON and EBCOM) are identified at every reset, and the device drivers are initialized accordingly. Some system configurations require certain EBs to be installed.
For example, if LiftID=2 or greater, than an EBCOM is required for group communication. A minimum of one EBLON is always required for LON communication.

Extra Info

ID of the missing EB:
90: ASILOG3 missing
C0: EBLON2 missing
C1: EBCOM missing
C2: EBLON1 missing
C3: Embedded EBLON (in CRIP) missing
B0: CRIPHNA missing
E0: Embedded SMLCD missing

Source ID

None.
