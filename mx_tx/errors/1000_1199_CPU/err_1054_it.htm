1054 ClockLossReset

Summary

 This section describes the CPU error ClockLossReset.

Description

The CPU was reset due to the Clock Synthesizer loosing the Crystal Frequency Reference signal.
The GCIO360 uses a 4.192 MHz crystal and generates 25.0 or 33.3 MHz using a Phase Lock Loop. If the Crystal Reference is lost, the processor resets.
