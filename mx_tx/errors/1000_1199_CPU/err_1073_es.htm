1073 ManualDoubleReset

Summary

 This section describes the CPU error ManualDoubleReset.

Description

The CPU was reset because the reset button was pressed twice. One region (memory range) of the NVRAM is cleared. Be aware that certain regions of the NVRAM are not cleared by a double reset. See the description of CPU error 1074 for a description of the cases when the whole NVRAM gets cleared.
