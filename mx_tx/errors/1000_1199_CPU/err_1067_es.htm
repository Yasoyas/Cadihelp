1067 BatteryLow50

Summary

 This section describes the CPU error BatteryLow50.

Description

The CPU was reset, and the battery has less than 50% charge remaining
TX5 Note: Normally, the battery level is tested at every reset, and also once every day. If the battery is consistently low, the battery should be checked to see if it is OK, and replaced if necessary.

Extra Info

None.
