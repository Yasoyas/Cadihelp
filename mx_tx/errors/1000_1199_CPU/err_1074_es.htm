1074 Init NVRAM

Summary

 This section describes the CPU error Init NVRAM.

Description

At startup the software detects that NVRAM contains a recorded software version signature different from the FLASH and reinitializes all of the NVRAM (non volatile RAM) memory. This happens typically if either after a new SW is downloaded, or (more rarely) if the elevator has been switched off for a very long time and the NVRAM has lost its data. When the NVRAM is initialized the following data is lost: Content of Errorlogs, error counters, on-site error adjustments (limits).
