1037 FP Underflow

Summary

 This section describes the CPU error FP Underflow.

Description

The floating-point coprocessor informed the CPU of an underflow condition. (Values smaller than 2.2E-308 or 1.7E-4932, depending on precision) This corresponds to the ‚UNFL bit in the FPU exception byte.

Extra Info

Return Program counter after the fault

Source ID

None.
