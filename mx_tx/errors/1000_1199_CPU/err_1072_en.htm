1072 Saved Registers

Summary

 This section describes the CPU error Saved Registers.

Description

This error is reported along with another CPU error to provide extra information as to the circumstances of the error. See the actual error for more details. Only the registers most useful to locating the error are saved.
Should the error occur on an installation immediately save the error log and contact the VX-Center or the R&D hotline.

Extra Info

Value of the register.

Source ID

Identification number of the register which was saved (R&D internal use).
