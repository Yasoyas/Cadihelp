1049 CPM Interrupt

Summary

 This section describes the CPU error CPM Interrupt.

Description

The RISC Communication CoProcessor Module interrupted the CPU but there was no installed interrupt handler, or Communication Processing Module informed the CPU of an error condition, such as a DMA bus error, or if a CPM interrupt was masked prior to servicing.

Extra Info

Return Program counter after the fault
