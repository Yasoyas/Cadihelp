1064 Lift24vFail

Summary

 This section describes the CPU error Lift24vFail.

Description

The CPU was reset, and the Group 24V supply is below tolerence (24V20%). If warning 1063 GroupPowerFail also appears at the same time as this warning, the system may be running on the backup battery power only.
TX5 Note: Normally, the battery level is tested at every reset, however if either 1064 LiftPowerFail or 1064 LiftPowerFail event appears, then the reserve level of the battery is not tested, since testing the battery could possibly interrupt power to the control in this case.
