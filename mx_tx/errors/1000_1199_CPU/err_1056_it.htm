1056 BErr Released Write

Summary

 This section describes the CPU error BErr Released Write.

Description

The 32-bit Motorola 68020, 68360 and newer processors may still be writing data (Released Write) from a previous instruction while the next instruction is being fetched or executed. The CPU was reset due to a Bus Error while performing a released-write.

Extra Info

Program counter before the fault

Source ID

None.
