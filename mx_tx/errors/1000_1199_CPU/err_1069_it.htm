1069 BatteryDeadOrMissing

Summary

 This section describes the CPU error BatteryDeadOrMissing.

Description

The CPU was reset, and the battery was dead or missing
TX5 Note: Normally, the battery level is tested at every reset (As long as Group24V and Lift24V to ASILOG are OK) and also once every day. The battery voltage was tested, and found to be too low. The battery is either completely discharged, dead, damaged, or disconnected.

Extra Info

None.

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Check the wiring and connections to the battery
2 Leave the controller powered on, see if the battery is charged eventually
3 Measure the voltage of the battery. It should be about 12v
4 If the error continues, check the battery and replace if necessary
