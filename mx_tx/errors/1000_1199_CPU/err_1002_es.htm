1002 Illegal Instruction

Summary

 This section describes the CPU error Illegal Instruction.

Description

The (CPU) has fetched an illegal or inimplemented instruction, or a permament breakpoint instruction, such as ILLEGAL or BGND. This can also happen when attempting to execute Floating-point Coprocessor, cache, or memory management instructions on a processor that does not support these features, or if the co-processor is not functioning. For example, the 68360 can execute some 68020 instructions, but the 68000 cannot. Other instructions are reserved by Motorola for future 68XXX family processors.

Extra Info

Return Program counter after the fault
