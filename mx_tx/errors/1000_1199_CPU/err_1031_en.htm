1031 Autovector 6

Summary

 This section describes the CPU error Autovector 6.

Description

The CPU was interrupted by a external device connected to IRQ6, but the Exception Vector Table has not been initialized by the device driver for that device.
TX5-ASILOG: IRQ6 is connected to Slot 3 (Slot 1 is closest to GCIO360)
HX-CRIPHNA: IRQ6 is connected to the Embedded LON channel (Buried Slot 3)

Extra Info

Return Program counter after the fault

Source ID

None.
