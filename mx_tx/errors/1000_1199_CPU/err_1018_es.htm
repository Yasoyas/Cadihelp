1018 Appl. Requested Reset

Summary

 This section describes the CPU error Appl. Requested Reset.

Description

The CPU was reset at the request of the normally running software. Resets due to errors or software failures are reported and logged by other CPU error ID's.

Extra Info

Identifies the case in the application that required the reset (For R&D internal use)

Source ID

Task identification number (depends on the SW version; for R&D internal use)

Causes and Actions
ID Possible causes of message Actions to be taken
1 none
