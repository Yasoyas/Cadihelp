1017 Event Handler Reset

Summary

 This section describes the CPU error Event Handler Reset.

Description

The CPU is reset because another error with the error severity set to "RESET" occurs. The severity "RESET" for an error will cause the CPU to be reset, and the event 'CPU Event Handler Reset' will be reported once the system restarts. The causing error itself will normally be logged before the reset happens.

(Note: "CPU" errors 1000-1199 always report the reason for a reset AFTER it has happened.)

Extra Info

Identifies the case in the event handler process that required the reset (For R&D internal use)

Source ID

Task identification number (depends on the SW version; for R&D internal use)
