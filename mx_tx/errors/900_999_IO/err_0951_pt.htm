0951 IO_Configuration_Error

Summary

 This section describes the IO error IO_Configuration_Error.

Description

The software detects a problem with the configuration of the inputs/outputs. May be caused by:

- an error in the IO configuration in the PCT (TX-GC/TXR5)
- a conflict when the software adds BMK's to the internal IO database (MX-GC/TX-GC2)

Extra Info

x02000200: Problem retrieving information from BMK name
x02aabbcc: Invalid entry in PCT section PCT_INPUT_OUTPUT (aa:node_id; bb:ioType;cc:pinID )
xdddddddd: Problem reading PCT (dddddddd: subsys_id)

Source ID

None.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Incorrect IO configuration in PCT (TX-GC/TX-R5) Check/correct PCT
2 Incorrect configuration of peripheral IO interface boards Verify correct configuration of configurable IO interface boards.
