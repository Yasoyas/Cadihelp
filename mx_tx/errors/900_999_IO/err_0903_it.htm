0903 IO Table Overflow

Summary

 This section describes the IO error IO Table Overflow.

Description

The software detects an overflow in one of the following data tables handled by the core software.
1: ascii_table
2: message_table
3: table melody
4: table pin_to_floor
5: table ga_volume
6: table access_request
7: table io_sdb[]

Extra Info

The extra info value identifies the table and the index value that caused the overflow.

Interpretation of extra info value xAABBCCDD:
AA: Table number (see description)
BB: index number
CC: not used (00)
DD: not used (00)

Source ID

Identifies the source code file that caused the data overflow.

Causes and Actions
ID Possible causes of message Actions to be taken
1 Wrong configuration section in PCT Check PCT
2 Configuration out of range Check application limits
3 Problem in software with table limits Report to R&D Ebikon
