 attr_1 : vxt_M10_terminal_type

data_length: 6
data[0] : first character mask
data[1] : second character mask
data[2] : node_type (high byte)
data[3] : node_type (low byte)
data[4] : node_id (high byte)
data[5] : node_id (low byte)

Description: Text (2 characters) to display on an M10 terminal.

character mask:
bit 0..7: ASCII code.
bit 8 : blinking mode [0: steady, 1: blinking].

If the highest bit of the character is set the
character is displayed blinking.

vxt_M10_terminal_type:
1: floor terminal
2: car terminal
3: CADI terminal
4: Lobby Vision terminal

The node_type and node_id are addresses that identify the terminal where

The text has to be sent.
