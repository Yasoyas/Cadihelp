Description: Reply for VCOM_get on telegram VXT_GET_EXTENDED_LOG.
Multiple telegrams can be delivered. VXT_START, VXT_END and

data[0] are used for sequence control.


log_type: (see VXT_T_EH_WHICH_LOG in vx_tel_def.h)
1: VXT_SELECT_TOTAL_LOG, Total Log.
2: VXT_SELECT_EVENT_LOG, Event Log.
3: VXT_SELECT_ERROR_LOG, Error Log.

action: (see VXT_T_EH_ACTION in vx_tel_def.h)
1: VXT_START
Value used for all entries except the last one when
replying to a get extended log command.
2: VXT_END

Value used for the last entry when replying to
a get extended log command.
3: VXT_EMPTY

When log is empty.
4: VXT_UPDATE
When there is a new entry in the log.
5: VXT_DELETED ?
