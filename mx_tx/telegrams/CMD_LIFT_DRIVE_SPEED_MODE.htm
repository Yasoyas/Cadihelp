CMD_LIFT (3)  /  DRIVE_SPEED_MODE (9)

Description

 Alter drive speed to be either normal / reduced or specified by value. The data [1] and [2] are only relevant if data[0] is 3.

vxt_speed : VALUE DEFINITION
----- -----------------
1 VXT_NORMAL_SPEED
2 VXT_REDUCED_SPEED
3 VXT_NEW_SPEED_VALUE
4 Not valid for this telegram
5 Not valid for this telegram

priority: The value can be from 1 to 7 and will specify the importance of the setting in the context of the other

priorities sent. The value 7 is the highest priority.
