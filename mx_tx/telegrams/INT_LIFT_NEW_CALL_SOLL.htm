Description: This telegram is used to request a new call soll.
The possible values are described in file 'stsw_def.h'

VALIDATE_MODE: MINIMAL_LOAD_MODE:
1 : validate first standing 1 : minimal load none
2 : validate first 2 : minimal load type 1
3 : validate last 3 : minimal load type 2
4 : validate all
5 : validate none FULL_LOAD_MODE:

6 : validate all below 1 : full load off
7 : validate all above 2 : full load on
8 : validate last below
9 : validate last above PARKING_SERVICE_FLAG_TYPE:
10 : validate shuttle 2 : parking not allowed
11 : validate in travel direction 3 : parking allowed

FLOOR_LOCK_OVERRIDE_MODE: FLOOR_SIG_MODE:
1 : floor lock override off 1 : direction arrows disabled

2 : floor lock override on 2 : direction arrows mode KS
3 : direction arrows mode 2
GONG_MODE: 4 : in use indicator
1 : disabled 5 : car arriving indicator
2 : mode 1 6 : direction arrows blinking
3 : mode 2 7 : direction arrows mode KS no both
8 : M10 boarding lamp

POS_DISP_MODE: POS_DISP_TYPE:
1 : drive selector 1 : none
2 : drive position 2 : constant
3 : destination floor 3 : blinking
