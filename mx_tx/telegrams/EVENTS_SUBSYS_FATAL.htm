Description

 attr_1 : vxt_subsys_id
attr_2 : --
data_length: 2
data[0] : vxt_state
data[1] : duration

Description: This telegram is sent when the subsystem detected a
block_short, block_long, block_permanent error.

subsys_id: id of the blocked subsystem
vxt_state: ON = an error with the severity block short, long or

permanent occured.
OFF = the timeout after block short or long error has
elapsed and the subsys is no longer blocked.
duration: if vxt_state == ON: how long the subsys will be blocked.
unit: [min]
value == 0 => block permanently
