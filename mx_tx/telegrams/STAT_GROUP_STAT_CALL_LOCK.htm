Description: This telegram is used to send the call_lock states of
the specified call type as containd in attr_1.
The get's can be requested for a specific lift or with wildcard.
In case of wildcard the information which returns is an 'OR' of all lifts.

That means if one of the lift has a locked call of the corresponding
'VXT_T_CALL_LOCK_TYPE' then the floor-Bit is set to TRUE.


The get-handling vcom_get() is supported with the call type
VXT_T_CALL_LOCK_TYPE and has to be interpreted as follow.

attr_1: VXT_T_CALL_LOCK_TYPE

VXT_C_FL_CALL_SIDE_ALL = 1,/* There is at least one locked-call-type
on independent side */

VXT_C_FL_CALL_SIDE_1, /* 2:There is at least one locked-call-type on
side_1 */
VXT_C_FL_CALL_SIDE_20, /* 3:There is at least one locked-call-type on
side_20*/ */
VXT_C_CALL_SIDE_ALL, /* 4:There is at least one locked cabin_call
on independent side */
VXT_C_CALL_SIDE_1, /* 5:cabin_call locked on side_1 */
VXT_C_CALL_SIDE_20, /* 6:cabin_call locked on side_20 */

VXT_FL_CALL_SIDE_ALL, /* 7:There is at least one locked floor_call
on independent side */
VXT_FL_CALL_SIDE_1, /* 8:floor_call locked on side_1 */
VXT_FL_CALL_SIDE_20, /* 9:floor_call locked on side_20*/
VXT_FL_CALL_UP_SIDE_ALL /*10:There is at least one floor_call_up
locked on independent side */
VXT_FL_CALL_DOWN_SIDE_ALL /*11:There is at least one floor_call_down

locked on independent side */
VXT_FL_CALL_UP_SIDE_1 /*12:floor_call_up locked on side_1 */
VXT_FL_CALL_DOWN_SIDE_1 /*13:floor_call_down locked on side_1 */
VXT_FL_CALL_UP_SIDE_20 /*14:floor_call_up locked on side_20 */
VXT_FL_CALL_DOWN_SIDE_20 /*15:floor_call_down locked on side_20 */
