OUT_DRIVE (18)  /  LM_CMS_STATE (8)

Description

 attr_1 : vxt_lm_cms_cmd_type
attr_2 : vxt_deck
data_length: 8
data[0] : commission mask (high byte)
data[1] : commission mask (low byte)
data[2] : current value (high byte)
data[3] : current value (low byte)
data[4] : minimal value (high byte)
data[5] : minimal value (low byte)
data[6] : maximal value (high byte)

data[7] : maximal value (low byte)

Description: informs the SMLCD about existing history of LMS device as
well as about the parameter associated with used command.

Commission Mask: The mask identifying commission history of load
measurement device.
The mask is set if corresponding command has not been done.
The mask is cleared if the command has been already

successfully done.
Nominal Load Command 0x01
Basic Load Command 0x02
Full Load Command 0x04
System Check Command 0x08
Empty Bottom Command 0x10
Empty Top Command 0x20
Empty Middle Command 0x40

Current Value: default Value of parameter associated with used command.

Minimal Value: minimal limit of parameter associated with used command.
Maximal Value: maximal limit of parameter associated with used command..
