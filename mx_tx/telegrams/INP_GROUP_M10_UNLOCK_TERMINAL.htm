INP_GROUP (15)  /  M10_UNLOCK_TERMINAL (77)

Description

 attr_1 : --
attr_2 : --
data_length: 2
data[0] : unlock time [minute] (0..255)
data[1] : vxt_state

Description: To unlock the M10 terminal (to be able to modify
the floor number, the position,...).

unlock time | vxt_state | effect
------------+--------------+-------------------------------

0 | VXT_STATE_0 | lock the terminal
------------+--------------+-------------------------------
0 | VXT_STATE_1 | unlock the terminal (until
| | a lock command occur)
------------+--------------+-------------------------------
> 0 | VXT_STATE_0 | no effect
------------+--------------+-------------------------------
> 0 | VXT_STATE_1 | unlock the terminal for the

| | specified time, then lock it
| | again automatically.
------------+--------------+-------------------------------

The combination '> 0 and VXT_STATE_0' has no effect to allow
to use a key switch to unlock the terminal temporary without
having to let the key in the switch.
