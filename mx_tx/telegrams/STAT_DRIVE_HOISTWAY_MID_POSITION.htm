 attr_1 : --

data[0] : position
data[1] : next_selector

Description: position of the car in respect to the counterweight.

position:
0: invalid
1: car below counterweight
2: car above counterweight

next selector: only valid if position (data[0]) is valid. Nearest

floor in the opposite direction of where the
the counterweight is (up if car above, down if
car below).

Remark: the drive must be asked to send this telegram using a 'get'.
It will then sent it when the car is standing (immediately if already
standing or, if not yet standing, the next time after the 'get' the
car is standing). The Drive send this telegram with a normal 'send'

not with a 'reply' => a 'request' must be done to receive it.
