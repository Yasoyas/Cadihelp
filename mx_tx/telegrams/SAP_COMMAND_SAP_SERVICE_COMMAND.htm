SAP_COMMAND (42)  /  SAP_SERVICE_COMMAND (1)

Description

 AP_SERVICE_COMMAND werden von einem Benutzer von SAP ausgeloest (CADI, LCD, Technische Zentrale, ...). Als Lift_id muss der entsprechende Lift angegeben werden.

vxt_sap_service_command:

VXT_SAP_PARAM_UPDATE_CMD:

Parameter in der PCT wurden geaendert und muessen nun durch die Applikation (SAP) uebernommen werden.

VXT_SAP_TEST_CALL_CMD:
attr_2 : VXT_REG_CENTER
SAP soll ein Testanruf auf Regionalzentrale- Telefonnummer starten.

attr_2 : VXT_AUX_CENTER
SAP soll ein Testanruf auf die Hilfsalarm-Telefonnummer starten.

attr_2 : VXT_AL_CENTER
SAP soll ein Testanruf auf die Alarmzentrale-Telefonnummer starten.

attr_2 : VXT_PIC_CENTER
SAP soll ein Testanruf auf die Pikketzentrale- Telefonnummer starten.

attr_2 : VXT_MINI_CENTER
SAP soll ein Testanruf auf die Kundenzentrale- Telefonnummer starten.

attr_2 : VXT_RESERVE_CENTER
SAP soll ein Testanruf auf die Reserve-Telefonnummer starten.

VXT_ENGINEER_ON_SITE_CMD:
Monteur ist an der Anlage.

VXT_ENGINEER_OFF_SITE_CMD:
Monteur ist nicht an der Anlage.

VXT_ENGINEER_STATE_CMD:
Zustand des EngineerOnOFFSiteFlag wird gefragt.

VXT_COLD_RESET_CMD:
Bei einem Kaltreset werden folgende Aktionen gestartet:
- alle Modelle werden initialisiert
- alle Hostbuffer werden geloescht
- der lokale Warnungsbuffer wird geloescht
- Engineer On Site
