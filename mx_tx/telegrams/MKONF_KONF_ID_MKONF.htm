Description: Make a configuration from the PCT
Telegramm zum senden der Konfigurationsdaten von
einem Driver zu DATA_CONFIG und/oder zur Konfigura-
tionsfunktion vom entsprechenden Subsystem. DATA-

CONFIG wird alle VXT_MKONF-Telegramme erhalten, da
sie ein Request auf alle Parametergruppen gemacht
hat.
Eine genaue Beschreibung von diesem Telegramm ist
im Dokument 'EZ-ST 90 223'.
konf_id und konf_data sind im vorletzten Telegramm
VXT_KONF beschrieben.Von einem MKONF-Telegramm wer-
den nur die Parameterdefinitionen verwendet.
