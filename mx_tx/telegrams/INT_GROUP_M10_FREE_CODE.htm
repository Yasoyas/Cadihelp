Description: Activate customized coded service
Sent by the master to a lift to activate a customized
coded service. This telegram writes several service-variables
and service infos which can be evaluated in the
activation-/deactivation-conditions of a customized service to
activate/deactivate it.
Depending service data[1]..[4] can be zero if not used.
