CMD_LIFT (3)  /  PARK_CAR (4)

Description

 data_length: 5 if mode = 1, 0 if mode = 2
data[0] : floor_number
data[1] : vxt_dir
data[2] : door_action
data[3] : door_open_time (high byte)
data[4] : door_open_time (low byte)

Description: command to the lift to park or unpark its car.

mode: 1: park
2: unpark (no data in that case). Used when parking parameters or

master changed to reinitialize the parked cars.

floor_number: parking floor.

vxt_dir : direction of the parked car that the direction
arrow(s) must show.
Caution: this is not a preselected direction.
Only relevant if a door will open (door_action not
equal to zero).

door_action: door mask to specify which door must open

or not:
bit_0 door_action on side_1 (LSB)
bit_1 door_action on side_20
0 = no door_action
1 = door_action

door_open_time: door open time in [0.1s]
