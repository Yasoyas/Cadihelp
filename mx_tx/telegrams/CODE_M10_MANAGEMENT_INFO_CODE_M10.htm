CODE_M10_MANAGEMENT (54)  /  INFO_CODE_M10 (3)

Description

 attr_1 : info type
attr_2 : --
data_length: 8
data[0] : internal service code [0.255]
data[1] : external service code (high byte) [0..81]
data[2] : external service code (low byte) [0..91]
data[3] : mode descriptor
data[4] : start hours [0..23]
data[5] : start minutes [0.59]
data[6] : end hours [0..23]

data[7] : end minutes [0..59]

Description: information about a M10 code.

info type:
2: service tools (CADI, LobbyVision,...) [CADI]
3: initialisation [REBOOTING_LIFT]

mode descriptor: (refer to external service code)
0: ACTIVATED, the code can activate a service via a terminal.
1: DEACTIVATED, the code cannot activate a service via a terminal.
2: TIME_MODE, the code activates a service only in a specified time.

range (see data[4]..data[7]).
3: NOT_USED, code is not used.
4: CHANGE_CODE, it is the code used to activate the change code mode
via a terminal.
