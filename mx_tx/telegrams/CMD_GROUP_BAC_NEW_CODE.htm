CMD_GROUP (4)  /  BAC_NEW_CODE (9)

Description

 attr_1 : vxt_side
attr_2 : --
data_length: 4
data[0] : floor_number
data[1] : code_1
data[2] : code_2
data[3] : code_3

Description: new button access code to be stored in group for the
specified car call (for service BAC, normally sent by
Lobby monitor to change car call code).

code_1: first number of the car call code
code_2: second number of the car call code
code_3: third number of the car call code

CAUTION: the numbers of this code must correspond to existing floor
numbers (floor number in group notation (first floor within the
group is floor 1), not floor name) because the code is given
using the car call buttons.

Specification
Telegram Group: CMD_GROUP
Telegram: BAC_NEW_CODE
Attribute 1: T_SIDE
Attribute 2: -
Datalength: 4
Data[0]: T_FLOOR
Data[1]: T_CODE
Data[2]: T_VALUE
Data[3]: T_VALUE

 door side
