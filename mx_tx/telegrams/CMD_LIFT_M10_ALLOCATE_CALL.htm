CMD_LIFT (3)  /  M10_ALLOCATE_CALL (12)

Description

 This telegram would be sent by the master to the elevator that is to be
allocated the M10 call.

[This telegram is added as part of a new mechanism that is put in place to allocate an M10 call. In this mechanism, the master would not blindly allocate a call to the cheapest elevator, but have an answer in return if the elevator accepted the call. This answer comes in as the second telegram that is added below.]

data_length: 10
data[0] : boarding floor (starting with 1)
data[1] : boarding side
data[2] : exiting floor (starting with 1)
data[3] : exiting side
data[4] : absolute halfround [0..max-1]
data[5] : relative halfround [1..max]
data[6] : number of passenger
data[7] : terminal position
data[8] : 'over the hill' flag
data[9] : allocation id [0..255]
