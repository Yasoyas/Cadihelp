INT_LIFT (23)  /  SIM_CABIN_CALL (4)

Description

 data_length: 6
data[0] : floor_number
data[1] : call_lock_check_mode
data[2] : door_action
data[3] : vxt_sim_type
data[4] : special door open time high byte
data[5] : special door open time low byte

Description: Give a simulated call

call_lock_check_mode:
0 no check
1 check

door_action:
0 = no door_action
1 = door_action
bit_0 door_action on side_1 (LSB)
bit_1 door_action on side_20
bit_2 door_action on side_30
bit_3 door_action on side_40

vxt_sim_type:
1 VXT_SIM_NORMAL
- Simulated and volatile Cabin-Call
2 VXT_SIM_SEARCH_NEXT_FLOOR
- if floor_number == 0: sim call at selector position.
- if floor_number == x: sim call to the next possible floor x (x excluded).
3 VXT_SIM_NONE_VOLATILE
- Simulated none volatile Cabin-Call
not automaticaly cleared when other calls occur).
4 VXT_SIM_NEXT_FLOOR_UP
- Simulated call (volatile) to the next possible floor up. No call generated if car currently travelling down.
5 VXT_SIM_NEXT_FLOOR_DOWN
- Simulated call (volatile) to the next possible floor down. No call generated if car currently travelling up.

door open time:
0: standard door open time (like for car call).
1..65534 (0xFFFE): special door open time, as specified in [0.1s].
65535 (OxFFFF): keep door open.
