1. Introduction

1.1 Purpose

This document describes the errors and their indication. In addition, some hints for trouble shooting are given.

1.2 Scope

This description is valid for mains converters controlled by SW FCR 5.1:

  • PVEC/PIOFCR: VF155/VF288

1.3 References to documentation

  • none

2 Errorlog

2.1 General concept

The errors of the mains converter are stored in an error log. This can be accessed using GDDT. In addition, pending errors are indicated on the display of PIOFCR.

2.2 Error state

The error state is indicated by the LED's on PIOFCR as follows:
ERROROKPARAMETERDescription
OFFOFFOFFFCR not available, check power supply.
OFFOFFONFCR is not commissioned. Load parameters using GDDT.
OFFONOFFFCR is ready for operation or is in operation.
ONONOFFFCR is blocked. Check 7 segment display.
ONOFFOFFFCR is in dead state. Check 7 segment display.

36PLL not ok.

This error occures only on parallel inverter and only on slave!

PWM of master and slave line inverter is not synchronized.

Possible reason:

  • Wrong adjustment of switches on CANPCON.
  • Interruption of communication cable between the CANPCON of master and slave line converter.

Check 7 segment display on CANPCON: slave line inverter must show "PLL".
