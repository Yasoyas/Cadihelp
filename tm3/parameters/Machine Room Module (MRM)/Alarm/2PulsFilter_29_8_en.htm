2 Puls Filter

Summary

 This section describes the parameters in the parameter window 2 Puls Filter of the parameter group Alarm in subsystem Machine Room Module (MRM).

DAKA

Time before alarm is created after pressing DAKA.
Also see 'Filter Type'. 














T0


Holding period for first alarm key puls. 
Also see 'Filter Type'. 





















T1


Delay time interval before repressing of alarm key. 
Also see 'Filter Type'. 





















T2


Holding period for second alarm key puls. 
Also see 'Filter Type'. 





















T3


Maximum time for two-puls filter before reseting to first puls detection.
Also see 'Filter Type'.
