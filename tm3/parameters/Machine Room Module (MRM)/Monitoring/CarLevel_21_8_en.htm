Car Level

Summary

 This section describes the parameters in the parameter window Car Level of the parameter group Monitoring in subsystem Machine Room Module (MRM).

Monitoring

The value 'Monitoring ON/OFF' enables or disables the car level monitoring.
This test checks the proper stop position of the car. If leveling is insufficient, a warning or an error (depending on given values) will be produced There is a distinction between a leveling problem on a specific floor and a general leveling problem. 














Problems on same floor until Warning


The value 'Problems until Warning' specifies the necessary amount of problems on the same floor to throw a warning. Also see 'Monitoring'. 





















Problems on same floor until Error


The value 'Problems until Error' specifies the necessary amount of problems on the same floor to throw an error. Also see 'Monitoring'. 





















Problems on all floors until Warning


The value 'Problems until Warning' specifies the necessary amount of problems on any floor to throw a warning. Also see 'Monitoring'. 





















Problems on all floors until Error


The value 'Problems until Error' specifies the necessary amount of problems on any floor to throw an error. Also see 'Monitoring'.
