Phone Numbers

Summary

 This section describes the parameters in the parameter window Phone Numbers of the parameter group Phone Numbers in subsystem System.

Phone Number 1

The TM3 module can store up to 5 telephone numbers.
Format: 24 digits
For example:
Phone number 1: Regional Center
Phone number 2: Alarm Center
Phone number 3: Picket Center
Phone number 4: Auxiliary Alarm Center
Phone number 5: Customer Center (NOTE: This number is not used in TAM-mode)
The use of these telephone numbers can be defined in the parameter group Dial sequences.
The TAM sequence can not be changed!
Special Codes
The telephone numbers and the dial prefix may contain the following special codes:
*2 for to insert a 2 second delay while dialing
*3 DTMF "*" key
*4 DTMF "#" key
*5 DTMF "A" key
*6 DTMF "B" key
*7 DTMF "C" key
*8 DTMF "D" key
*9 enable microphone / call timeout 3 minutes
Example:
0 *2 1234 *9
dial 0, wait 2 seconds, dial 1234 and switch on the microphone
