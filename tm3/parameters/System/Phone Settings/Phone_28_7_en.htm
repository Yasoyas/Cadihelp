Phone

Summary

 This section describes the parameters in the parameter window Phone of the parameter group Phone Settings in subsystem System.

Line Type

For TAM mode only, not configurable. 














Interrupt Detection


For TAM mode only, not configurable. 





















DTMF Sensitivity


For TAM mode only, not configurable. 





















Country Code


Enter the code of the country where the TM3 Module is installed (e.g. '41' for Switzerland) 






















Connection Watchdog


After this time the connection to the control center is interrupted. (Control Center Operator can extend the connection time during call handling.) 





















Recall Delay


Default value: 0 
If there has been no alarm or error, within this time period, the TM3 makes a call to the Control Center (data sequence) to test the telephone line and to transfer statistic data. 
The time is set automatically by the Control Center after the first testalarm. If these parameter is set to 0, this function is disabled otherwise the time value must be larger than 10 min. (Smaller values are set automatically to 10 min) 





















Delay between Attempted Calls


Delay between 2 sequences 





















Maximum Ring Time


Maximum time the TM3 tries to reach a defined telephone number before dialing the next one. 





















Delay DTMF Reply


Delay between a DTMF command and the answer of the TM3. Default: 50 Don't change. 





















MSN


Telemonitoring parameter
